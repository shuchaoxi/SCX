# 已有工作与创新点

## 已有相关工作

### 1. Teacher-student MLIP 蒸馏

代表文献：

- Sakib Matin et al., *Teacher-student training improves the accuracy and efficiency of machine learning interatomic potentials*, Digital Discovery, 2025.
- DOI: `10.1039/D5DD00085H`
- 原文：https://pubs.rsc.org/en/content/articlehtml/2025/dd/d5dd00085h

核心思想：

- 用 teacher 的 latent knowledge，尤其是 atomic energies，增强 student 训练。
- 目标是让轻量 student 保持较高精度，同时提升 MD 速度、降低内存。

与本想法的关系：

- 它证明 MLIP 可以做 teacher-student 蒸馏。
- 但它重点是模型压缩和效率，不是多个专用势函数的化学空间并集合并。

### 2. Ensemble knowledge distillation for MLIPs

代表方向：

- 多个 teacher 的 ensemble 信息蒸馏到一个 student。
- 相关预印本题目：*Ensemble Knowledge Distillation for Machine Learning Interatomic Potentials*。

与本想法的关系：

- 这是最接近“多个老师教一个学生”的方向。
- 但通常仍偏向同一数据分布或同一任务空间，不一定解决“多个二元势函数合成四元势函数”的问题。

### 3. Relational knowledge distillation

代表文献：

- *Angular relational knowledge distillation of machine learning interatomic potentials for scalable catalyst exploration*, npj Computational Materials, 2026.
- 原文：https://www.nature.com/articles/s41524-026-02062-5

核心思想：

- 不只蒸馏能量和力，还蒸馏 GNN teacher 中的几何/角向关系知识。

与本想法的关系：

- 提醒我们：蒸馏目标不一定只有 `E/F/virial`。
- 未来可以考虑蒸馏 descriptor、局部环境相似度、uncertainty 或 teacher committee disagreement。

### 4. Universal MLIP / foundation model

相关方向：

- M3GNet、CHGNet、MACE-MP、SevenNet 等。
- 这些模型主要从大规模 DFT 数据训练，目标是覆盖更多元素和材料空间。

与本想法的关系：

- 它们说明“多元素通用势函数”是大趋势。
- 但你的问题是：是否能利用已有专用势函数，低成本构造局部材料族的 semi-universal potential。

## 本想法的潜在创新点

### 创新点 1：从模型压缩转向化学空间并集

一般蒸馏问的是：

```text
大 teacher -> 小 student
```

这里问的是：

```text
多个专用 teacher -> 更大材料族 student
```

重点不是压缩，而是合并化学空间。

### 创新点 2：研究交叉空间的失效边界

例如：

```text
AlAs teacher
GaAs teacher
InAs teacher
```

是否能推出：

```text
Al-Ga-In-As 三元/四元合金局部环境
```

需要回答：

- 哪些局部环境可以由 teacher 内插得到？
- 哪些局部环境是 teacher 完全没见过的？
- 少量 DFT 数据补在哪里最有效？

### 创新点 3：用半导体材料族做可控 proof-of-concept

半导体合金比 `O-H-Cu`、`Si-C-O-H` 更干净。

优势：

- 结构类型统一。
- 键型相对一致。
- 物理性质可验证。
- 合金空间天然适合测试泛化。

### 创新点 4：蒸馏 + 少量 DFT 主动校准

更现实的路线不是纯蒸馏，而是：

```text
teacher soft labels 负责铺开势能面
少量 DFT 负责校准交叉区域
active learning 负责发现 student 不确定区域
```

这个组合比单纯“从零训练多元素势函数”更有现实意义。
