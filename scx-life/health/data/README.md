# Data Directory

Datasets used in SCX-Health experiments.

## MedMNIST v2
- **PathMNIST**: 107,180 patches from 9 types of colon pathology
- **DermaMNIST**: 10,015 dermatoscopic images of 7 skin lesion types
- **BloodMNIST**: 17,092 blood cell images of 8 cell types

Download via: `python download_medmnist.py`

## HAM10000
- 10,015 dermatoscopic images for skin lesion classification
- 7 diagnostic categories: akiec, bcc, bkl, df, mel, nv, vasc

Download via: `python download_ham10000.py`
Note: HAM10000 requires manual download from Harvard Dataverse.
