# 🧬 药物靶点全景图：SCX「药物发现工厂」计算筛选 TOP 靶点

> **编制日期**: 2026年6月27日  
> **数据来源**: ChEMBL、PDB、BindingDB、ClinicalTrials.gov、PubMed，基于2024-2026年真实临床/结构数据  
> **核心洞察**: SCX 的药物管线是一个「药物发现工厂」——只要有靶点结构和数据，就能批量计算。以下按计算可行性排序。

## A. 神经退行性疾病

### 🅰️-1 靶点卡片：BACE1（β-分泌酶1）

| 维度 | 评估 |
|---|---|
| **疾病** | 阿尔茨海默病（AD） |
| **未满足需求** | ⭐⭐⭐⭐⭐ 全球~5500万AD患者，现有药物仅2款抗体（lecanemab/donanemab），疗效有限（认知衰退减缓~27-35%） |
| **靶点可药性** | ⭐⭐⭐⭐⭐ 天冬氨酸蛋白酶，活性位点清晰，底物口袋深且明确 |
| **结构数据** | **PDB: 300+** 晶体结构（含多种抑制剂共晶）；ChEMBL: **>8,000** 个活性化合物 |
| **临床阶段** | 历史上14个化合物进入临床，**100%失败**。核心教训：过度抑制（>50% Aβ下降）导致认知恶化 + BACE1底物（Sez6/NRG1）受扰。**低剂量部分抑制（~30%）+ 底物选择性抑制剂是新方向** |
| **计算可行性** | ⭐⭐⭐⭐⭐ **极高**。大量结构数据可实现精准分子对接；核心计算问题：设计高BACE1/BACE2选择性 + 底物选择性抑制剂。失败历史反而是计算优势——已知"不要做什么"，可逆向设计 |
| **市场价值** | ⭐⭐⭐⭐⭐ 2023年AD市场~$70亿，2030年预计>$300亿 |

> 💡 **SCX策略**: BACE1是计算筛选的"终极试金石"——数据最丰富、失败经验最明确的靶点。应聚焦**低剂量部分抑制**策略，计算筛选BACE1选择性好且不过度抑制的新型化学型。

### 🅰️-2 靶点卡片：GSK-3β（糖原合成酶激酶-3β）

| 维度 | 评估 |
|---|---|
| **疾病** | AD（Tau磷酸化）、帕金森、糖尿病、癌症（多效激酶） |
| **未满足需求** | ⭐⭐⭐⭐⭐ Tau磷酸化是AD核心病理之一，但尚无靶向药物 |
| **靶点可药性** | ⭐⭐⭐⭐⭐ ATP-竞争性激酶抑制剂，经典的可药口袋 |
| **结构数据** | **PDB: 100+** 晶体结构（含多种化学型共晶，如9HUK、4DIT、8AUZ、3F7Z等）；ChEMBL: **>5,000** 活性化合物；BindingDB: 大量亲和力数据 |
| **临床阶段** | Tideglusib（不可逆GSK-3β抑制剂）Phase 2b在AD中失败；锂盐（间接抑制）有微弱认知益处。选择性vs其他激酶是核心挑战 |
| **计算可行性** | ⭐⭐⭐⭐⭐ **极高**。ATP口袋适合虚拟筛选 + 分子生成；选择性（vs GSK-3α/CDK/CK1）是计算设计的明确目标函数 |
| **市场价值** | ⭐⭐⭐⭐ AD + PD + 糖尿病，适应症跨多个大市场 |

> 💡 **SCX策略**: 最经典的"数据丰富 + 可药性强"靶点。计算目标：设计高选择性GSK-3β抑制剂（选择率>100× vs CDK家族），避免tideglusib的脱靶问题。

## B. 癌症（未被充分靶向的）

### 🅱️-1 靶点卡片：KRAS G12D

| 维度 | 评估 |
|---|---|
| **疾病** | 胰腺癌（~40%）、结直肠癌（~13%）、NSCLC（~4%） |
| **未满足需求** | ⭐⭐⭐⭐⭐ 胰腺癌5年生存率仅12%，全球每年新发~50万例。KRAS G12D是最大单一突变亚型 |
| **靶点可药性** | ⭐⭐⭐⭐ 2022年前不可药，现已突破。Switch II口袋可被MRTX1133（非共价）锁定；RMC-9805（共价tri-complex）靶向ON状态 |
| **结构数据** | PDB: 新兴增长，已有多组KRAS G12D-抑制剂共晶结构；ChEMBL: G12D特异性数据积累中（G12C有大量先例数据可借鉴） |
| **临床阶段** | MRTX1133 Phase I/II（NCT05737706）；RMC-9805 Phase I（NCT06040541），ENA 2024报告59%肿瘤缩小、ctDNA清零；RMC-9805+RMC-6236组合Phase I |
| **计算可行性** | ⭐⭐⭐⭐⭐ **极高**。Switch II口袋明确、共价/非共价两种策略可选、G12C的计算经验可迁移。**这是当前最"热"的可计算靶点** |
| **市场价值** | ⭐⭐⭐⭐⭐ KRAS药物市场2030年预计>$150亿；胰腺癌领域尚无靶向药，first-in-class价值巨大 |

> 💡 **SCX策略**: 最高优先级。计算方向：(1) G12D选择性vs G12C/G12V/WT；(2) ON vs OFF状态选择性；(3) 共价warhead优化；(4) 口服生物利用度优化。

| 维度 | 评估 |
|---|---|
| **疾病** | MSI-H/dMMR实体瘤：结直肠癌（~15%）、胃癌（~15-30%）、子宫内膜癌（~15-30%）、卵巢癌（~12%） |

| **靶点可药性** | ⭐⭐⭐⭐⭐ 合成致死靶点——MSI肿瘤必需，正常细胞不需要；HRO761确认了变构口袋的可药性（D1-D2解旋酶结构域界面） |
| **结构数据** | 2019年CRISPR筛选发现WRN-MSI合成致死；2024年Nature发表HRO761-WRN共晶结构；2024年Nature Comm发表染色质捕获/降解机制；2025年已有5+篇综述和多个专利（GSK/IDEAYA、Moma、Hansoh） |
| **临床阶段** | HRO761（Novartis）Phase I（NCT05838768）；HS-10515（Hansoh）推进临床；GSK/IDEAYA专利阶段 |

> 💡 **SCX策略**: **顶级优先级**。这个靶点2024年才进入临床，竞争非常不饱和。计算先发优势：(1) 基于HRO761变构口袋的虚拟筛选发现新化学型；(2) ATP竞争性抑制剂（GSK/IDEAYA策略）作为平行路线；(3) PROTAC设计（WRN降解在MSI中天然发生，PROTAC增强此效应）。

## E. 自身免疫/炎症

| **计算可行性** | ⭐⭐⭐⭐⭐ **极高**。假激酶变构口袋——一个独特的计算目标：选择性好（JAK1/2/3无此口袋），已有临床验证 |
| **市场价值** | ⭐⭐⭐⭐⭐ 银屑病+SLE+IBD+PsA市场>$300亿 |

| 维度 | 评估 |
|---|---|

## 📊 计算策略建议

### 一次性批量计算路线图

```
Phase 1: 结构准备（1周）
├── PDB结构下载 + 结构比对 + 口袋定义
├── 靶点口袋保守性分析（KRAS G12D vs G12C/WT，WRN变构 vs ATP）
└── 已知配体/抑制剂收集（ChEMBL + BindingDB）

Phase 2: 虚拟筛选（2-3周）
├── 每个靶点：~10^6-10^7 分子的数据库虚拟筛选
├── 多构象对接（ON/OFF状态，DFG-in/out）
├── 基于已知配体的药效团筛选
└── 选择性筛选（靶点 vs 反靶点）

Phase 3: 分子生成（2-3周）
├── 基于口袋的de novo分子生成（DiffDock/AlphaFold3/RFdiffusionAA）
├── 基于已知配体的分子优化（活性悬崖分析）
├── ADMET预测过滤（尤其是BBB穿透性神经靶点）
└── 合成可行性评分

Phase 4: 精炼+排名（1周）
├── FEP/MD精细自由能计算（TOP 100 分子/靶点）
├── 多参数优化排名（亲和力 + 选择性 + 药代 + 新颖性）
└── 合成优先列表（每靶点 TOP 10-50）
```

### 预期产出

- **10个靶点 × ~10^5 精筛分子/靶点 = ~10^6 分子**进入精细对接
- **10个靶点 × TOP 50 排名分子 = 500个**推荐合成候选
- **总周期**: 6-8周（并行计算）
- **核心价值**: 一次性覆盖当今最重要药物的计算蓝海
