# 📊 仪表盘

> 需要安装 [Dataview](https://github.com/blacksmithgu/obsidian-dataview) 插件。

## 概念图谱

```dataview
TABLE tags, created
FROM "02_概念"
SORT created ASC
```

## 项目状态

```dataview
TABLE status, version
FROM "03_项目"
```

## 最近更新的时间线

```dataview
TABLE date, tags
FROM "01_时间线"
WHERE date != null
SORT date DESC
```

## 待办事项（来自所有文件）

```dataview
TASK
FROM "knowledge"
WHERE !completed
SORT created ASC
```

## 所有 GPT 对话

```dataview
TABLE date, tags
FROM "04_对话/GPT"
WHERE date != null
SORT date DESC
```

## 所有 Agent 讨论

```dataview
TABLE date, tags
FROM "04_对话/Agent"
WHERE date != null
SORT date DESC
```
