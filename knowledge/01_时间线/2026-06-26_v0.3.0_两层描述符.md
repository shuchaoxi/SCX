# 2026-06-26 · v0.3.0 通用模块化架构 + 两层描述符 + 数据防中毒验证

*这是迄今为止产出最密集的一天。*

## 三大主线

### 1. v0.3.0 → v4.0 Phase A 通用模块化重构

用户在与 GPT 讨论中指出："框架还是需要更通用一些，避免一直加模块迭代麻烦，应该模块化，不同的行业可以调用相同的代码。"

**核心设计**：
- **SCXStateEncoder / SCXExpert / SCXDataPoint 抽象基类** — 所有核心模块只依赖抽象接口
- **Encoders 插件式** — 每个领域继承抽象基类，实现自己的 encoder
- **Config 声明式** — YAML 描述完整任务

**效果**：v0.3.0 新增 StateConditionedInfluence、OnlineSCXFramework、CIFAR 实验、MedMNIST 实验。

| 命题/定理 | 3 | 5 (+2: Compress Theorem + Governance Protocol) |

### 2. 两层描述符（Two-Layer State Discovery）

**核心洞察**：单一描述符无法同时捕获物理可解释性和误差结构。

- **Layer 1**：人工设计的物理描述符（如 ACE/SOAP、组分、应变）→ 粗筛
- **Layer 2**：基于 baseline 模型误差的聚类 → 细粒度状态发现

**AlN v3 实验效果**：
- 最大 blob 从 50% 降至 33%
- Noise F1 从 0.253 提升至 **0.585**（2.3x）
- Phonon 结构 100% 隔离到独立状态

### 3. 数据防中毒验证

**SCX vs DFT 对比**：
- SCX 噪声帧 **100% 覆盖** REPORT 最差帧
- 力 RMSE 预估降 **29-48%**

**关键发现**：三条线汇聚验证了 SCX 的边界条件——
- AlN 12-dim descriptor
- SimpleCNN 128-dim latent
- DermaMNIST uniform noise

→ **弱特征**会导致 SCX 失效。这不是 bug，而是一条定理：[弱特征失效]]
