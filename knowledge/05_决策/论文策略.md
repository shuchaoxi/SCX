# 论文发表策略

## 核心策略：EGP-first

```
EGP (Paper 1, concrete) → arXiv 先占坑
    ↓
SCX-Theory (Paper 2, abstract) → 后续投稿
    ↓
SCX-MLIP (Paper 3, apply back) → 整合验证
    ↓
SCX-Sim (Paper 4) → 跨领域扩展
    ↓
SCX-Health (Paper 5) → 跨领域扩展
```

## 决策理由

### 为什么 EGP 先发？

1. EGP 有**完整 DFT 数据**（AlN v3），实验结果 concrete
2. 材料科学方向审稿人容易理解（vs 纯 ML 理论审稿人要求严格）
3. EGP 先发表建立一个"根据地"
4. SCX-Theory 引用 EGP 作为 motivation，形成故事线

### 为什么 SCX-Theory 第二？

1. 数学框架本身是独立的，不依赖 EGP 的实验结果
2. 但有了 EGP 的 arXiv preprint 作为 motivation 引用，故事更有说服力
3. TMLR / AISTATS 接受纯理论 + 合成实验验证的论文
4. arXiv 占坑保护 prior art

### 为什么 SCX-MLIP 第三？

1. 这是"回马枪"论文——用 SCX 理论解决 EGP 的实际问题
2. 整合了理论（Paper 2）和数据（Paper 1），贡献最大
3. 目标 Nat. Commun. / npj Comput. Mater.
