# 两层描述符

## 问题

单一描述符无法同时捕获物理可解释性和误差结构。

- 纯物理描述符（如 ACE/SOAP）→ 可解释但可能错过误差热点
- 纯数据驱动聚类（如 UMAP/HDBSCAN）→ 能发现误差聚类但缺乏物理解释

## 方案

```
Layer 1: 人工设计的物理描述符（组分、应变、配位）
    ↓ 粗筛（可解释的状态分组）
Layer 2: 基于 baseline 模型误差的聚类
    ↓ 细粒度状态发现
最终状态 = Layer 1 分区 ∩ Layer 2 聚类
```

## 数学形式

**命题 6**（两层状态发现定理）：存在一种两层状态发现策略，使得最终状态划分同时满足：
1. Layer 1 的可解释性保证（每个状态有明确的物理描述）
2. Layer 2 的误差驱动精度（状态内专家可靠性方差最小化）

## AlN v3 实验效果

| 指标 | 单层 | 两层 |
|------|------|------|
| 最大 blob 占比 | 50% | 33% |
| Noise F1 | 0.253 | **0.585** (2.3x) |
| Phonon 隔离 | 混入 bulk | **100% 独立状态** |
