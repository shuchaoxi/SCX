# 数据四分类

## 定义

基于 SCX 框架，数据点被分类为四类：

| 类别 | 特征 | 处理策略 |
|------|------|---------|
| **Valuable** | 高 SCX_m(s)，低冗余，高可学习性 | **优先采集/训练** |
| **Redundant** | 高冗余度，已有充足覆盖 | **压缩/去重** |
| **Noisy** | 低 SCX_m(s) 对所有 m，不可学习 | **排除/清洗** |
| **Expert-Dependent** | 仅部分专家可靠 | **路由到可靠专家** |

## 分类规则

```
if D(s) > threshold_redundant:
    → Redundant
elif max_m SCX_m(s) < threshold_noise:
    → Noisy  
elif SCX 方差 > threshold_expert_dep:
    → Expert-Dependent
else:
    → Valuable
```

## 核心贡献：区分 Noise 和 Hard

这是 SCX 最重要的区分之一。传统 loss-based 方法无法区分：

- **Noise**（标签错误、测量误差）→ 所有专家都预测不好 → 应排除
- **Hard**（模棱两可、分布外）→ 部分专家可能可靠 → 是训练价值最高的数据

SCX 通过**多专家一致性**来区分：如果所有专家在某个状态上都高误差 → 可能是噪声；如果只有部分专家高误差 → 可能是可学习的困难。
