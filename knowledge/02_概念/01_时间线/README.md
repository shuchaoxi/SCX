# 时间线 · 使用说明

按日期组织的开发日志，每天一个文件。每个文件记录：
- 当天产出了什么
- 概念如何演化
- 关键文件的链接
- 前一天/后一天的导航

## 阅读建议

- **想了解思想起源**：从 [[2026-06-14_EGP概念萌芽|6/14]] 开始按顺序读
