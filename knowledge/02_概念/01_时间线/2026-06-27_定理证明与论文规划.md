## 午间：Bug 修复 — 独立验证发现 5 个问题

独立验证（[[决策日志#决策 14：独立定理验证]]）对定理 1 和定理 2 的证明进行了逐行审查，发现并修复了以下问题：

### 定理 1：2 个 bug

1. **Lemma 3 的 Hoeffding → Chernoff 校正**：原证明在 Lemma 3 中使用 Hoeffding 不等式对专家一致检测的假阳性概率进行上界估计。但 Lemma 3 中涉及的是"至少 k 个专家一致"的事件，其尾部行为由大偏差控制而非 Hoeffding 的对称尾。改用 Chernoff 不等式（乘性形式）后，上界从 O(exp(-2Mε²)) 收紧为 O(exp(-M·D(ε∥ρ)))，其中 D 是 KL 散度。这一松紧差异在小样本（M < 30）时可达 2-3 个数量级。

2. **Chernoff 界全局收紧**：更换后不仅修复了 Lemma 3，还发现原定理 1 的 Chernoff 界在全局上可以被统一收紧。核心洞察：Chernoff 乘性版本在大偏差区间的表现优于 Hoeffding 加性版本。全局替换后，定理 1 的保证从"F1 ≥ 1 - 0.05 当 M ≥ 20"提升为"F1 ≥ 1 - 0.03 当 M ≥ 20"。

### 定理 2：3 个 issue

1. **AUC η 定义边界条件**：原定义中 η 在连续情况下的上限未明确处理边界值（η = 1 时完美排序退化为常数）。修正为 η ∈ (0, 1] 的开区间上限处理：当 η = 1 时退化到 trivial bound。

2. **k-means 稳定性论证缺口**：在 Lemma 1 (Fano) 到主定理的桥接中，k-means 聚类的稳定性论证存在隐含假设——假设簇的直径上界已知。补充了 Talagrand 不等式对 k-means 经验风险的浓度估计，填补了论证缺口。

3. **Cluster balance 条件调整**：为处理非平衡簇场景，原证明要求各簇大小均匀。修正后引入"最大-最小簇比"参数 γ = max|C_k| / min|C_k|，将 bound 调整为含 γ 的显式形式：F1 ≤ baseline + C_F √(γδ)。非平衡场景（γ ≥ 10）时 bound 会松弛一个因子 ~3，但仍在可控范围。

### 验证方法论

独立验证采用"白盒审查 + 对抗性测试"双轨并行：
- 白盒：每步假设→推导→结论的逐行逻辑审查
- 对抗性：对每个 bound 构造极端参数测试其是否自洽（如 M = 1, δ = 0, δ = 1, ρ = 0.5 等）

**教训**：独立验证发现了作者/实现者自身的认知盲点——特别是 Lemma 3 的 Hoeffding/Chernoff 误用，这是经典"看起来对但实际上不够紧"的错误。这验证了[[决策日志#决策 14：独立定理验证]]决策的正确性。

## 午间：Theorem 3 — 噪声不可辨识性

**类型**：不可能性定理（负向保证）

**核心思想**：当不访问状态信息 s(x) 时，数据中的噪声模式本质上是不可辨识的——具体地说，存在不同的噪声分布和真实标签的组合，在观察层面完全等价。

**形式化陈述**：
```
令 D 为在特征空间 X × {0,1} 上的噪声标注分布，
对于任意噪声率 ρ_1(x) ≠ ρ_2(x) 且满足边际一致性条件，
存在数据集 (X, Y) 使得无状态信息时无法区分两者。
```

**证明策略**：
1. 构造两个不同的噪声机制 ρ_1, ρ_2，它们在观察分布 P(Y|X) 上等价
2. 利用条件概率的链式分解：P(Y|X) = Σ_s P(Y|X,s)P(s|X)
3. 构造不同的 P(s|X) 和 P(Y|X,s) 组合，保持 P(Y|X) 不变
4. 结论：无 s(x) 信息时，对噪声结构的推断必然依赖强假设

**与定理 1+2 的关系**：

```
Theorem 1: 有状态信息 → 噪声可被检测
Theorem 2: 弱特征 → 检测必然失败
Theorem 3: 无状态信息 → 噪声根本不可辨识  
                ↓
共同叙事：SCX 的必要性论证
  1. 噪声需要被处理（动机，已有共识）
  2. 无状态信息则噪声不可辨识（Thm 3，新洞察）
  3. 有状态信息且特征强时噪声可检测（Thm 1，正向保证）
  4. 特征弱时诚实告知失败（Thm 2，反向边界）
```

**贡献定位**：定理 3 在 SCX 论文中扮演"必要性论证"角色——证明 SCX 的 state-conditioned 方法不是锦上添花的选择，而是在理论上必要的。删除状态信息后，任何噪声处理方法都必然依赖不可验证的假设。

## 午间：Proposition 1'+3' 重构

旧命题系统在 v0.3.0 中有 6 个命题，其中 Proposition 1/3/4 需要在新定理框架下更新。

### Proposition 1'：State-Conditioned Noise Detection

**原命题 1**：状态条件标注优于无条件聚合。
**重构**：不再作为独立命题存在，而是被吸收为 Theorem 1 的推论 1.1。

**重构理由**：原命题 1 的证明依赖于启发式论证（标注准确率比较），而 Theorem 1 提供了严格的 F1 下界。推论 1.1 的形式更精确：

```
给定 Theorem 1 条件，状态条件聚合的 F1 ≥ 无条件聚合的 F1 + Δ(M, η)
其中 Δ(M, η) = (1 - η) · exp(-2MΔ²) > 0
```

### Proposition 3'：SCX 数据价值排序

**原命题 3**：SCX 可靠性分数可有效排序数据价值。
**重构**：整合了 Theorem 2（弱特征时排序退化为随机）和新的 StateValue 架构。

**重构理由**：原命题 3 仅断言"排序有效"而未说明何时失效。重构后：

```
若特征 φ 满足 δ-弱特征条件 (I(φ;s) ≤ δ)，则
Spearman rank correlation(SCX_rank, true_value_rank) ≤ ρ_max(δ)
其中 ρ_max(δ) → 0 当 δ → 0
```

**意义**：重构后的 Proposition 3' 与 Theorem 2 形成"同一硬币的两面"——Theorem 2 是 F1 界，Proposition 3' 是排序质量界。

## 深水区理论探索与审查

核心定理完成后的剩余时间，向三个深水区理论方向发起快速冲锋。每个方向写了完整证明，然后经独立审查全部被 reject。

### 1. Minimax 下界 — Reject

**尝试**：在 [[../../theory/explorations/minimax_lower_bound_proof]] 中尝试用 Le Cam 两点法证明 F1 下界的速率最优性。

**审查结论** ([[../../theory/explorations/review_minimax_lower_bound|审查报告]])：
- **Slud 不等式数值错误**：证明中使用的 Slud 不等式下界在 M=5, p=0.1 时数值上被直接证伪（计算得 1.75 > τ(0.1)），导致整个 lower bound chain 失效
- **F1 推导代数错误**：从分类错误率到 F1 的转换中，分母出现负值（当 η > 1/2 时 2-η < 1.5，代入后 2-1/(1-η) 在 η > 0.5 时为负），造成 F1 bound 代数无意义
- **核方法 gap**：F1 的分子（TP）和分母（TP+FP）依赖于核方法的泛化误差，但现有文献中核方法在 SCX 耦合结构下的泛化界尚未建立

**可修复性**：是。需要 (a) 替换 Slud 为数值精确的 Berry-Esseen 或 Chernoff 下界，(b) 重做 F1 的 TP/FP 分解，(c) 建立耦合结构下的核泛化界。估计 7-10 周。

### 2. 聚类一致性 — Reject

**尝试**：在 [[../../theory/explorations/cluster_consistency_proof]] 中尝试推广 Pollard (1981) k-means 一致性到 SCX 两层架构。

**审查结论** ([[../../theory/explorations/review_cluster_consistency|审查报告]])：
- **bound 发散（致命）**：核心 bound 中出现 1/(1-K/m) 项，当 K > m/2 时分母为负、K=m 时发散。这是代数推导错误——声称"足够大的 m 可使分母为正"，但当 bound 本身包含 m 时，这个论证循环
- **k-means NP-hard 假设间隙**：证明假设 k-means 能在大规模数据 (n→∞, K→∞) 下找到全局最优解——但在 NP-hard 条件下，即使 n→∞ 也无法保证
- **修正版 bound 依然发散**：修正后第 2 个 bound 出现 K/c 项，当 K 增长时分母比值趋于 0，bound 仍发散

**可修复性**：是。需要 (a) 引入 k-means++ 初始化或谱聚类的近似比保证，(b) 重做无发散的 bound 推导，(c) 可能需要修改 K 的 scaling 条件。估计 2-4 周。

### 3. BBP 谱代理 — Reject

**尝试**：在 [[../../theory/explorations/bbp_spectral_proxy]] 中尝试用 Gram 矩阵最大特征值替代互信息 δ 作为弱特征诊断的可计算代理。

**审查结论** ([[../../theory/explorations/review_bbp_spectral_proxy|审查报告]])：
- **互信息 ≠ 最大特征值**：核心论证 I(φ;s) = log(λ₁) - log(λ_{MP}) 成立仅当 φ 的协方差矩阵的低秩结构完全由 s 决定——这在一般情况下不成立。最大特征值可能反映了与 s 无关的其他结构（如样本伪影、维度相关）
- **高斯假设致命**：证明依赖 φ|s 为高斯分布的假设。当 φ|s 是多模态、heavy-tailed 或离散时（这在 MLIP 中常见），Marchenko-Pastur 的相位过渡阈值不再适用，整个代理失效
- **Novelty 不足**：即使代理成立，Lei & Zhu (2018) 等多篇论文已建立谱聚类+特征值阈值方法。增量贡献不够支撑一篇论文

**可修复性**：是但更需要重新思考方向。需要 (a) 放弃"替代 δ"的强主张，改为"谱诊断+互信息"双通道，(b) 处理非高斯→需要更一般的随机矩阵结果或经验谱分析方法。估计 3-4 周。

### 4. 教训总结

三个方向全部 reject 不是偶然——暴露出深水区理论开发的系统性问题：

| 问题 | 表现 | 修复策略 |
|------|------|---------|
| **推导节奏太快** | 代数错误本可以在慢速推导中发现 | 深水区需要"证明→休息→重推→验证"的周期 |
| **假设审查不严** | Slud 适用条件、NP-hard 假设、高斯假设，都是事后才发现的明显漏洞 | 推导前先做假设清单 + 边界条件测试 |
| **缺乏数值 sanity check** | Slud 不等式数值错误如果在推导当时计算就不会出现 | 每条 bound 先代入极端参数算数值 |
| **低估 SCX 耦合结构的难度** | 标准工具（Le Cam, Pollard k-means, BBP）在耦合结构下都失效 | 需要先开发/理解耦合结构下的工具，而非直接套用经典结果 |

**决策**：深水区理论全部延期。Paper 1 投稿后、等 reviews 期间是最佳恢复窗口。

---

### 收盘前 v2 轮：全部修复/转向 ✅

首轮 reject 之后，在收盘前进行了第二轮冲刺——修复所有致命错误。

#### Minimax 下界 v2：Resurrected

**文件**：[[../../theory/explorations/minimax_lower_bound_v2]]（1002+ 行）

**核心变更**：
- 弃用 **Slud 不等式**（数值上被证伪）→ 改用 **Bretagnolle-Huber 不等式**（Hellinger 距离版本）。Bretagnolle-Huber 在稀疏信号（M 小、p 小）下保持紧致，且不需要对称性假设。
- 修复 **2 个 F1 推导 bug**：一个是 FP 的非对称缩放（v1 中未对 FP 做状态条件分解导致高估），一个是核方法泛化界的耦合结构处理错误。

**结果**：核心下界证明通过独立审查。`liminf (-1/M) log(lower_bound) = 2Δ²` — 证明指数 2MΔ² 是 minimax 最优的。

**关键教训**：F1 问题中 Hellinger 距离优于总变差——Hellinger 的乘法结构更适合处理 F1 的比值形式。

#### 聚类一致性 v2：Rewritten

**文件**：[[../../theory/explorations/cluster_consistency_v2]]（749 行）

**核心变更**：
- v1 的致命错误：bound 出现 1/(1-K/m) 项，K > m/2 时分母为负。根因是**同时让 n 和 K 增长**——高维渐近的边界条件在 SCX 耦合结构下不成立。
- v2 修正：**固定 K**（K 为常量）。固定-K 的 k-means 一致性是 Pollard (1981) 经典结果的直接推广。
- **Pollard-style 4 引理结构**：(i) 经验风险一致性、(ii) 参数空间紧致性、(iii) 唯一辨识性、(iv) 全局最优性保证。
- **Lloyd's + random restarts**：放弃 NP-hard 假设，证明 Lloyd's 结合多项式次 random restarts 能以高概率找到 ε-近似解。

**结果**：修复后证明在固定-K 设定下完全成立。增长-K 放松可留作未来工作。

#### BBP 谱代理 v2：Pivoted to Stability Diagnostic

**文件**：[[../../theory/explorations/feature_strength_via_stability]]

**核心变更**：
- v1 审查确认核心桥接 I(φ;s) = log(λ₁) - log(λ_{MP}) 在一般情况下不成立。**信息-谱桥接断裂**——互信息与 Gram 矩阵最大特征值之间不存在通用不等式。
- v2 转向：**Bootstrap ARI 聚类稳定性诊断**。对特征 φ 进行 Bootstrap 重采样，对每个重采样样本运行聚类，计算 ARI 在 Bootstrap 样本间的稳定性。稳定 ARI → 强特征；不稳定 → 弱特征。
- **无分布假设**：Bootstrap ARI 对 φ 的分布类型完全无要求，适用于 SOAP/MTP（多模态/heavy-tailed）等 MLIP 特征。
- **可计算性**：产生 0-1 的数值指标，替代 Thm 2 中不可计算的 δ = I(φ;s)。

**对 Paper 3 的价值**：提供一个可计算、可验证的弱特征诊断方案，且不依赖任何光谱假设或分布假设。

### v2 轮关键教训汇总

| v1 的致命错误 | v2 解决方案 | 一般性教训 |
|:--------------|:-----------|:-----------|
| Slud 不等式数值错误 | Bretagnolle-Huber (Hellinger) | **Hellinger 优于 TV**：比值型风险（F1）需要乘法工具 |
| bound 发散（增长-K） | 固定 K，Pollard-style | **经典工具够用**：Pollard 固定-K 已足够，无需强行增长-K |
| 信息-谱桥接断裂 | Bootstrap ARI 稳定性 | **不要用谱方法绕过信息论**：谱假设在实践数据中不成立 |

三个方向的共同模式：**"直接套用经典工具"→ reject → "定制化方法"→ 通过**。这确认了一个重要的元教训：SCX 的耦合结构要求每个工具都要先在 SCX 环境下重新检验适用性。
