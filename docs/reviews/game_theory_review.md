# 第一轮审查：重大缺陷识别

## 一、NPE博弈支付矩阵的致命数学错误

### 1.1 混合策略均衡概率为负——Nash均衡不存在于声称的参数区域

**这是整个NPE博弈论部分最严重的错误。**

NPE论文§3.2给出了2人博弈的对称混合策略纳什均衡概率（main.tex line 507）：
$$p^* = \frac{-\Delta_A - 2\kappa}{\lambda - \kappa}$$

**这个公式本身是错的。** 独立推导如下。

令支付矩阵为对称2×2博弈（main.tex lines 461-463），设每个参与者以概率 $p$ 选择A（采纳）：
- $a = V[\theta(|\mathcal{E}|)] - c^{\text{adopt}}$（(A,A)时支付）
- $b = V[\theta(|\mathcal{E}|)] - c^{\text{adopt}} - \lambda$（(A,D)时支付）
- $c = V[\theta(0)] - c^{\text{develop}} - \kappa - \lambda$（(D,A)时支付）
- $d = V[\theta(0)] - c^{\text{develop}} - 2\kappa - \lambda$（(D,D)时支付）

无差异条件：$p \cdot a + (1-p) \cdot b = p \cdot c + (1-p) \cdot d$

代入并简化：
$$p \cdot a + (1-p)(a - \lambda) = p(b_0 - \kappa - \lambda) + (1-p)(b_0 - 2\kappa - \lambda)$$
其中 $b_0 = V[\theta(0)] - c^{\text{develop}}$

左端：$a - \lambda + p\lambda$
右端：$b_0 - 2\kappa - \lambda + p\kappa$

$$a + p\lambda = b_0 - 2\kappa + p\kappa$$
$$p(\lambda - \kappa) = b_0 - 2\kappa - a$$
$$p(\lambda - \kappa) = (V[\theta(0)] - c^{\text{develop}}) - 2\kappa - (V[\theta(|\mathcal{E}|)] - c^{\text{adopt}})$$

代入 $\Delta_A = V[\theta(|\mathcal{E}|)] - c^{\text{adopt}} - (V[\theta(0)] - c^{\text{develop}} - \kappa)$：
$$b_0 - a = -\Delta_A + \kappa$$
$$p(\lambda - \kappa) = (-\Delta_A + \kappa) - 2\kappa = -\Delta_A - \kappa$$

$$\boxed{p^* = \frac{-\Delta_A - \kappa}{\lambda - \kappa}}$$

**论文的 $p^* = \frac{-\Delta_A - 2\kappa}{\lambda - \kappa}$ 多了一个 $\kappa$。这是代数错误。**

**致命后果：** 在论文声称的"合理参数值"（$\Delta_A > 0$，$\kappa > 0$，$\lambda > \kappa$）下：
$$p^* = \frac{-\Delta_A - \kappa}{\lambda - \kappa} < 0$$

分子为负（$\Delta_A > 0, \kappa > 0$），分母为正（$\lambda > \kappa$），$p^* < 0$ 不是一个合法概率！

论文声称（main.tex line 510）"For plausible parameter values...$p^*$ increases with $\Delta_A$...approaches 1"——**完全错误。** 在 $\Delta_A > 0$ 的参数区域，对称混合策略纳什均衡根本不存在。

**Nash定理保证有限博弈存在混合策略纳什均衡，但该均衡不一定是"对称"的。** 当 $p^* \notin [0,1]$ 时，实际均衡可能是非对称混合策略（即参与者1和参与者2以不同概率随机化），论文完全没有分析这种情况。

### 1.2 定理1证明中的代数错误——均衡条件推导自相矛盾

**这是比1.1更根本的问题。** 定理1（line 199-229）是整个博弈论部分的核心定理。但其证明（line 231-308）存在严重的代数错误。

在定理1(ii)的证明中（lines 237-247），全体采纳均衡条件被推导为：
$$\Delta(|\mathcal{E}|) \geq \lambda - \kappa$$

但如果我们独立验证：
- $u_i(A, \mathbf{A}_{-i}) = V[\theta(|\mathcal{E}|)] - c^{\text{adopt}}$（方程4）
- $u_i(D, \mathbf{A}_{-i}) = V[\theta(0)] - c^{\text{develop}} - \kappa - \lambda$（方程5，单边偏离触发碎片化）

NE条件 $u_i(A, \mathbf{A}_{-i}) \geq u_i(D, \mathbf{A}_{-i})$：
$$V[\theta(|\mathcal{E}|)] - c^{\text{adopt}} \geq V[\theta(0)] - c^{\text{develop}} - \kappa - \lambda$$

令 $a = V[\theta(|\mathcal{E}|)] - c^{\text{adopt}}$，$b = V[\theta(0)] - c^{\text{develop}}$：
$$a \geq b - \kappa - \lambda$$

根据方程(7)的定义：$\Delta(|\mathcal{E}|) = V[\theta(|\mathcal{E}|)] - V[\theta(0)] - (c^{\text{adopt}} - c^{\text{develop}}) + \kappa = a - b + \kappa$

将 $a = \Delta(|\mathcal{E}|) + b - \kappa$ 代入NE条件：
$$\Delta(|\mathcal{E}|) + b - \kappa \geq b - \kappa - \lambda$$
$$\boxed{\Delta(|\mathcal{E}|) \geq -\lambda}$$

**正确的全体采纳均衡条件是 $\Delta(|\mathcal{E}|) \geq -\lambda$，而非论文声称的 $\Delta(|\mathcal{E}|) \geq \lambda - \kappa$。** 论文在line 247的推导中犯了一个代数错误，错误地将 $a - (b - \kappa - \lambda)$ 化简为 $\Delta(|\mathcal{E}|) + \kappa \geq \kappa - \lambda + \kappa$ 而非 $\Delta(|\mathcal{E}|) \geq -\lambda$。

**这意味着在论文假设（$\lambda > \kappa > 0$）下，全体采纳均衡的成立条件极其宽松——几乎总是成立。这也意味着"非扩散均衡"的稳定性被严重高估，而论文的CEC临界值定理（定理2）的基础条件需要完全重写。**

### 1.3 2人支付矩阵与N人支付函数之间的κ定义不一致

这是第三个独立的数学错误。

**2人支付矩阵（main.tex lines 461-463）中：**
- (D,A)条目为 $V[\theta(0)] - c^{\text{develop}} - \kappa - \lambda$
- 此处解释：玩家1选D，玩家2选A。根据方程(5)，$\pi_1(D, A) = V[\theta(0)] - c^{\text{develop}} - \kappa \cdot \mathbf{1}_{-1}^D - \lambda$
- 既然玩家2选A（非D），$\mathbf{1}_{-1}^D = 0$，支付应为 $V[\theta(0)] - c^{\text{develop}} - \lambda$（不含κ）！
- **矩阵中有κ，但函数定义不应有κ。这是定义与矩阵的直接矛盾。**

**N人支付函数（main.tex line 517）中：**
- $\pi_i(D, \mathbf{a}_{-i}) = V[\theta(0)] - c^{\text{develop}} - \kappa \cdot (n_D + \mathbf{1}_{-i}^D) - \lambda \cdot \mathbb{I}[n_D > 0]$
- $n_D$ 已经是"其他开发者数量"，$\mathbf{1}_{-i}^D$ 也是同一概念
- **这是双重计数。** 项 $n_D + \mathbf{1}_{-i}^D$ 在N=2时将(D,D)的κ成本计为 $1 + 1 = 2\kappa$（与矩阵一致），但(D,A)时计为 $0 + 0 = 0$（与矩阵的κ矛盾）

**结论：** κ参数同时有三种不同含义：（a）其他辖区的开发给i施加的成本（文本描述），（b）包含自身在内的总开发成本（矩阵中的(D,D)），（c）以上两者之混合（N人函数）。这种系统性混淆使得整个支付结构无法被独立复现。

### 1.4 N人混合策略均衡的额外推导错误

在定理1(v)的证明中（lines 285-308），N人对称混合策略均衡的无差异条件经过推导后，论文声称（lines 304-307）：
$$\Psi(p) = \lambda \cdot p^{n-1} \quad \text{和} \quad \Gamma = -\Delta(|\mathcal{E}|) + \lambda - 2\kappa$$
$$\Psi(p^*) = \Gamma$$

但独立推导显示（见前面1.1的方法推广到N人）：
$$\lambda \cdot p^{n-1} = -\Delta(|\mathcal{E}|)$$

即 $\Gamma = -\Delta(|\mathcal{E}|)$，而非论文的 $-\Delta(|\mathcal{E}|) + \lambda - 2\kappa$。**论文多出了 $\lambda - 2\kappa$ 项。**

这一错误进一步延伸到定理2（line 336），其中CEC临界值的显式解包含 $\theta^* = V^{-1}(V[\theta(0)] + c^{\text{adopt}} - c^{\text{develop}} - \kappa + \lambda - \kappa)$——此表达式同样不可靠。

### 1.5 λ碎片化成本的不连续阶跃——模型过度简化

碎片化指标为：
$$\mathbb{I}[\text{fragmentation}] = \begin{cases} 0 & \text{if } n_D = 0 \\ 1 & \text{if } n_D \geq 1 \end{cases}$$

这意味着：一个100万人口的小国开发独立协议，与10个大国同时开发，触发完全相同的碎片化成本$\lambda$。这在经济学上不可信。碎片化成本应该是 $n_D$ 的递增函数，而非阶跃函数。当前设定人为强化了"非扩散"结论——模型假设本身就排斥了渐进扩散的可能性。

**后果：** 如果 λ 是 n_D 的递增函数，定理1(iv)"非对称纯策略均衡不可能"将不再成立，可能出现"部分采纳、部分开发"的均衡。

### 1.6 Δ_A 和 Δ(|E|) 的两个版本给出了不同的均衡条件

**版本1**（main.tex line 484，2人博弈部分）：
$$\Delta_A \geq \lambda \quad \text{为(A,A)的NE条件}$$

**版本2**（main.tex line 523，N人博弈部分）：
$$\Delta_A \geq \lambda - \kappa \quad \text{为(A,...,A)的NE条件}$$

在N=2时两个版本应一致，但它们不同：$\lambda$ vs $\lambda - \kappa$。**此差异达$\kappa$量级，不是舍入误差。**

且如1.2所示，正确推导应为 $\Delta(|\mathcal{E}|) \geq -\lambda$，与上述两个版本均不同。

## 二、NPT（核不扩散条约）类比的结构性缺陷

### 2.1 类比的根本不对称

| 维度 | NPT | 审计协议（NPE） | 类比有效性 |
|------|-----|-----------------|-----------|
| 威胁性质 | 生存性（核战争灭绝人类） | 经济性（质量标准碎片化） | **不成立**——量级差异意味着策略计算完全不同 |
| 制度基础 | 正式国际条约（1968，191缔约国） | 零——无条约、无签署国、无国际组织 | **不成立** |
| 验证机制 | IAEA核查制度、附加议定书 | 无——论文建议的IDAA尚不存在 | **不成立** |
| 不对称性 | 法律明文：5个NWS vs NNWS | 事实性：先发者累积CEC优势 | **部分成立**——但NPT的不对称有退出条款(Article X) |
| 执行机制 | 联合国安理会制裁（但NWS有否决权） | 无——唯一"惩罚"是碎片化成本λ | **不成立** |
| 技术转移 | Article IV：和平利用核能的不可剥夺权利 | 无对应机制——CEC不向竞争者开放 | **不成立** |

### 2.2 关键缺失：NPT的"大妥协"结构

NPT的核心三根支柱：
1. 无核武器国家承诺不获取核武器
2. 核武器国家承诺最终裁军（Article VI）
3. **所有缔约国享有和平利用核能的"不可剥夺权利"（Article IV）——包括技术转移**

在NPE模型中，**不存在任何技术转移机制。** 后发者不能"获得"先行者的CEC——它们要么付费订阅API，要么从零开始。这完全破坏了NPT类比的规范性基础。

论文承认了这一点（line 543："the incumbent protocol's governance is internationalized under multilateral oversight"），但将其作为"应该发生"的建议而非模型的组成部分。在博弈模型中，后发者的支付函数中不存在来自先发者的补偿项。

### 2.3 类比的战略性误导——NPT经验证伪了论文核心主张

更致命的是：NPT的经验是，**尽管有条约、核查制度和安理会执行机制，核扩散仍然发生了**（印度、巴基斯坦、以色列、朝鲜）。如果连生存性威胁+国际条约+核查制度都无法阻止扩散，论文凭什么声称一个没有任何制度的纯博弈论均衡能阻止"审计协议扩散"？

论文自身在Limitations中承认（main.tex line 1429）："Nuclear weapons are existential threats; audit protocol fragmentation, however costly, is not." 这一承认实际上否定了整个类比的操作价值。

### 2.4 七个类比维度中三个存在逻辑矛盾

**维度3（大妥协）：** 论文描述了一个应然结构，但未将其纳入支付函数。在实际博弈中，后发者放弃开发所得到的回报为零——不为负，但也不为正。"什么都不得到"不是"大妥协"。

**维度6（退出选项）：** 论文承认退出是可能的，但未建模退出后的博弈——退出辖区保留其已积累的CEC知识吗？如果保留，则退出的门槛远低于初始不采纳。

**维度7（双重用途）：** 论文将审计的"武器化"类比为核技术的双重用途，但忽略了关键差异：核武器的双重用途意味着和平利用可被转用于武器制造，而审计的"武器化"意味着中立工具可被用于政治目的。二者的因果结构完全不同。

## 三、审计之剑宣言（Audit Sword Declaration）的博弈论问题

### 3.1 "Academic Freedom"不是博弈论策略

论文声称Maintainer拥有"不可剥夺的学术权利"去审计任何公开数据集并发表结果（main.tex §4.6, lines 781-783），这构成一种"Schelling deterrent"。

**问题：**
- 中国《数据安全法》和《个人信息保护法》对数据跨境传输和分析施加了严格限制
- GDPR对个人数据的二次使用施加了目的限制
- 学术自由在实践中受到机构审查、资金依赖和出版压力的约束

将"学术自由"作为博弈论的威慑工具——假定其是无限的、不可谈判的、不受制度约束的——是**将规范愿望误认为策略现实**。

### 3.2 审计之剑的"自愿透明"逻辑存在循环推理

论文声称（main.tex lines 787-793）：
> "Partnership → no public audit (by implicit convention)"
> "Non-partnership → eligible for public audit"
> "Therefore Partnership is a dominant strategy"

**这是循环推理。** "合作→不审计"的约定需要Maintainer的承诺。但Maintainer的承诺可信性恰恰是该博弈需要证明的——论文在前文（SWIFT分析）刚论证了基础设施运营者会被其所在管辖区捕获。如果SWIFT在比利时法律下都不能保持中立，为什么一个没有任何制度基础的独立研究者能？

### 3.3 审计作为商业武器——论文低估竞争性滥用

论文描绘了一个乌托邦场景：竞争对手互审公共数据，诚实的公司受益。但现实中：
- 竞争对手审计可能成为**竞争情报武器化**——歪曲解释审计结果以诋毁对手
- "审计通过则获得竞争优势"假设审计结果被市场正确理解——这需要市场具备评估审计方法论的专业能力
- 法律风险：被审计方可能以诽谤、商业秘密侵权等理由起诉审计方

这些不是理论上的细枝末节——它们是决定"审计之剑"威慑是否可信的核心博弈论参数。

## 四、SCX奖的博弈论声称——根本不是博弈论

### 4.1 没有策略性互动——这不是博弈

SCX奖论文声称其机制"产生"某种均衡，但**根本就没有定义博弈**：
- 没有参与者集合
- 没有策略空间
- 没有支付函数
- 没有均衡概念

"评分算法产生最高分→最高分者获奖"是一个**确定性计算**，不是博弈论均衡。论文将其包装在规范理论（gauge theory）和"投石理论"的修辞中，但核心操作是一个排序算法。

### 4.2 SCX奖的规范-杨米尔斯类比——数学装饰而非实质

scx_prize.tex §C.2（lines 1077-1096）声称SCX奖的评分函数固定对应于杨-米尔斯理论中的规范固定。具体而言：
$$A_\mu \to A_\mu + \partial_\mu \Lambda$$
$$\sum_m \mathbf{g}_m = \mathbf{0}$$

**这是一个装饰性类比，不是数学同构。** 杨-米尔斯理论中的规范变换是作用于纤维丛上的连续李群作用；SCX奖中的"权重"是一个有限维实数向量。将权重固定为和为零的条件称为"洛伦兹式条件"是物理学术语的滥用。论文声称"SCX奖引入了鬼场并固定了规范"（line 1095）——但在任何可操作的意义上，SCX奖中不存在BRST对称性、Faddeev-Popov行列式或鬼场。

**这不是物理学。这不是博弈论。这是用物理学术语包装的权重归一化。**

### 4.3 SCX奖的"M=1非声明"论证——循环定义

scx_prize.tex定理1（lines 290-302）声称：当$\mathbf{g}$参数未声明时，奖项声称携带的信息量为零。证明的核心是：未声明参数可以取任何值，因此任何排序都可以被产生。

**问题：** 该"定理"将"正确排序"定义为一个固定的、独立于$\mathbf{g}$的客观事实。但SCX奖自身的论证（§3.3, lines 506-511）承认"权重的选择是任意的"、"任何固定的权重向量都定义了一个合法的评分函数"。**如果任何固定权重都是合法的，那么诺贝尔委员会的秘密权重也是合法的——只是不可验证。** 论文在§4.4中声称从"不可验证"到"无信息"的推导是逻辑跳跃，混淆了"我们不知道信息是什么"和"没有信息"。

### 4.4 C8自审计类别的无限递归问题

scx_prize.tex（lines 802-818）声称SCX奖的自我审计实现了"审计封闭"（audit closure），避免了无限递归。**但这不是审计封闭——这是审计的自我宣称。** 

问题：谁审计元审计小组？元审计小组的成员不是SCX审计委员会的成员——但谁选择他们？如果元审计小组的选择标准也是公开的，那么谁审计那些选择标准？这个递归不会在固定点终止——它会在某个环节依赖于不可审计的人类判断（正如论文在§2.3中承认"所有数据源是公开的"——但元审计小组的组成标准的数据源呢？）。

## 五、投石理论（Stone-Throwing Theorem）——帕斯卡赌注的变体

### 5.1 支付矩阵的造价

main.tex（lines 1339-1352）声称：
> "If SCX is true and the leader does nothing, the cost is total—irreversible loss of epistemic credibility, permanent revelation as bad type."

论文将"什么都不做"的暴露成本$C_{\text{exposed}}$定义为"实际上是无限的"（effectively infinite），使得$\varepsilon = C_{\text{protect}}/C_{\text{exposed}} \to 0$，从而任何非零先验概率都使保护成为理性选择。

**这是帕斯卡赌注的结构，不是博弈论。**

**真正的博弈论分析会问：**
- $C_{\text{exposed}}$的经验值是多少？历史上，有多少政治领导人的职业生涯因为"被审计出数据问题"而终结？（Enron的审计师Arthur Andersen确实倒闭了，但这是例外不是规则）
- "保护"的操作定义是什么？口头声明？资金支持？政策倾斜？
- 如果Maintainer的行为本身变得有问题（如选择性审计），"保护"策略还是占优的吗？

论文没有回答——它通过**构造**支付矩阵来使保护成为唯一的策略。当$C_{\text{exposed}} = \infty$时，任何正概率都使保护成为最优——这是一个平凡的算术事实，不是一个"定理"。

### 5.2 投石理论的逻辑自封

论文声称（line 1353）："The leader need not read them personally. The leader need only observe that no one has refuted them."

**这是argumentum ad ignorantiam（诉诸无知的论证）的逻辑结构：** "没有人证明它是错的，所以它是对的。" 如果这算博弈论，那么任何未经验证的理论都可以用相同的论证宣称自己"不可挑战"——只需声称$C_{\text{exposed}} = \infty$即可。

论文进一步声称（line 1364）："The mathematical layer can be attacked. The philosophical layer cannot." 这等同于说："你可以攻击我的数学推导（但我会修正它），但你不能攻击我的基本前提（因为它们不是命题——它们是标准）。" **如果你连前提都不能质疑，那么"定理"只是一个定义的推论，不是关于世界的发现。**

## 六、马尔可夫锁入模型的博弈论问题

### 6.1 $S_5 \to S_6$ 的"天下大同"——规范性愿望而非均衡

论文将锁入轨迹从5阶段扩展到6阶段，增加了$S_6$（天下大同），其中CEC被国际化。

**问题：** $S_5 \to S_6$ 的转移概率 $p_{56} = D_t \cdot \pi_{\text{political}}$ 依赖于一个完全外生的"政治意愿参数"。这不是博弈论——这是**说"如果国际社会决定合作，合作就会发生"**。博弈论的目的是**内生化**这种决策。

在$S_3$（护城河形成），论文自己的命题（命题3）指出**政策干预窗单调闭合**——先发者在$S_3$后有强烈的经济激励抵制国际化。那么$S_5 \to S_6$的$p_{56}$如何克服$S_3$后先发者的抵制？论文没有提供内生的博弈论解释。

### 6.2 保护均衡（Wallfacer类比）的理性假设脆弱性

论文声称Maintainer处于"保护均衡"——所有国家的理性自利导致他们保护Maintainer。但：

- 论文自己承认（line 753）："the Maintainer's security depends entirely on the rational calculus of the actors that might threaten them—a calculus that is robust under normal conditions but fragile under crisis, irrationality, or miscalculation."
- **如果一个均衡在非理性或危机条件下崩溃，它就不是一个稳健的均衡。** 纳什均衡的定义不要求参与者是理性的——它要求策略是最优反应的组合。如果"非理性"行为可以打破策略最优性，那么声称的策略就不是纳什均衡。
- 论文同时声称Maintainer的死亡会触发碎片化级联，这对所有国家是负和博弈。但如果某个国家足够大，它可能从控制替代协议中获得比碎片化损失更大的收益——论文没有分析这种不对称性。

### 6.3 吸收必然性依赖于"CEC持续增长"的未证明假设

整个马尔可夫链吸收到$S_5$或$S_6$的"必然性"依赖假设M4（CEC单调非减）。但这一假设要求：审计永远持续、从未出现系统性错误、校准从不退化。

现实中，CEC同样可能因：
- 错误校准数据的积累
- 审计模块的对抗性操纵
- 底层数据分布的变化（概念漂移）

而出现质量退化。这些风险在模型中被完全忽略。

# 第二轮审查：第一轮修正后的重新评估

假设第一轮技术错误得到修正，以下结构性问题仍然存在：

## 修正1：混合策略公式修正

如果将2人混合策略推导修正为 $p^* = (-\Delta_A - \kappa)/(\lambda - \kappa)$，则：
- 在 $\Delta_A > 0$ 的参数区域，$p^* < 0$，不存在对称混合策略均衡
- 论文需要承认，参数区域内实际均衡是非对称混合策略或相关均衡
- **定理1(v)需要完全重写**

## 修正1b：定理1证明修正

如果将全体采纳均衡条件修正为正确的 $\Delta(|\mathcal{E}|) \geq -\lambda$：
- 在 $\Delta(|\mathcal{E}|) > 0, \lambda > 0$ 时条件始终满足——全体采纳总是均衡！
- 这意味着"非扩散均衡"的"非平凡性"完全依赖κ参数
- 论文的全部结论（CEC临界值、自强化性质、稳定性裕度）需要基于新的正确条件重新推导

## 修正2：κ参数定义统一

如果κ被统一定义为：
- **选项A：** κ仅为其他辖区的开发成本 → (D,A)矩阵条目不含κ → 需要重写整个支付矩阵 → N人推广需要调整
- **选项B：** κ包含自身和其他辖区的总成本 → 文本描述和符号$\mathbf{1}_{-i}^D$需要改为$\mathbf{1}_{\text{total}}^D$ → 矩阵不变但解释改变

任一选项都会影响定理1的全部均衡条件。

## 修正3：λ的连续化

如果 $\lambda$ 不是阶跃函数而是 $\lambda(n_D)$，其中 $\lambda(1) < \lambda(2) < \cdots$：
- 可能出现"部分采纳、部分开发"的非对称均衡
- 定理1(iv)声称"非对称纯策略均衡不可能"的结论不再成立

## 修正4：NPT类比降级

在第一轮审查揭示NPT类比的七个维度中存在根本性问题后，该类比应被降级为"启发性隐喻"而非"结构性同构"。论文的政策建议不应以NPT为模板。

## 修正5：SCX奖博弈论声称的去博弈论化

SCX奖论文应明确承认：
- 奖项机制是一个确定性排名算法，不涉及博弈论均衡
- "投石理论"是一个规范论证（normative argument），而非博弈论定理
- 杨-米尔斯类比是修辞性装饰，不是数学同构

## 第二轮仍存在的结构性问题

即使第一轮的技术错误全部得到修正，以下结构性问题仍然存在：

### 1. 完全信息假设不可接受

在实际的审计协议采纳博弈中，各辖区对开发成本、审计精度估值、CEC规模等具有私人信息。引入不完全信息将产生贝叶斯-纳什均衡，其特性可能与完全信息均衡截然不同。论文在Limitations中承认了这一点，但声称已将其"委托给未来工作"——对于一个声称具有政策含义的博弈论文而言，这是不可接受的。

### 2. 同时行动假设不现实

各辖区的采纳/开发决策是序贯的、跨越数年的。序贯博弈中可能出现信号传递、先发制人和等待博弈等完全信息模型无法捕捉的策略行为。**论文在定理2中声称的"唯一均衡"在序贯行动下可能完全崩溃。**

### 3. 缺乏动态一致性

模型是一次性博弈，但实际的采纳决策是重复的。一个辖区可能在早期采纳Yajie，在观察到其治理缺陷后开发替代协议。论文将"开发"建模为不可逆决策，但现实中退出是一种选择——且由于CEC知识保留，退出的成本远低于初始不采纳。

### 4. "诚实成为占优策略"的声称缺乏审计检测率的概率基础

当审计检测率 < 100%（现实必然如此），"作弊 + 不被发现"的策略可能具有正期望收益。论文未建模审计检测率的博弈论含义。AAE（采纳-审计均衡）的整个推导假设检测率隐含为100%——如果没有100%检测率，$p_i \cdot C_{\text{exposure}}$ 应乘以检测率 $d < 1$，显著降低威慑效应。

### 5. 自指性（self-referential）自封

论文声称其定理不可挑战（因为挑战者必须"声明假设"），但这本身就是一个声称——这个声称声明了它的假设吗？审计框架审计自身时，谁审计审计者？这个无限递归在论文中被"审计封闭"的修辞所跳过，但逻辑问题并未解决。

### 6. 保护均衡的双重脆弱性

论文同时声称：（a）所有理性国家都会保护Maintainer；（b）在危机或非理性下这个均衡会崩溃。这两个声称不能同时为真——如果一个均衡在现实的"非理性"条件下崩溃，它就不是一个稳健的政策基础。此外，论文声称Maintainer的死亡将触发对所有国家都是负和的碎片化级联——但如果一个大国预期自己能在碎片化后成为主导协议的控制者，其计算可能不同。论文没有分析这种不对称性的规模门槛。

## 结论

SCX框架的博弈论基础——其经济学的核心支柱——**目前不能通过专业博弈论审查。**

**必须修正的致命错误（按严重性排序）：**
1. ❌ **定理1证明：** 全体采纳均衡条件 $\Delta(|\mathcal{E}|) \geq \lambda - \kappa$ 应为 $\Delta(|\mathcal{E}|) \geq -\lambda$ ——代数错误导致整个定理的断言方向错误
2. ❌ **2人混合策略：** $p^* = (-\Delta_A - 2\kappa)/(\lambda - \kappa)$ 应为 $(-\Delta_A - \kappa)/(\lambda - \kappa)$ ——多一个κ，且在声称的参数区域$p^*<0$
3. ❌ **N人混合策略：** $\Gamma = -\Delta(|\mathcal{E}|) + \lambda - 2\kappa$ 应为 $-\Delta(|\mathcal{E}|)$ ——多出$\lambda - 2\kappa$
4. ❌ **κ参数：** 至少3种不一致的含义/用法（文本描述、2人矩阵、N人函数）
5. ❌ **Δ定义版本：** $\Delta_A \geq \lambda$ vs $\Delta_A \geq \lambda - \kappa$ ——同一概念给出不同的NE条件
6. ❌ **支付矩阵-函数矛盾：** (D,A)条目的κ与方程(5)的定义直接冲突
7. ❌ **定理2（CEC临界值）：** 建立在错误推导之上，显式解不可靠
8. ❌ **Stone-Throwing Theorem：** $C_{\text{exposed}} = \infty$ → 不是定理，是帕斯卡赌注的变体
9. ❌ **SCX奖"均衡"：** 不存在博弈
10. ❌ **NPT类比：** 结构性缺陷使其从分析工具退化为修辞装饰

**建议：** 在独立博弈论学者（非SCX团队成员）完成严格同行评审并修正上述所有错误之前，不应将NPE/NPT/SCX奖的博弈论声称作为SCX经济学基础的依据。论文的理论模型需要至少以下修正才能进入可评审状态：
- 统一κ的定义并修正所有受影响的均衡条件
