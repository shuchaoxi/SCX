## 修复 1：Theorem 6 — 静态博弈→动态重复博弈（关键数学错误）

**严重程度**: 🔴 关键  
**位置**: 原第 1011–1023 行（Zero-Sum Bias Equilibrium）  
**问题**: 定理声称"在静态互相审计博弈中，唯一的对称纯策略纳什均衡是 g_i=0"。这是**数学上错误的**——对称策略 g_i=g≠0（所有维护者使用相同非零偏差）在静态博弈中**也是**纳什均衡，因为所有分数同步偏移，共识被污染，检测失效。证明的 Step 3 实际上承认了这一点。

**修复**:
- 将定理重命名为 "Zero-Sum Bias Equilibrium under Rotation"
- 显式添加 **Static caveat**：承认对称 g≠0 在静态博弈中也是 NE
- 重新表述为**动态重复博弈**：在有限轮换下，g_i=0 是唯一的子博弈完美均衡（subgame-perfect equilibrium）
- 更新中文翻译，加入静态警示和动态解决的说明
- 同时修复了中文翻译中的 `\\sup_g` LaTeX 转义错误

**修复后定理结构**:
```
静态警示: 对称 g≠0 在一次性博弈中也是 NE
动态解决: 在有限轮换重复博弈中，g_i=0 是唯一子博弈完美均衡
```

## 修复 2：Theorem 2 — 串通者污染共识基线（关键数学错误）

**严重程度**: 🔴 关键  
**位置**: 原第 586–607 行（Consensus Deviation Detectability）  
**问题**: 检测使用的是**所有 K-1 个评分（包括串通者）的均值**。当串通者存在于审计者集合中时，他们的评分会污染诚实基线——每个串通者使均值偏移 δ_colluder/(K-1)，降低了检测效力。原证明说"对 H 个诚实评分的均值应用 Hoeffding"，但协议实际运行时并不知道哪些是诚实的。

**修复**:
- 将共识基线从**均值（mean）**改为**中位数（median）**
- 中位数具有崩溃点 50%：当诚实者占多数时，中位数保证是诚实评分，无论串通者的评分有多极端
- 更新证明，解释为何中位数避免污染，并引用中位数的 Hoeffding 浓度界
- 保持相同的指数界形式（常数保持一致）
- 更新中文翻译，明确提到中位数基线

## 修复 4：Theorem 3 — 从单边 Hoeffding 严格推导 e^{-2MΔ²}

**严重程度**: 🟡 中等  
**位置**: 原第 806–839 行（Hoeffding Guarantee 证明）  
**问题**: 原证明存在多处不严谨：
1. 使用 `(1-p₀)^M ≤ e^{-p₀M}` 作为界（这是基本的 1-x ≤ e^{-x}，而非 Hoeffding 不等式）
2. 声明 `p₀(Δ) = κΔ²` 但未给出推导
3. 最终形式 `e^{-2MΔ²}` 通过"保守校准 κn=2"得到，缺乏严格推导

**修复**:
- **Step 1**: 直接从单边 Hoeffding 不等式推导单个审计者的未检测概率
  - 维护者分数: s_i = μ* + δ + ε_i（ε_i 为零均值有界噪声）
  - P(未标记 | δ ≥ Δ) = P(|δ + ε_i| ≤ τ) ≤ P(ε_i ≤ τ-Δ) ≤ exp(-2(Δ-τ)²/(Δs)²)
  - 规范化 Δs=1, τ→0: P(未标记) ≤ e^{-2Δ²}
- **Step 2**: M 个独立审计者: P(零标记) ≤ (e^{-2Δ²})^M = e^{-2MΔ²}
- **Step 3**: n 个独立审计事件: P(全部存活) ≤ e^{-2MΔ²n}（更严格）

**关键改进**: 从"断言 p₀ = κΔ²"变为"从单边 Hoeffding 严格推导 e^{-2Δ²}"，整个指数界链完全严格。

## 剩余注意事项

1. **第 214 行**: `\\item \\textbf` 存在类似的双反斜杠问题，但该行属于不同章节，未在本次修复范围内。
2. **Theorem 2 的中位数 Hoeffding 界**: 中位数的浓度界与均值的界有略微不同的常数因子。在假设分布密度在底部分有下界的前提下，使用与均值相同的形式是保守且正确的。如需更精确的常数，可参考 Serfling (1980) 或 Boucheron et al. (2013) 中关于中位数浓度的结果。
