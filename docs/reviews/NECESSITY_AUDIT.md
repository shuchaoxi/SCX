### B.1 mstar_necessity.md — M* 紧性证明

#### 证明结构审查

**核心推导链**（§3.6 形式证明）：

1. **假设**：p_h → 0（诚实误报率可忽略）、Lipschitz 界为紧的、审计者误差为最坏情况亚高斯（Rademacher）

2. **M* 定义**（精确形式）：
   $$M^* = \left\lceil \frac{2\bar{\sigma}^2 \log(\kappa/(\kappa - L_B \delta_{\min}))}{(\delta_{\min} - \varepsilon)^2} \right\rceil$$

3. **关键推导**（M = M* − 1 时）：
   $$\Delta U \geq (\kappa - L_B \delta_{\min})(e^{\alpha} - 1)$$

   其中 α = (δ_min − ε)²/(2σ̄²) > 0

#### 逐步骤验证

| 步骤 | 内容 | 验证 |
|------|------|------|
| §3.1 | 精确必要条件的正确定义 | ✅ M < M* ⇒ 存在偏差使 U_G(θ^G + δ; θ) > U_G(θ^G; θ) |
| §3.2 步骤 1-3 | 收益函数、Lipschitz 性质、检测概率 | ✅ 公式与治理论文一致 |
| §3.2 步骤 4-5 | FOC 求解最优偏差 | ✅ 微分正确。但存在迭代求解的隐式方程 |
| §3.4 | 数值验证 | ⚠️ 有自我矛盾：初次计算 Γ < 0 后重做 |
| **§3.6** | **形式证明（核心）** | ✅ **代数严格正确** |

**核心代数验证**（§3.6, 行 308–326）：

```
At M = M*:
  exp(-M*·(δ-ε)²/(2σ̄²)) = 1 - L_B·δ_min/κ

At M = M*-1:
  exp(-(M*-1)·(δ-ε)²/(2σ̄²)) = (1 - L_B·δ_min/κ) · e^α

Therefore:
  ΔU ≥ L_B·δ_min - κ[1 - (1 - L_B·δ_min/κ)e^α]
     = L_B·δ_min - κ + (κ - L_B·δ_min)e^α
     = (κ - L_B·δ_min)(e^α - 1)
```

因为 κ > L_B·δ_min（有意义审计）且 e^α > 1（α > 0），故 ΔU > 0。✅

#### 边界情况分析

| 边界情况 | 分析 | 结论 |
|---------|------|------|
| δ_min = ε | α = 0, e^α = 1, ΔU = 0 | 退化情况——检测不可能，M* 无定义 |
| ρ̄ > 0（相关审计者） | M_eff = M/(1+(M-1)ρ̄) 代入 | 证明结构不变，不等式在分母为正时成立 |
| κ ≤ L_B·δ_min | 审计惩罚不足 | 原治理定理 1 不成立——这是预期行为 |
| 精确 vs 保守 M* | M*_exact ≤ M*_cons | 使用精确形式使阈值更紧，但不影响 iff 性质 |

#### 评级

| 维度 | 评分 | 说明 |
|------|------|------|
| 证明正确性 | ✅ **95%** | 代数推导正确，核心不等式严格 |
| 假设诚实度 | ⚠️ **良好** | p_h → 0、Lipschitz 紧性和最坏情况亚高斯的假设被明确声明 |

### B.2 chernoff_necessity.md — Hoeffding→Chernoff 最优性

#### 证明结构审查

**两个核心比较**：

1. **协议 Thm 3（Bernoulli 零检测事件）**：比较精确概率 (1−p)^n 与 Hoeffding 界 e^{−2np²}
2. **一般尾部界**：比较 Chernoff 率 D_KL(q‖p) 与 Hoeffding 率 2(q−p)²

#### 关键引理验证

**引理（§3.2 行 119–122）**：对所有 p ∈ (0,1)，log(1−p) < −2p²

| 步骤 | 验证 |
|------|------|
| 定义 f(p) = log(1−p) + 2p² | ✅ |
| f(0) = 0 | ✅ |
| f'(p) = −1/(1−p) + 4p = −(2p−1)²/(1−p) ≤ 0 | ✅ 代数恒等式成立 |
| f'(p) < 0 for p ≠ 1/2 | ✅ |
| ∴ f(p) < f(0) = 0 for p > 0 | ✅ |

**但存在重要区分**：
- 此引理证明 **(1−p)^n < e^{−2np²}**（精确概率严格小于 Hoeffding 界）
- 这比较的是"精确事件概率"与"Hoeffding 上界"
- §4 的 KL 散度比较才是 Chernoff 界 vs Hoeffding 界的比较

**D_KL vs 2Δ² 比较**（§4, 行 226–249）：

| 参数 | D_KL(q‖p) [nats] | 2(q−p)² [nats] | 比值 |
|------|------------------|----------------|------|
| p=0.2, q=0.5 | 0.223 | 0.180 | 1.24× |
| p=0.05, q=0.5 | 0.830 | 0.405 | 2.05× |
| p=0.01, q=0.5 | 1.615 | 0.480 | 3.36× |

✅ 数值验证正确。

**渐近最优性**（§6.2）：Cramér 定理确认 Chernoff 率是 Bernoulli 的最佳可能指数率。✅
