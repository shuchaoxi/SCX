## 一、定理清单

### 定理 1：多专家一致性噪声检测保证（The Unlikely Lone Genius Theorem）
- **位置：** 主论文第21–27行；SI S1 完整证明
- **一句话描述：** 在假设 A1–A6 下，基于多专家一致性的 SCX 噪声检测器，其 F1 分数以指数速率 $2M\Delta_s^2$ 收敛到 1，下界为 $\mathrm{F1} \geq 1 - \frac{1}{\eta}\sum_s \rho_s \exp(-2M\Delta_s^2)$。
- **证明状态：** 严格证明（基于 Hoeffding 不等式 + Chernoff 界收紧）
- **核心引理：**
  - 引理 1（均值分离）：干净样本与噪声样本的期望一致性分数存在可分离间隙 $\Delta_s$
  - 引理 2（假阳性率上界）：$\mathbb{P}(C>\theta \mid \text{clean}) \leq \exp(-2M(\theta-\mu_s)^2)$
  - 引理 3（真阳性率下界）：$\mathbb{P}(C>\theta \mid \text{noise}) \geq 1 - \exp(-2M(1-C_{bal}\cdot\frac{\mu_s}{K-1}-\theta)^2)$
- **推论：**
  - 推论 1（对称专家）：当所有专家干净误差率相同时，F1 界简化
  - 推论 2（最优阈值选择）：给出 $\theta^*$ 闭式解和所需最少专家数 $M$
  - 推论 3（均匀可检测性）：固定阈值 $\theta=1/2$ 的指数保证
  - 推论 4（有限样本修正）：用经验估计 $\hat{\mu}_s$ 替代 $\mu_s$ 的置信界
- **关联：** Dawid–Skene 模型作为 SCX 在平凡划分 $\mathcal{S}=\{\mathcal{X}\}$ 下的特例

### 定理 2：弱特征失效边界（The Impossible Demand Theorem）
- **位置：** 主论文第37–43行；SI S2 完整证明
- **一句话描述：** 当特征表示 $\phi(X)$ 与真实状态 $S$ 之间的互信息 $\delta = I(\phi(X);S)$ 不足时，SCX 噪声检测器的性能被限制在损失基线附近：$\mathrm{F1}_{SCX} \leq \mathrm{F1}_{base} + C_F\sqrt{\delta/2}$。
- **证明状态：** 严格证明（信息论：Fano 不等式 + Pinsker 不等式 + 全变差距离传递）
- **核心引理：**
  - 引理 1（Fano 下界）：状态估计错误概率 $\mathbb{P}(\hat{S}\neq S) \geq \frac{H(S)-\delta-\log 2}{\log K_{\mathcal{S}}}$
  - 引理 2（SCX 可靠性退化）：$\mathbb{E}[|C(\hat{S})-C(S)|] \leq \mathbb{P}(\hat{S}\neq S) + O(1/\sqrt{n_{\min}})$
- **推论：**
  - 推论 1（完全失效 $\delta=0$）：SCX 完全退化为损失基线
  - 推论 2（损失不信息噪声）：均匀标签噪声下，损失基线退化为随机猜测
  - 推论 3（最小信息阈值）：达到 F1 改进 $\Delta$ 所需最小 $\delta_{\min}$
  - 推论 4（状态数量效应）：归一化弱度 $\varepsilon_\phi = \delta/\log K_{\mathcal{S}}$ 决定 SCX 有效性
- **实用诊断：** 4 个可操作的诊断方法（组内一致性方差、随机基线比较、监督 ARI、互信息估计）

### 定理 3：诚实人定理（The Honest Person Theorem）
- **位置：** 主论文第15–18行；SI S3 完整证明
- **一句话描述：** 从观测数据中区分标签噪声与样本固有难度在数学上是不可能的——存在两个数据生成过程 $\mathcal{P}_{noise}$ 和 $\mathcal{P}_{hard}$，产生完全相同的可观测联合分布，使得任何算法的最小错误率至少为 $\rho\eta/2$。
- **证明状态：** 严格证明（显式构造，$K=2$ 和 $K>2$ 两种情况）
- **K=2 证明：** 构造两个世界：
  - 世界 A（噪声驱动）：状态 $s_1$ 中真标签 $y^*=0$，以概率 $\eta_{err}$ 翻转为 1
  - 世界 B（难度驱动）：状态 $s_1$ 中包含 $\eta_{amb}$ 比例的 $y^*=1$ 样本（difficult but correct），无噪声
  - 验证：$\mathcal{P}_{noise}(x,y,\{f_m\}) = \mathcal{P}_{hard}(x,y,\{f_m\})$
- **K>2 证明：** 使用"完全随机专家"构造（世界 B 中专家与真标签条件独立）
- **算法不可能性：** $\max(\mathrm{Error}_{noise}, \mathrm{Error}_{hard}) \geq \rho\eta/2$
- **推论（打破不可辨识性的假设）：**
  - 推论 1（锚定标签）：已知真实标签的子集打破不可辨识性
  - 推论 2（不相交训练数据）：A1 假设打破不可辨识性
  - 推论 3（输入独立噪声）：A4 假设打破不可辨识性
  - 推论 4（状态同质性）：A5 假设打破不可辨识性
  - 推论 5（平衡错误分布）：A6 假设打破不可辨识性
- **最小充分假设集：**
  - $\mathcal{A}_*^{(1)} = \{A1, A4, A5\}$
  - $\mathcal{A}_*^{(2)} = \{A1, A4, A6\}$
  - $\mathcal{A}_*^{(3)} = \{A5, A6\}$ （需 $|\mathcal{S}|\geq 2$）
- **附加推论：** 人人平等定理（Everyone Equal Theorem）——任何声称能区分噪声和难度的观察者，要么声明其结构性假设，要么其声称在数学上是空洞的
- **猜想：** 好人收敛猜想（Good Person Convergence Conjecture）——**未证明**，原文仅有 13 行"证明草图"，被明确标记为猜想而非定理

### 定理 4/4'：精确常数极小极大最优性（The Extreme Precision Theorem）
- **位置：** 主论文第29–35行；SI S4 完整证明
- **一句话描述：** SCX 自适应阈值检测器不仅实现指数收敛率（Chernoff 信息 $\kappa$ 替代 Hoeffding 的 $2\Delta^2$），而且其渐近常数精确匹配极小极大下界：$\lim_{M\to\infty} e^{M\kappa}\sqrt{2\pi M}(1-\mathrm{F1}) = C_*/\eta$，且没有任何算法可以改进这个常数。
- **证明状态：** 严格证明
- **核心引理：**
  - 引理 A（Bahadur-Rao 精确渐近）：Bernoulli 尾概率的精确展开，含 $1/\sqrt{2\pi M}$ 因子
  - 引理 B（F1 渐近展开）：$1-\mathrm{F1} = \frac{\mathrm{FNR}}{2} + \frac{1-\eta}{2\eta}\mathrm{FPR} + O(r^2)$
  - 引理 C（Chernoff 信息闭式解）：$\theta^*$ 和 $\kappa$ 的解析表达式
  - 引理 D（自适应阈值最优性）：$\theta^\dagger = \theta^* + \frac{1}{M}\frac{\log((1-\eta)/\eta)}{D^*} + O(M^{-2})$
  - 引理 E（极小极大下界）：通过 Neyman-Pearson 引理 + Bahadur-Rao 建立下界
  - 引理 F（多状态聚合）：全局 F1 受最慢指数衰减状态（瓶颈率）支配
- **关键机制：** $O(1/M)$ 的阈值偏移产生 $O(1)$ 的指数前因子，且 FPR 和 FNR 贡献出现关键对消，产生相同的 $((1-\eta)/\eta)^s$ 因子
- **数值验证（SI S8）：** 5 组参数集，自适应极限 $K_{ad} = C_*/\eta$ 在所有情况下匹配至机器精度（$<10^{-15}$）

### 定理 5：状态发现的聚类一致性
- **位置：** SI S5
- **一句话描述：** 在强特征区域（$\Delta_{\min}^2/(\sigma^2 d_\phi) \geq C_0$）下，使用 $O(\log n)$ 次随机初始化的 Lloyd's k-means 以指数高概率恢复真实状态划分：$P(\hat{\mathcal{C}} \neq \mathcal{C}^*) \leq K\cdot\exp(-c_1 \frac{n_{\min}\Delta_{\min}^2}{\sigma^2 d_\phi}) + o(1)$。
- **证明状态：** 严格证明（强分离条件下的完整分析）
- **核心引理：**
  - 引理 1（总体极小值邻近性）：$\|\theta^* - \mu\| \leq \epsilon_{pop} < \Delta_{\min}/8$
  - 引理 2（经验极小值指数收敛）：$P(\|\hat{\theta}_n - \theta^*\| \geq t) \leq C_4 \exp(-c_2 \frac{n t^2}{\sigma^2 d_\phi})$
  - 引理 3（确定性划分恢复）：中心邻近性 $\Rightarrow$ 正确划分（对 $\|\varepsilon\| < 3\Delta_{\min}/8$ 的点）
  - 引理 4（Lloyd 算法在强分离下）：$R = C_R\log n$ 次重启足以以高概率找到全局极小值
- **局限性说明：** NP-hard 鸿沟在强分离区域内被解决（良好景观，唯一局部极小值）；弱分离区域不提供保证

### 命题 6：Bootstrap ARI 稳定性作为特征强度诊断
- **位置：** SI S6
- **一句话描述：** Bootstrap ARI 稳定性 $S(\Phi,K)$ 可作为特征强度的实用代理指标：$S > 0.7$ 表明 $\delta/\log K \lesssim 0.2$（SCX 可能有效），$S \leq 0.7$ 则特征不足。
- **证明状态：** 部分证明——(a) 强特征 ⇒ 高稳定性有证明草图；(b) 无结构 ⇒ 低稳定性有形式论证；(c) 诊断阈值是启发式的（明确标注为"并非精确数学陈述"）
- **算法：** Bootstrap ARI 稳定性诊断（$B=50$ 次重采样，$R=20$ 次 k-means 初始化，Hungarian 算法对齐标签）
- **阈值校准：** Landis–Koch 的 Cohen's $\kappa$ "substantial agreement" 约定（0.7），可进行领域特定校准
- **局限性明确列出：** 充分非必要、计算成本、启发式阈值、与 $\delta$ 的关系是经验性而非形式化的

### 认识论形式化系统定理（E1–E5 + K 算子）
- **位置：** SI S3，第435–580行
- **一句话描述：** 在公理 E1–E5 和 SCX 框架条件下，知识算子 $\mathbf{K}$ 满足 $\mathbb{P}(\mathbf{K}(\mathcal{S},\mathcal{C}) \neq T(\mathcal{S})) \leq \sum_s \rho_s \exp(-2M\Delta_s^2)$，且此界是紧的（由定理 4 保证极小极大最优）。
- **证明状态：** 严格证明（直接应用定理 1 的集中论证）
- **K 算子性质：** 非平凡性、指数可靠性、审计单调性、Gettier 免疫性（操作化）、认知平等性
- **与经典认识论的关系：** SCX 不重新定义"知识"，而是操作化——知识是一个在声明的假设下经多专家审计幸存并具有可证明统计保证的声明的数学状态

## 二、关键概念定义

### 核心检测框架
| 概念 | 定义 | 位置 |
|------|------|------|
| **一致性分数（Consistency Score）** $C(x)$ | $C(x) = \frac{1}{M}\sum_{m=1}^M e_m(x,y)$，其中 $e_m = \mathbf{1}\{\ell(f_m(x), y) > \tau\}$ | 主论文 / S1 |
| **检测规则（Detection Rule）** | 标记为噪声当且仅当 $C(x) > \theta$ | S1 |
| **标签噪声模型** | $y = y^*$ 以概率 $1-\eta$，$y \sim \text{Uniform}(\mathcal{Y}\setminus\{y^*\})$ 以概率 $\eta$ | S1 |
| **状态空间** $\mathcal{S}$ | 输入空间 $\mathcal{X}$ 的可测划分 $\Pi = \{s_1, \ldots, s_{|\mathcal{S}|}\}$ | S1 |
| **分离间隙** $\Delta_s(\theta)$ | $\Delta_s = \min(\theta - \mu_s,\; 1 - C_{bal}\cdot\frac{\mu_s}{K-1} - \theta)$ | S1 |
| **最优阈值** $\theta_s^*$ | $\theta_s^* = \frac{1}{2}(1 + \mu_s\cdot\frac{K-2}{K-1})$ | S1 |

### 信息论概念（定理 2）
| 概念 | 定义 | 位置 |
|------|------|------|
| **$\delta$-弱特征** | $I(\phi(X); s(X)) \leq \delta$（以 nats 计） | S2 |
| **归一化弱度** $\varepsilon_\phi$ | $\varepsilon_\phi = \delta / \log K_{\mathcal{S}} \in [0,1]$ | S2 |
| **全变差弱度** $\Delta_\phi$ | $\Delta_\phi = \max_{s_1\neq s_2} \mathrm{TV}(P_{\phi\mid S=s_1}, P_{\phi\mid S=s_2})$ | S2 |
| **损失基线检测器** | $\hat{z}_{loss}(x) = \mathbf{1}\{\max_m \ell(f_m(x), y) > t\}$ | S2 |
| **随机基线性能** | $\mathrm{AUC}_{rand}=0.5$, $\mathrm{F1}_{rand}=2\eta/(1+\eta)$ | S2 |

### 精确常数最优性概念（定理 4）
| 概念 | 定义 | 位置 |
|------|------|------|
| **Chernoff 信息** $\kappa$ | $C(\mathrm{Bern}(p_0), \mathrm{Bern}(p_1))$ | S4 |
| **Chernoff 点** $\theta^*$ | $\mathrm{KL}(\theta^*\|p_0) = \mathrm{KL}(\theta^*\|p_1)$ 的唯一解 | S4 |
| **鞍点** $\lambda_0^*, \lambda_1^*$ | $\lambda_0^* = \log\frac{\theta^*(1-p_0)}{p_0(1-\theta^*)} > 0$, $\lambda_1^* = \log\frac{\theta^*(1-p_1)}{p_1(1-\theta^*)} < 0$ | S4 |

| **精确常数** $C_*$ | $C_* = \frac{\eta^r(1-\eta)^{1-r}}{2} \cdot \frac{1/\lambda_0^* + 1/\|\lambda_1^*\|}{\sqrt{\theta^*(1-\theta^*)}}$ | S4 |

## 三、基础假设完整清单

### 假设 A1–A6（定理 1 的核心假设）

| 编号 | 名称 | 内容概述 | 作用 |
|------|------|----------|------|
| **A1** | 不相交训练集 | $M$ 个专家在 $M$ 个不相交数据子集上训练 | 确保专家独立性，使集中不等式成立 |
| **A2** | 干净数据上的条件独立 | 在给定 $x$ 的条件下，专家错误指标 $\{e_m\}$ 相互独立 | 允许 Hoeffding/Chernoff 界分解乘积 |
| **A2'** | 有界条件相关（A2 的松弛） | 专家错误具有有界成对相关性 $\bar{\rho} \geq 0$，有效专家数 $M_{eff} = M/(1+(M-1)\bar{\rho})$ | 当专家共享架构/训练范式时仍适用 |
| **A3** | 有界损失 | 损失函数 $\ell: \mathcal{Y}\times\mathcal{Y} \to [0,B]$，$B<\infty$ | 允许应用集中不等式 |
| **A4** | 均匀独立噪声 | 标签噪声独立于输入 $x$，均匀分布在错误类别上，全局噪声率 $\eta$ | 使噪声和干净的期望一致性分数可分离 |
| **A5** | 状态同质性 | 在同一状态 $s$ 内，干净样本的期望专家错误率 $\mu_s$ 为常数 | 保证状态内均值分离间隙一致 |
| **A6** | 平衡错误分布 | 每类专家错误率的最大/最小比值受 $C_{bal}$ 约束 | 控制由于类间错误不对称导致的检测退化 |

### 最小充分假设集（打破定理 3 不可辨识性）

| 组合 | 包含假设 | 说明 |
|------|----------|------|
| $\mathcal{A}_*^{(1)}$ | A1, A4, A5 | SCX 框架的标准操作假设集 |
| $\mathcal{A}_*^{(2)}$ | A1, A4, A6 | 替代组合 |
| $\mathcal{A}_*^{(3)}$ | A5, A6（需 $\|\mathcal{S}\|\geq 2$） | 最简组合，无需不相交训练或均匀噪声 |

若三个组合**全部不成立**，定理 3 完全适用——没有任何算法能区分标签噪声和样本难度。

### 认识论公理 E1–E5（SCX 公理系统）

| 编号 | 名称 | 内容概述 |
|------|------|----------|
| **E1** | 假设透明性 | 声明 $\mathcal{S}$ 必须伴随其结构性假设 $\mathcal{A}_{\mathcal{S}}$ |
| **E2** | 假设健全性 | 在 $\mathcal{A}_{\mathcal{S}}$ 下，验证者判断 $J_m$ 是真实值 $T(\mathcal{S})$ 的无偏估计 |
| **E3** | 状态同质验证者质量 | 每个状态 $s$ 内的验证者质量是常数 |
| **E4** | 观测不可区分性 | 验证者只能访问 SCX 框架定义的观测数据 |
| **E5** | 反例记录 | Yajie CEC $\mathcal{E}_t$ 累积反例供审计 |

### 定理 5 的附加假设（强分离聚类）

| 假设 | 内容 |
|------|------|
| **生成模型** | $\phi(x) = \mu_{s(x)} + \varepsilon$，$\varepsilon$ 为零均值次高斯随机向量，$K$ 固定 |
| **聚类结构** | 真实划分为 $\mathcal{C}^* = \{C_1^*, \ldots, C_K^*\}$，$\pi_k = P(S=k) > 0$ |
| **强分离** | $\Delta_{\min}^2/(\sigma^2 d_\phi) \geq C_0$（$C_0$ 为足够大的普适常数） |

| Theorem 2 的 PR-AUC 界 | **需补充步骤** | 原文承认"完全严格的阈值积分处理需要额外步骤" |

## 七、开放问题

1. **异质专家质量：** 定理 4' 依赖 i.i.d. Bernoulli 观测的 Bahadur-Rao 定理；扩展至非相同专家错误率需要广义大偏差原理
2. **假设个体必要性：** 六个辨识性假设的充分性已建立，但个体必要性仅对对称二分类情况证明
3. **更强的信息论界：** 定理 2 使用 Pinsker 不等式（保守），强数据处理不等式可能收紧诊断
4. **增长 $K$ 的聚类：** 定理 5 假设 $K$ 固定，$K\to\infty$ 需要单独分析
5. **好人收敛猜想的形式证明：** 需要形式化主体模型、动态过程、收敛定义和 Lyapunov 函数
