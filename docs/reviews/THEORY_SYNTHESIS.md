## 5. 新研究方向

### 5.1 现已可证明（今日工作使之前不可证的方向成为可能）

#### D1: 治理 M* 精确公式的 Chernoff 紧化（**已可证明**）

**之前**: 治理 Thm 1 的 M* 阈值仅为 Hoeffding 保守界下的充分条件。
**现在**: M* 紧性已证明（mstar_necessity.md）。下一步：将 Hoeffding 形式 M*_H 替换为 Chernoff 形式 M*_C ≈ 0.81 M*_H（chernoff_necessity.md §5.1 已给出约 19% 的缩减因子）。这意味着更少的审计者即可保障诚实。

**精确声明**:
$$M^*_C = \left\lceil \frac{2\bar{\sigma}^2 \cdot D_{\text{KL}}(\mu_s + \Delta \| \mu_s)}{(\delta_{\min} - \varepsilon)^2} \cdot \log\left(\frac{\kappa}{\kappa - L_B\delta_{\min}}\right) \cdot \frac{1}{D_{\text{KL}}} \right\rceil$$

**可行性**: **HIGH**。已知技术（Chernoff + M* 紧性证明的组合），无需新数学。

#### D2: 修正后 NPE 均衡条件的完整下游重算（**纯代数**）

**之前**: NPE 定理 1 的 4 个代数错误（C1-C4）导致 C8 传导性错误。
**现在**: 均衡条件已确定为 Δ(|E|) ≥ −λ。下一步：使用此正确条件重新计算 CEC 临界值 |E|*、稳定性裕度 M(|E|)、先发优势衰减 δ(|E|)、混合策略 p* = −Δ_A/λ。

**可行性**: **HIGH**。纯代数推导，不需要新数学。是所有下游工作的前置条件。

#### D3: 协议 Thm 3 的 Chernoff 升级（**标准技术**）

**之前**: 协议 Thm 3 使用 Hoeffding 界 P(未检测) ≤ e^(−2MΔ²)。
**现在**: chernoff_necessity.md 已证明 Chernoff 严格紧于 Hoeffding。协议 Thm 3 应升级为：
$$P(\text{未检测}) = (1-p_{\text{det}})^{MT} = \exp(MT \log(1-p_{\text{det}})) < \exp(-2MT p_{\text{det}}^2)$$

**关键推论**: 所需维护者数从 O(1/Δ²) 缩放变为 O(1/p_det) 缩放——稀有事件下差异极大（50× 改善）。

**可行性**: **HIGH**。标准大偏差技术。

## 附录 A：今日完成的关键证明摘要

### A.1 M* 紧性证明 (mstar_necessity.md)

**核心不等式**（§3.6 形式证明）：

当 M = M* − 1 时，政府偏离 δ = δ_min 的净收益为：

$$\Delta U \geq (\kappa - L_B \delta_{\min})(e^\alpha - 1) > 0$$

其中 α = (δ_min − ε)²/(2σ̄²) > 0，κ > L_B·δ_min（有意义审计条件）。

由于 κ > L_B·δ_min 且 e^α > 1，乘积严格正。

**含义**：M* 阈值是紧的——治理 Thm 1 升级为完整充要条件（iff）。

### A.2 Chernoff 最优性证明 (chernoff_necessity.md)

**关键引理**（§3.2）：对所有 p ∈ (0,1)，log(1−p) < −2p²。

**证明**：f(p) = log(1−p) + 2p²，f(0) = 0，f'(p) = −1/(1−p) + 4p = −(2p−1)²/(1−p) ≤ 0。

因此精确未检测概率 (1−p_det)^(MT) = exp(MT·log(1−p_det)) < exp(−2MT·p_det²)（Hoeffding 界）。

**数值比较**（§4，D_KL vs 2Δ²）：

| p₀ | q | D_KL(q‖p₀) | 2(q−p₀)² | Chernoff/Hoeffding |
|----|---|-----------|----------|-------------------|
| 0.2 | 0.5 | 0.223 | 0.180 | 1.24× |
| 0.05 | 0.5 | 0.830 | 0.405 | 2.05× |
| 0.01 | 0.5 | 1.615 | 0.480 | 3.36× |

### A.3 0-1 损失必要性 (scx_thm1_scope.md)

**退化定理**（§8.2）：对任意 ε > 0，存在有界损失函数 ℓ 和阈值 τ，使得：
$$|E[C | noise] − E[C | clean]| < ε$$

而专家的 0-1 误分类率在干净和噪声条件下差异显著。

**构造**：ℓ(ŷ, y) 在"容错集" A(y) 上取 ε/2，在其他错误类上取 1。τ = 3ε/4。当 A(y*) 很大时，干净错误（落入 A(y*)）的 e_m = 0，噪声时正确预测（f_m = y*，但 y ∈ A(y*)）的 e_m 也为 0。两期望值差在 O(ε) 内。

**结论**：定理 1 的 0-1 损失限制是**根本必要的**（置信度 90%）。

## 附录 B：NPE 修正后的均衡条件速查

### 修正前（错误）

| 条件 | 含义 |
|------|------|
| Δ(|E|) ≥ λ − κ | (A,...,A) 为 NE（错误！） |
| Δ_A ≥ −λ + κ（2人）vs −λ + 2κ（N人） | 内部不一致 |
| p* = (−Δ_A − 2κ)/(λ − κ) | 混合策略（错误） |

### 修正后

| 条件 | 含义 |
|------|------|
| **Δ(|E|) ≥ −λ** | (A,...,A) 为纯策略 NE ✅ |
| Δ(|E|) ≤ 0 | (D,...,D) 为纯策略 NE |
| 0 < Δ(|E|) < −λ | 多重均衡区间 |
| **p* = −Δ_A/λ** | 对称混合策略 NE |
| **(A,...,A) 总是唯一纯策略 NE**（当 λ > 0 时） | 核心简化 |

**关键简化**：原错误条件 Δ ≥ λ − κ 要求精度优势超过"净碎片化成本"（λ − κ）。修正后 Δ ≥ −λ 意味着精度优势只需超过"负的碎片化成本"——即只要精度优势不是太负（太不利），采纳就是均衡。在 λ > 0 的实际参数下，这远弱于原条件——采纳均衡在更广范围内成立。
