### Thm1 (AC-Theorem): Hoeffding 常数问题

**文件**: `papers/theorems/theorem_ac_complexity.tex`

**发现的问题**:

1. **Hoeffding 不等式双侧/单侧混淆 (第373-389行)**
   - 方程(6) (第373-375行) 给出了 Hoeffding 不等式的**双侧版本**：
     ```
     P(|T_K - E[T_K]| ≥ ε | H0) ≤ 2exp(-2Kε²)
     ```
   - 但在步骤7 (第385行)，引用该方程为"单侧版本"，并得出：
     ```
     P(T_K - β̄ > Δ/2 | H0) ≤ exp(-2K(Δ/2)²) = exp(-KΔ²/2)
     ```
   - **诊断**：双侧版本的常数是 `2exp(...)`，而单侧版本是 `exp(...)`。证明中实际使用的是单侧版本（因为检测规则是单侧的：仅当 `T_K > θ` 时拒绝 H0），但方程(6)写的是双侧形式。这不影响最终结果（常数 `2log(1/δ)` 是正确的），但会造成读者困惑。
   - **严重性**：中等（数学上正确，表述不一致）

2. **常数 2 的验证**
   - 单侧 Hoeffding: `P(X̄ - μ ≥ t) ≤ exp(-2nt²)`
   - 取 `t = Δ/2`, `n = K`: `exp(-2K(Δ/2)²) = exp(-KΔ²/2)`
   - 要求 `≤ δ ⇒ K ≥ 2log(1/δ)/Δ²`
   - 代入 `Δ ≥ η(α_min - β_max)`: `K ≥ 2log(1/δ) / [η²(α_min - β_max)²]`
   - **结论**：定理中的常数 `2` 是正确的，无误。

3. **Bernstein 替代界（诚实暴击中提出）**
   - 第436行给出 Bernstein 替代界：`K ≥ 8β_max(1-β_max)log(2/δ)/Δ²`
   - 该界在 `β_max → 0` 时优于 Hoeffding，但常数 `8` 与主定理的 `2` 不直接可比（因为 Bernstein 引入了方差项 `β_max(1-β_max)`）
   - **意见**：这些替代界应在诚实暴击中与主界定性比较，而非作为替代定理提出

**修复方案**:
- 在第6步明确列出双侧和单侧两个版本
- 在第7步标注"使用单侧版本"并解释转换
- 将方程(6)的标签改为非公式引用，或保留但添加脚注说明

### Thm2 (Weak Feature Theorem): 证明链中 C_F 常数错误

**文件**: `papers/theorems/theorem_2_weak_feature.tex`

**发现的问题**:

1. **关键错误：C_F 缺少因子 1/4 (第661-680行, 引理4)**

   证明链：
   ```
   gap ≤ (K/(K-1)) · (TV/2) · gap_max       (引理3, 式eq:class-f1-tv)
   TV ≤ √(δ/2)                                 (引理1, Pinsker)
   gap_max ≤ 1/[η(1-η)] · √(log K/(2n_min))   (引理4, Hoeffding)
   ```

   合并：
   ```
   gap ≤ (K/(K-1)) · [√(δ/2)]/2 · 1/[η(1-η)] · √(log K/(2n_min))
       = (K/(K-1)) · √(δ/8) · 1/[η(1-η)] · √(log K/(2n_min))
   ```

   由于 `√(δ/8) = √(2δ)/4`，所以：
   ```
   gap ≤ (K/(K-1)) · (1/4) · 1/[η(1-η)] · √(log K/(2n_min)) · √(2δ)
   ```

   **当前定理中 C_F 的定义** (第317-320行):
   ```
   C_F = (K/(K-1)) · √(log K/(2n_min)) · 1/[η(1-η)]
   ```

   **正确的 C_F 应为**:
   ```
   C_F = (1/4) · (K/(K-1)) · √(log K/(2n_min)) · 1/[η(1-η)]
   ```

   - **严重性**：高。定理陈述的 C_F 比正确值大 4 倍，意味着定理声称的 F1 上界比实际紧界宽松 4 倍。
   - **证据**：第682-688行注释中承认 `1/2 · √(δ/2) = √(2δ)/4`，但随后说"经过归一化使 C_F 吸收所有常数"。这种吸收是不合法的——不能在不改变 C_F 定义的情况下吸收因子。

2. **Pinsker 不等式的紧致性声明 (第403-410行)**
   - 声明 Pinsker 不等式在 `ε → 0` 时渐近紧（TV ≈ √(KL/2)），但这仅对 Bernoulli 分布成立
   - 对于一般的特征分布，Pinsker 可能显著松弛（例如接近高斯时松弛因子可达 √2）
   - **意见**：承认此限制已在诚实暴击中提及（第1047-1052行），足够

3. **\$K/(K-1)\$ 因子的推导 (第582-601行)**
   - 推导依赖均匀先验假设 `P(S=s) = 1/K`
   - 非均匀先验时需引入 `Δ_prior` 项（引理2中已定义但引理3中未使用）
   - **意见**：应在定理陈述中显式假设均匀先验

**修复方案**:
- 在定理陈述（第317-320行）中将 C_F 修正为包含 1/4 因子
- 在引理4证明中移除"吸收常数"的模糊表述
- 在定理前提中明确列出"均匀先验"假设
- 同步更新完整证明（第754-764行）中的推导，正确显示 1/4 因子

### Thm4' (state_value.py): Bahadur-Rao 常数问题

**文件**: `src/scx/valuation/state_value.py`

**发现的问题**:

1. **Saddlepoint 一致性：θ\* vs θ† (第723-830行)**

   `exact_constant_minimax` 方法计算 saddlepoints 在 Chernoff 点 `θ*`（满足 KL(θ*||p₀) = KL(θ*||p₁)）。

   但是 `adaptive_threshold_theorem4` 方法（第832-907行）计算的是自适应阈值：
   ```
   θ† = θ* + (1/M) · log((1-η)/η) / D*
   ```
   这与 θ* 相差 O(1/M)。

   随后 `noise_detection_f1_bound_bahadur_rao` （第909-1041行）调用 `exact_constant_minimax` 获取常数（在 θ* 处的 saddlepoints），但实际使用的阈值是 `θ†`。

   - **诊断**：Bahadur-Rao 公式中的 saddlepoint 应该在评估阈值处计算（即 θ† 处的 λ₀†, λ₁†），而不是在 θ* 处。由于 θ† → θ* 当 M → ∞，差异是 O(1/M) 的高阶效应，不影响主导指数项 `e^{-Mκ}`。但对于有限 M，预因子常数会有 O(1/M) 的偏差。
   - **严重性**：低（渐进一致，有限 M 偏差可控）

2. **\$C_{\\min}\$ 公式验证 (第813-818行)**

   Bahadur-Rao 公式：
   ```
   C_min = (η/2) · ((1-η)/η)^s · (1/λ₀ + 1/|λ₁|) / √(θ*(1-θ*))
   ```
   
   其中 `s = |λ₁|/D*`, `D* = log(p₁(1-p₀)/(p₀(1-p₁)))`。

   验证 saddlepoint 计算：
   - 对于 Bernoulli(p₀)，CGF: `ψ₀(λ) = log(1-p₀ + p₀e^λ)`
   - `ψ₀'(λ) = p₀e^λ/(1-p₀ + p₀e^λ)` 
   - 要求 `ψ₀'(λ₀*) = θ*` 得：`λ₀* = log(θ*(1-p₀)/(p₀(1-θ*)))`
   - 代码第797-799行：`lam0 = log(theta_star * (1-p0) / (p0 * (1-theta_star)))` ✓ **正确**
   
   同理 λ₁* 计算（第800-802行）也正确。

   在 saddlepoint 处的方差：`ψ₀''(λ₀*) = θ*(1-θ*)`（指数倾斜分布的已知恒等式）
   代码第817行：`denom = sqrt(theta_star * (1-theta_star))` ✓ **正确**

   预因子 `(η/2) · ((1-η)/η)^s` 来自 Chernoff 界的贝叶斯最优加权：
   - `((1-η)/η)^s` 是贝叶斯最优测试中先验权重比在最优指数处的值
   - 因子 `η/2` 来自错误概率分解：`P(error) = η·P(type-I) + (1-η)·P(type-II)`
   - `1/λ₀* + 1/|λ₁*|` 是两类错误 Bahadur-Rao 预因子的线性组合
   
   ✓ **公式整体合理**，与 Bahadur-Rao 标准形式一致。

3. **数值稳定性 (第719行)**
   - `log_num / log_den` 当 `log_den ≈ 0` 时会产生数值问题
   - 代码第716-717行有空值检查：`if abs(log_den) < eps: return 0.0`
   - 但当 `log_den` 非常小但未达到 `eps` 阈值时，`theta_star` 可能计算为极大/极小值
   - **意见**：可接受，但在极端参数下需要额外防护

**修复方案**:
- 在 `exact_constant_minimax` 的文档字符串中添加说明：saddlepoints 在 θ* 处计算，与 θ† 的差异在渐进意义下可忽略
- 在 `noise_detection_f1_bound_bahadur_rao` 的文档中说明这是渐进公式

## 第3轮审查：深度审查（修复后）

在第2轮发现的基础上，对修复后的问题进行更深层次的审查。

### Thm1: 深入常数分析

1. **阈值 `θ = β̄ + Δ/2` 的构造是否最优？**
   - Neyman-Pearson 引理：对于简单假设 `H0: β̄ vs H1: β̄ + Δ`，最优检验基于似然比 `Λ = P(data|H1)/P(data|H0)`
   - 对于独立 Bernoulli 试验，Λ 是 `T_K` 的单调函数，因此基于 `T_K > θ` 的检验是 UMP（一致最优势）
   - 但阈值 `θ = β̄ + Δ/2` 是对称选择。更优的选择应考虑两类错误的非对称代价或 β̄ 处的方差
   - **意见**：当前选择是合理的对称阈值，但应在诚实暴击中指出不对称代价场景下可进一步优化

2. **Hoeffding vs Bernstein 的定量比较**
   - 在第2轮中，我们确认了 Hoeffding 常数 `2log(1/δ)` 是正确的
   - 诚实暴击中提出的 Bernstein 替代界：`K ≥ 8β_max(1-β_max)log(2/δ)/Δ²`
   - 比较：当 `β_max = 0.01` 时，Bernstein 给出 `K ≥ 0.08·log(2/δ)/Δ²`，远优于 Hoeffding 的 `K ≥ 2log(1/δ)/Δ²`（约 25 倍差距）
   - **意见**：定理应明确标注使用 Hoeffding 是"保守但通用"的选择，并指引读者参考 Bernstein 的更紧版本

### Thm2: 深入证明链分析

1. **引理2的 TV-to-Classification 方向验证**
   - 引理2（第417-436行）给出的是**下界**：`e_cls ≥ (K-1)/K - TV/2`
   - 但在引理3中使用时，需要的是**上界**：`P(correct) ≤ 1/K + TV/2`
   - 验证：`e_cls ≥ (K-1)/K - TV/2` ⇒ `1 - e_cls ≤ 1/K + TV/2` ⇒ `P(correct) ≤ 1/K + TV/2` ✓
   - 在第872行紧致性构造中，实际达到的是 `P(correct) = 1/2 + ε = 1/K + TV`（对于 K=2），比引理的界 `1/K + TV/2` 更紧
   - **意见**：引理2的下界在最坏情形下是紧的（当特征和状态完全独立时），但在典型情形下可能松弛

2. **Fano 不等式的缺失**
   - 第2轮的标题提到 "Fano-Pinsker proof chain"，但实际证明链是：Pinsker → TV → Classification → F1
   - 原始计划中可能在 TV 到 Classification 之间使用 Fano 不等式，但当前使用的是直接推导
   - Fano 不等式给出：`H(S|φ) ≤ H(e_cls) + e_cls·log(K-1)`，可以导出更紧的界
   - **意见**：可以添加一条注释说明 Fano 替代路径，但不影响当前证明的正确性

3. **C_F 修正后的边缘情况验证**
   - `δ = 0`（无特征信息）：`C_F · √(2·0) = 0` → `F1 ≤ F1_base` ✓
   - `δ = log K`（完全特征信息）：
     - 原始 C_F: `F1 ≤ F1_base + [K/(K-1)] · √(log K/(2n_min)) · 1/[η(1-η)] · √(2log K)`
     - 修正 C_F: `F1 ≤ F1_base + (1/4) · [K/(K-1)] · √(log K/(2n_min)) · 1/[η(1-η)] · √(2log K)`
     - 修正后更紧，但仍声称恢复到定理1的无约束界
     - **问题**：修正 C_F 后，`δ = log K` 时的上界比定理1的界更紧，这是否意味着定理2在某些情况下比定理1更强？需要验证是否矛盾。
     - **分析**：定理1给出的是最大可能 F1 上界（under perfect state knowledge），定理2给出的是以特征信息为条件的上界。两者的常数可能不同，因为定理1的界基于完美状态知识下的 Hoeffding 界，而定理2通过 Pinsker+TV 路径引入了额外的松弛。修正 C_F 后定理2在 `δ = log K` 时不应超过定理1的界。
     - **验证**：如果 `(1/4)·[K/(K-1)]·√(2log K) ≤ 1`（即定理2的常数小于定理1），则一致。对于 `K=2`：`(1/4)·2·√(2log2) ≈ 0.59 < 1` ✓
     - 对于 `K=10`：`(1/4)·(10/9)·√(2log10) ≈ 0.56 < 1` ✓ 
     - **结论**：修正 C_F 后无矛盾，定理2在 δ=log K 时仍然弱于定理1，符合直觉。

### Thm3: 深入偏好域分析

1. **价值多元主义与注释噪声的"互斥性"假设**
   - AA.4 的证明假设两种成因互斥：World A 仅有噪声，World B 仅有价值多元
   - 现实中，一个标注可能同时受两种因素影响
   - **意见**：定理应承认这一简化，或在诚实暴击中讨论混合情况的含义

2. **T3 映射的哲学深度**
   - 第618-637行的诚实暴击指出了核心问题：
     "difficulty ↔ value pluralism" 的映射是否保持数学结构？
   - T3 原始构造中，困难样本的定义依赖于模型行为（系统性错误）
   - 偏好域中，价值多元的定义依赖于标注者群体（系统性分歧）
   - 这两个结构在数学上确实不同：T3 中 difficulty ⊂ noise 是可能的（一个样本可以既是困难又带噪声），但偏好域中价值多元与噪声在同一标注事件上互斥
   - **意见**：这是一个深层问题，应被提升为正文讨论而非仅出现在诚实暴击中

### Thm4': 深入 Bahadur-Rao 分析

1. **\$C_{\\min}\$ 的量级检查**
   - 对于典型参数 `p0=0.1, p1=0.3, η=0.1`：
     - `D* = log(0.3·0.9/(0.1·0.7)) = log(3.857) ≈ 1.35`
     - `θ* = log(0.9/0.7)/1.35 ≈ 0.186`
     - `λ₀* = log(0.186·0.9/(0.1·0.814)) ≈ 0.72`
     - `λ₁* = log(0.186·0.7/(0.3·0.814)) ≈ -0.63`
     - `s = 0.63/1.35 ≈ 0.467`
     - `κ = KL(0.186||0.1) ≈ 0.052`
     - `C_min ≈ (0.1/2)·((0.9/0.1)^0.467)·(1/0.72+1/0.63)/√(0.186·0.814) ≈ 0.05·2.68·2.97/0.389 ≈ 1.02`
   - 对于 `M=100`：`asym_error ≈ (1.02/0.1)·exp(-100·0.052)/√(2π·100) ≈ 10.2·0.0055/25.1 ≈ 0.0022`
   - `F1 ≈ 1 - 0.0022 = 0.998`
   - 这看起来合理：100个专家以小的 Chernoff 信息 `κ=0.052` 实现了高 F1

2. **\$\\kappa\$ 的渐近解释**
   - `κ = KL(θ*||p₀) = KL(θ*||p₁)` 是区分两个 Bernoulli 分布的 Chernoff 信息
   - 这控制了每个专家判断中的信息量
   - `M·κ` 是总信息量，主导指数衰减率 `e^{-Mκ}`
   - **意见**：可以在文档中增加这一解释

## 第4轮审查：边界条件测试

### Thm1: 极端参数测试

1. **β_max → 0（完美特异度专家）**
   - 公式：`K_min ≥ 2log(1/δ) / [η²(α_min - 0)²]`
   - 当 β_max → 0 时方差也趋于 0，Hoeffding 界可能显著松弛
   - Bernstein 界在此时给出显著更紧的结果
   - **意见**：定理在极端参数下仍然成立但保守，应在应用中提示

2. **α_min → 1/2（专家质量退化）**
   - 公式中 `(α_min - β_max) → (1/2 - β_max)`
   - 当 β_max 也趋近 1/2 时，分母趋于 0，`K_min → ∞`
   - 这与直觉一致：随机猜测的专家无法提供信息
   - 但在 `α_min = 1/2 + ε, β_max = 1/2 - ε` 的对称情形，公式给出有限 K_min
   - **意见**：行为合理

3. **小 K 下的 Berry-Esseen 误差**
   - 诚实暴击中提到 `C₀/B_K` 在 K=3 时可能超过 0.3
   - 计算：`B_K² ≥ K·β_min(1-β_min)`，若 `β_min = 0.1`，`B_K² ≥ 3·0.09 = 0.27`，`B_K ≥ 0.52`，`C₀/B_K ≤ 0.5583/0.52 ≈ 1.07`
   - 这意味着正态近似误差可能超过 1！在小 K 下完全不可靠
   - **意见**：诚实暴击中的担忧是有根据的。定理的 Θ(1/√K) 标度在 K < 20 时仅有方向性意义。

### Thm2: δ=0 和 δ=logK 的修正 C_F 测试

1. **δ = 0（无信息）**
   - 修正 C_F 后：`F1 ≤ F1_base + (1/4)·C_F_old·√(2·0) = F1_base` ✓

2. **δ = log K（完全信息）**
   - 修正 C_F 后：`F1 ≤ F1_base + (1/4)·[K/(K-1)]·√(log K/(2n_min))·1/[η(1-η)]·√(2log K)`
   - 对于 K=2, n_min=100, η=0.1：
     `F1_base ≈ 2·0.1/1.1 ≈ 0.182`
     `C_F·√(2log2) = (1/4)·2·√(log2/200)·1/0.09·√(2log2) ≈ 0.5·0.059·11.11·1.177 ≈ 0.386`
     `F1 ≤ 0.182 + 0.386 = 0.568`
   - 定理1的无约束上界（对于同样参数）：约为 0.6-0.7
   - **结论**：修正 C_F 后，定理2在 δ=logK 时给出的上界弱于定理1，符合理论预期 ✓

### Thm3: 单调性假设的可证伪性

1. **假设 AA.1.1 的经验内容**
   - 声称：存在单调函数 φ 使得 `|Score(f(x), τ(x)) - φ(Δ(x))| ≤ ε_mono`
   - 这里 Δ(x) = |P(τ=1|x) - 1/2| 依赖于不可观测的真实偏好 τ
   - 无法在不假定已知 τ 的情况下测试该假设
   - **诊断**：这是一个典型的"不可证伪"假设——如果 ε_mono 足够大，总可以找到满足条件的 φ
   - **意见**：AA.1 的 `[Conditionally Rigorous]` 标注是诚实的，但诚实暴击中应更强烈地指出该假设的经验空洞性

### Thm4': 数值稳定性边界

1. **p₀ ≈ p₁ 时的退化**
   - 当 `p₀ ≈ p₁`，`D* → 0`，`κ → 0`
   - `θ*` 的计算中分子分母都趋于 0
   - 代码通过 `eps` 检查处理，但 `eps = 1e-8` 可能过于宽松
   - 当 `D* < 1e-8` 时，`κ < 1e-8`，`exp(-M·κ) ≈ 1`，使得 Bahadur-Rao 公式退化为平凡界

2. **η → 0 或 η → 1 时的行为**
   - 当 `η → 0`，`C_min → 0`
   - 当 `η → 1`（几乎所有样本都是噪声），`((1-η)/η)^s → 0`
   - 这些是合理的极限行为 ✓

## 第5轮审查：跨定理一致性检查与最终润色

### 跨定理一致性

1. **Thm1 ↔ Thm2 的 C_F 连接**
   - Thm2 的 `C_F` 依赖于 Thm1 的 Hoeffding 界
   - 修正 C_F (乘以 1/4) 后，两个定理常数一致
   - Thm1 的 `2log(1/δ)` 常数和 Thm2 的 `√(log K/(2n_min))` 常数来自不同的集中不等式应用（一个是 Hoeffding 尾界，一个是 Bonferroni 一致界），在数学上不直接可比

2. **Thm1 ↔ Thm3 (AA.2) 的 T1 条件一致性**
   - Thm1 使用 `Δ = η(α_min - β_max)` 作为间隙
   - AA.2 使用 `Δ = η·E[Δ²]` 作为检测效应量
   - 两者不同但一致：Thm1 的是专家间的间隙，AA.2 的是偏好得分的变化
   - 这两个 Δ 代表不同的物理量，不应被混淆

3. **Thm3 ↔ Thm1 的专家/标注者类比**
   - 在 AA.2 的 T3 验证中，"多个标注者"的结构类比于 AC-Theorem 的"多个专家"
   - 但标注者的"错误"来源不同：AC 中是判断错误，AA 中是价值差异
   - **意见**：跨定理的术语映射（标注者 ↔ 专家，价值差异 ↔ 判断错误）需要显式说明

4. **Thm4' ↔ Thm1 的 Hoeffding 对比**
   - Thm1 使用 Hoeffding（保守，通用）
   - Thm4' 使用 Bahadur-Rao（精确，需要分布假设）
   - 在 `M → ∞` 极限下，Bahadur-Rao 的 `e^{-Mκ}` 与 Hoeffding 的 `e^{-2MΔ²}` 在 `κ = 2Δ²` 时一致（对于对称 Bernoulli）
   - 验证：对于 `Ber(1/2 ± ε)`，`κ ≈ 2ε²`（在 ε 小时），而 `Δ = 2ε`，所以 `2Δ² = 8ε² ≠ κ`
   - **诊断**：两者常数不同是因为 Hoeffding 界使用的是全局界而非精确的 KL 散度
   - **意见**：在跨定理引用时，应说明 Thm4' 是 Thm1 的精化（使用 Chernoff/KL 代替 Hoeffding）
