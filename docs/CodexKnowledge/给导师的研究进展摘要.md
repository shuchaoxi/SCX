## 一句话

**SCX (State-Conditioned eXpertise)**：专家可靠性不是全局常数，而是数据状态的条件函数。这个框架回答了三个问题——哪些数据是噪声（Thm 1）、方法在什么条件下会失效（Thm 2）、噪声和困难样本为什么区分不了（Thm 3）。

---

## 做了什么（2026.5 → 6）

### 理论：3 个定理，全部有完整证明

| 定理 | 核心结果 | 证明方法 |
|------|---------|----------|
| **Thm 1** 多专家一致性噪声检测 | F1 ≥ 1 - exp(-2MΔ²)/η，M≥20 名专家时 F1≥0.95 | Hoeffding + Chernoff + 条件期望分解 |
| **Thm 2** 弱特征失效下界 | F1 ≤ baseline + √(2δ)，特征互信息 δ→0 时退化到随机 | Fano 不等式 + 耦合论证 + Pinsker |
| **Thm 3** 噪声与困难不可区分 | 构造性反例证明：没有任何算法能区分噪声与困难样本 | Gödel 式构造 |

定理文件总计约 1,700 行数学证明，位于 `theory/theorems/`。
