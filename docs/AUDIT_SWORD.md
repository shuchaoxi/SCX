## English

The SCX framework (including Yajie, Situs, Spring, State Crystallization, and all derivative algorithms) is an open-source mathematical toolkit. Anyone may read, reproduce, and deploy it.

We do not prohibit any specific use.

We retain one inalienable right:

**Any organization using the SCX framework for data quality assessment renders its assessment results verifiable and auditable by independent third parties through the same mathematical methods. Military use is not exempt.**

This is not a restriction. It is deterrence. The audit sword does not discriminate.
