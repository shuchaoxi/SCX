## 攻击路线图

### 最短致命路径 (击穿 L0 → 全塌)

```
攻击 Thm1 (Hoeffding 界)
    │ 若常数错误、独立性假设不成立、或存在反例
    ▼
所有审计保证失效 → M>1 不必要 → Yajie 共识无意义
    │
    ▼
∑g=0 失去数学支撑 → 平等论全塌
```

### 次短路径 (击穿 L1 → 协议层蒸发)

```
攻击 NPE 均衡 (定理1代数错误已发现❗)
    │ 若博弈论基础不成立
    ▼
Yajie 采纳不是均衡 → 没有人会自愿接入 → 协议层无用户
    │
    ▼
SCX 退化为纯数学理论，无经济价值
```

### 绕过路径 (不攻击核心，攻击应用)

```
攻击社会推论7篇 (领域事实错误)
    │ 若教育/医疗/环境等应用不可靠
    ▼
SCX 被标记为"理论正确但应用不可行" → 公众信任丧失
```
