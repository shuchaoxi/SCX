## 论文定位

**SCX: State-Conditioned eXpertise for Data Valuation and Expert-Guided Learning**

中文：**SCX：用于数据价值评估与专家引导学习的状态条件专家性框架**

### 核心命题（已完成证明）

1. **Proposition 1**: 全局专家排序不足 — 严格反证 + Arrow不可能定理类比
2. **Proposition 2**: 高误差采样在噪声下非最优 — 下界证明
3. **Proposition 3**: 状态条件权重优于全局权重 — Gibbs不等式证明

### 四类数据

1. **Valuable** (值得标注) — 高误差 + 高密度 + 高一致性 + 低冗余 → acquire
2. **Redundant** (冗余) — 低误差 + 高覆盖 → compress
3. **Noisy** (噪声) — 高误差 + 孤立 + 标签不一致 → downweight/discard
4. **Expert-dependent** (依赖专家) — 仅特定专家可靠 → route to expert

### 公式体系

```
V(s) = r̄(s) · ρ(s) · L(s) · [1-D(s)] · max_m SCX_m(s)

SCX_m(s) = P(ℓ(f_m(x), y) < τ | x ∈ s)

R_m(s) = E_{x~P(·|s)}[ℓ(f_m(x), f*(x))]
```
