\subsection{1.
Problem：为什么一层Descriptor不够？}\label{ux95eeux9898ux4e3aux4ec0ux4e48ux4e00ux5c42ux63cfux8ff0ux7b26ux4e0dux591f}

\subsubsection{1.1
人类Descriptor的局限性}\label{ux4ebaux7c7bux63cfux8ff0ux7b26ux7684ux5c40ux9650ux6027}

现有势Function开发的共同路径：研究人员凭直觉选择Descriptor（元素种类、晶体结构、温度范围），然后在这些DescriptorDefinition的类别上做均匀采样。这一范式的根本缺陷在于：

\textbf{人类选的Descriptor可能不捕捉Model的真实失败模式。}

\paragraph{1.1.1 AlN v3 案例}\label{aln-v3-ux6848ux4f8b}

在 AlN v3 的实际开发中，这一Problem暴露得极为清晰：

\begin{itemize}
\tightlist
\item
  研究人员Definition了 12
  维手工Feature（化学组分、结构类型、温度区段等），试图覆盖所有的化学环境
\item
  但使用这些Feature对测试集做聚类后，约 \textbf{50\% 的帧被挤在一个State中}
\item
  这个''超级State''内部包含着物理上完全不同的结构（表面、缺陷、高应变构型），它们的力Error差异可达一个数量级
\item
  SCX 在这 50\%
  帧的区域内完全失效------因为单一State内Error方差太大，任何全局专家路由策略都无法有效区分
\end{itemize}

根源Analysis：人类Descriptor的Dimension选择受限于研究人员的\textbf{认知盲区}------人只能Note到自己已知的结构类别，无法预见Model会在哪些''意想不到''的构型中失效。

\paragraph{1.1.2 Layer 1
编码的信息瓶颈}\label{layer-1-ux7f16ux7801ux7684ux4fe1ux606fux74f6ux9888}

形式化地，设 Layer 1 编码为
\(\varphi_1: \mathcal{X} \rightarrow \mathbb{R}^d\)，残差为
\(r(x) = |f_{\text{model}}(x) - f_{\text{true}}(x)|\)。一层编码的信息瓶颈是：

\[I(\varphi_1(X); r) \leq I(X; r)\]

如果 \(\varphi_1\) 的Dimension选择与 \(r\)
不匹配，不等式左边远小于右边。换言之，\(\varphi_1\)
丢弃了预测残差中的Core信息。

\textbf{更根本的Problem}：\(\varphi_1\) 的编码空间 \(\mathbb{R}^d\)
由人类先验Definition，它优先编码''人认为Important的结构Feature''（如晶格常数、对称性），而非''Model失败的模式''。优化
\(I(\varphi_1; r)\) 与优化 \(I(\varphi_1; X)\)
是不同Goal------而人类Descriptor天然偏向后者。

\paragraph{1.1.3 为什么 PCA/SVD
不能解决}\label{ux4e3aux4ec0ux4e48-pcasvd-ux4e0dux80fdux89e3ux51b3}

一个自然的反驳：既然 12 维嵌入导致 50\% 帧拥挤，为什么不对 \(\varphi_1\)
做 PCA 降维或增加Dimension？

\begin{itemize}
\tightlist
\item
  \textbf{增加Dimension}：\(d=12\)
  已是手工Feature的极限。继续增加Dimension面临Dimension灾难------更多维意味着更稀疏的采样，且人类无法可靠地Definition更多有物理意义的Descriptor
\item
  \textbf{PCA 降维}：PCA 优化的是
  \(\text{Var}(\varphi_1)\)（方差最大方向），而非
  \(I(\varphi_1; r)\)（与Error的Correlation性）。PCA
  的前几个主成分几乎总是描述全局结构差异（如晶胞体积变化），而非局域的Error模式
\item
  \textbf{UMAP/t-SNE}：可视化友好，但保留的是局部邻域结构，仍不直接针对
  \(r\) 优化
\end{itemize}

\textbf{Core洞察}：Problem不在于''Dimension不够''或''空间不平衡''，而在于\textbf{Goal错配}------现有Descriptor的设计Goal是''完整描述结构''，而非''解释Error''。

\paragraph{1.2.1 DIRECT sampling}\label{direct-sampling}

DIRECT (D-optimality-guided Iterative REgion Cluster Training) 是最接近
SCX 的Method：它对 SOAP Descriptor做 PCA
降维后聚类，在每个簇中采样。但其Core限制：

\begin{itemize}
\tightlist
\item
  SOAP Descriptor编码的是\textbf{局域化学环境的相似性}，而非Error模式
\item
  聚类边界由 SOAP 距离决定，而非Model失败边界
\item
  即使两个结构在 SOAP
  空间中相邻，它们的Error行为可能完全不同（例如：一个完美表面和一个部分重构的表面）
\end{itemize}

\paragraph{1.2.2 DP-GEN}\label{dp-gen}

DP-GEN 用 model deviation
作为不确定性度量，选择高偏差的构型加入Training集。其优点在于直接瞄准Error，但：

\begin{itemize}
\tightlist
\item
  \textbf{不解释}：\(\epsilon_{\text{dev}}\)
  是一个标量，告诉你''这个结构Error大''，但不告诉你''因为配位数小于 3.5
  且局域应变大于 3\%''
\item
  \textbf{无结构化State}：DP-GEN
  产生的是一组''高偏差构型''，而非可解释的State划分
\item
  这使得 DP-GEN
  的主动采样策略难以迁移到新材料------你只知道''更多Data更好''，但不知道''什么样的Data更好''
\end{itemize}

\paragraph{1.2.3 MTP D-optimality}\label{mtp-d-optimality}

MTP 用 D-optimality 准则衡量 extrapolation grade，但其：

\begin{itemize}
\tightlist
\item
  extrapolation grade
  是线性代数度量（设计Matrix的行列式），而非Model失败行为的直接度量
\item
  高 extrapolation grade
  并不意味着高Error------某些线性外推区域可能Model表现良好
\item
  同样不提供失败模式的物理解释
\end{itemize}

\paragraph{1.2.4
共同的根本缺陷}\label{ux5171ux540cux7684ux6839ux672cux7f3aux9677}

所有CompetitionMethod都隐含地Assumption''人类知道什么是正确的State划分''------但人类不知道。

\begin{itemize}
\tightlist
\item
  人知道 AlN 有 bulk、surface、defect 三种结构
\item
  人不知道的是：Model在''配位数 3.0-3.5 且局域应变 \textgreater{}
  3\%``的区域Error比平均值高 8 倍
\item
  后者才是真正需要成为IndependentState的划分
\end{itemize}

\subsubsection{2.4 两层的关系}\label{ux4e24ux5c42ux7684ux5173ux7cfb}

两层之间不是并列关系，而是\textbf{Layer选择和细化的关系}：

\begin{verbatim}
研究人员："我要做 AlGaN，包括 bulk/surface/defect，温度范围 300-1800K"
         |
         v
Layer 1：生成初始 DFT 采样网格（按人的Definition分类）
         |
         v
Training Single ACE baseline
         |
         v
Layer 2：算法计算 per-atom error -> 映射到候选Descriptor空间
         自动筛选最CorrelationDescriptor -> 在ErrorCorrelation子空间中聚类
         |

         |
         v

         |
         v

\end{verbatim}

这是子集选择的贪心最优解（对非负Independent项，排序选择即全局最优）。\(\square\)

\subsubsection{3.4 优势Theorem}\label{ux4f18ux52bfux5b9aux7406}

\textbf{Theorem 3.2（Error驱动State占优人类DefinitionState）}。令
\(\mathcal{S}_{\text{human}}\)
为人类先验知识Definition的State划分（如按晶体结构、化学组分分类），\(\mathcal{S}^*\)
为Error驱动发现的State划分。在相同的State数 \(K = |\mathcal{S}|\) 下：

\[\mathbb{E}_{s \sim \mathcal{S}^*}[\text{Var}(r \mid s)] \leq \mathbb{E}_{s \sim \mathcal{S}_{\text{human}}}[\text{Var}(r \mid s)]\]

即 error-driven states 的组内残差方差不超过人类DefinitionState的组内残差方差。

\textbf{Proof}。

\textbf{Lemma 3.2.1}。对人类Definition的任意State
\(s \in \mathcal{S}_{\text{human}}\)，其在 \(\varphi_1\) 空间中的表示
\(\mu_s = \mathbb{E}[\varphi_1(x) \mid x \in s]\) 可能包含大量与 \(r\)
无关的Dimension。

\textbf{Proof}。人类DefinitionState时使用的Feature
\(\psi(x)\)（如晶体结构、化学组分）可能与 \(\varphi_1\) 存在非线性关系
\(\varphi_1(x) = g(\psi(x)) + \epsilon\)。如果 \(g\) 的某些分量与 \(r\)
无关，则这些分量在聚类中作为噪声存在，增加组内Error方差。

\textbf{Lemma 3.2.2}。K-means 在投影子空间 \(D^*\) 上的GoalFunction：

\[\min_{\mathcal{S}} \sum_{s \in \mathcal{S}} \sum_{x \in s} \|\varphi_1(x)[D^*] - \mu_s\|^2\]

等价于在残差CorrelationDimension上的方差最小化。其中
\(\mu_s = \frac{1}{|s|} \sum_{x \in s} \varphi_1(x)[D^*]\)。

\textbf{Proof}。由于 \(D^*\) 的每个Dimension \(j\) 满足
\(\text{MI}(\varphi_1[:, j], r) > \tau\)，在 \(\varphi_1[:, D^*]\)
上的聚类直接捕捉了与 \(r\) 最Correlation的结构。K-means
的组内平方和最小化等价于在每个簇内最小化Feature方差，而由于这些Feature与 \(r\)
Correlation，Feature方差的减小意味着 \(r\) 条件方差的减小。

\textbf{主TheoremProof}。由 \(\mathcal{S}^*\) 的Definition，\(\mathcal{S}^*\)
直接优化 \(\text{Var}(r \mid s)\) 的上界（通过在 \(D^*\)
中聚类）。\(\mathcal{S}_{\text{human}}\) 在 \(\varphi_1\)
的全空间上聚类，包含了与 \(r\) 无关的''噪声Dimension''。由Lemma 3.2.1 和
3.2.2，\(\mathcal{S}^*\) 在 \(r\) 上的条件方差不大于
\(\mathcal{S}_{\text{human}}\)
的条件方差。等号成立当且仅当人类Definition的State恰好在 \(D^*\)
中是可分的。\(\square\)

\textbf{Corollary 3.2.1（严格优势）}。如果 \(\mathcal{S}_{\text{human}}\)
不是基于 \(r\) 的充分统计量，即存在
\(u \neq v \in \mathcal{S}_{\text{human}}\) 使得
\(\mathbb{E}[r \mid u] \neq \mathbb{E}[r \mid v]\) 但
\(\mu_u[D^*] = \mu_v[D^*]\)的概率小于 1，则不等号严格成立：

\[\mathbb{E}_{s \sim \mathcal{S}^*}[\text{Var}(r \mid s)] < \mathbb{E}_{s \sim \mathcal{S}_{\text{human}}}[\text{Var}(r \mid s)]\]

\subsubsection{3.5
与信息瓶颈的关系}\label{ux4e0eux4fe1ux606fux74f6ux9888ux7684ux5173ux7cfb}

两层State发现可以理解为\textbf{条件信息瓶颈}的迭代求解。信息瓶颈原理
(Tishby et al., 1999) 给出了最优表示 \(\tilde{X}\) 的变分Goal：

\[\min_{\Pi} I(X; \tilde{X}) - \beta I(\tilde{X}; Y)\]

在 SCX 的两层Framework中，这一原理被条件化为：

\[\min_{\Pi} I(\varphi_1; \tilde{X} \mid \mathcal{F}) - \beta I(\tilde{X}; r \mid \mathcal{F})\]

其中 \(\mathcal{F} = \{f_1, \dots, f_M\}\) 为专家集合，\(r\)
为残差向量。与标准信息瓶颈的Core区别：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \textbf{以 \(\varphi_1\) 替代 \(X\)}：表示从原始输入空间提升到 Layer 1
  嵌入空间，降低复杂度
\item
  \textbf{以残差 \(r\) 替代标签
  \(Y\)}：不是预测''结构是什么''，而是预测''Model在哪里失败''
\item
  \textbf{以专家 \(\mathcal{F}\)
  为条件}：State划分依赖于当前的专家集合，当专家更新时State可演化
\end{enumerate}

条件信息瓶颈的迭代求解通过 SCX 的 Phase A-E
流程序列Implementation。每轮迭代等价于一次信息瓶颈的变分更新： - Phase B 的聚类
\(\approx\) 压缩 \(I(\varphi_1; \tilde{X} \mid \mathcal{F})\) - Phase C
的定向采样 \(\approx\) 增加 \(I(\tilde{X}; r \mid \mathcal{F})\) - Phase
D 的Model更新 \(\approx\) 更新 \(\mathcal{F}\)，改变条件Distribution

\subsubsection{3.6
State演化的动力学}\label{ux72b6ux6001ux6f14ux5316ux7684ux52a8ux529bux5b66}

这是两层DescriptorFramework独有的理论Feature------State划分不固定。

\textbf{Definition 3.3（State演化算子）}。令 \(\mathcal{F}^{(t)}\) 为第 \(t\)
轮迭代后的专家集合，\(\tilde{X}^{(t)}\) 为对应的State划分。State演化算子
\(T\) Definition为：

\[\tilde{X}^{(t+1)} = T(\tilde{X}^{(t)} \mid \mathcal{F}^{(t+1)}, r^{(t+1)})\]

其中 \(r^{(t+1)}\) 是更新后的ModelError。

\textbf{学习轨迹}：

\begin{verbatim}
t=0: 人类DefinitionState S_human ≈ 信息瓶颈的"先验"State
t=1: 在 baseline Error上聚类 → S*(1)，发现"CN<3.5 区域Error高"
t=2: 在该区域补充 DFT → Error下降 → Error景观变化
t=3: 重新聚类 → S*(2)，可能发现"新的高Error区域：sp2 过渡态"
...
t=∞: Error景观平坦 → State划分稳定（收敛）
\end{verbatim}

\textbf{Theorem 3.3（State收敛性）}。如果Error景观的 Lipschitz 常数 \(L_{r}\)
有限，且每次迭代补充的 DFT 样本数 \(N_t\) 满足
\(\sum_{t=1}^\infty N_t = \infty\)，则State划分 \(\tilde{X}^{(t)}\)
几乎必然收敛到一个固定点。

\textbf{Proof概要}。将State演化视为 \(\varphi_1\) 空间中一个 Markov
链。由于每轮迭代补充的 DFT Data覆盖了高Error区域，\(r^{(t)}\)
逐点递减。当 \(r^{(t)}\) 收敛到平稳过程时，\(D^*\)
和聚类Result随之稳定。\(\square\)

\subsubsection{5.2 DIRECT
的局限性}\label{direct-ux7684ux5c40ux9650ux6027}

DIRECT 是最接近 SCX+两层Descriptor的CompetitionMethod，但根本区别在于聚类空间的选取：

\begin{itemize}
\tightlist
\item
  DIRECT 用 SOAP-PCA 空间聚类 → 优化的是''结构相似性''
\item
  SCX 用 \(\Phi_1[:, D^*]\) 空间聚类 → 优化的是''Error模式相似性''
\end{itemize}

\textbf{两种空间的几何差异}：设 \(d_{\text{SOAP}}(x_i, x_j)\) 为 SOAP
距离，\(d_{\text{error}}(x_i, x_j) = \|\phi(x_i)[D^*] - \phi(x_j)[D^*]\|\)
为Error空间距离。存在结构相似但Error不相似的情况（如：两个不同表面的 SOAP
距离小但力Error差异大），也存在结构不相似但Error相似的情况（如：表面和缺陷在某些DescriptorDimension上Error模式相同）。

DIRECT 无法区分这些情况，因为它没有使用Error反馈。

\subsubsection{5.3 DP-GEN
的局限性}\label{dp-gen-ux7684ux5c40ux9650ux6027}

DP-GEN 最大的Contribution是认识到''Model偏差是采样信号''，但它止步于标量信号：

\begin{itemize}
\tightlist
\item
  DP-GEN:
  \(\{\epsilon_{\text{dev}}(x) > \tau\} \rightarrow \text{add to training}\)
\item
  SCX:
  \(\{\epsilon(x), \phi_1(x)\} \rightarrow \text{cluster in error space} \rightarrow \text{discover pattern: "CN<3.5, stretch>10%"} \rightarrow \text{targeted sampling}
  \)
\end{itemize}

用信息论的术语说： - DP-GEN 只保留了 \(I(x; r)\)（知道''Error大''） - SCX
保留了 \(I(\phi_1; r)\) 的结构化分解（知道''在配位数 \textless{} 3.5
时Error大''）

\subsubsection{5.4 SCX
的独特Positioning}\label{scx-ux7684ux72ecux7279ux5b9aux4f4d}

SCX+两层Descriptor是唯一的\textbf{完全Error驱动 + 可解释State划分 +
条件专家路由}Framework。

CompetitionMethod可以归为三类： 1. \textbf{结构驱动}（DIRECT 等）→
State划分基于结构相似性，不关联Error 2.
\textbf{标量Error驱动}（DP-GEN、MTP）→ 使用Error信号但不结构化 3.
\textbf{SCX} → 使用Error信号且\textbf{结构化地发现失败模式}

这是 SCX
在新材料开发中的根本Competition力：当材料遇到全新的化学环境时，人类经验不可靠，只有''算法从Error中自动学习的失败模式''才能可靠地指导采样和专家路由。

是几何度量（设计Matrix的行列式），不与特定物理Descriptor关联；SCX
的ErrorAnalysis直接关联到可解释的物理量（CN、键长、局部应变）。

\textbf{根本差异}：SCX
是唯一将Error信号结构化并用于State划分的Method------不是''人DefinitionState后评估Error''，而是''从ErrorDistribution中自动发现State''。
