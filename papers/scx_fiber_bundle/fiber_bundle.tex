\\item $\\verts$: a finite set of vertices, $|\\verts| = n$;\n223|    \\item $\\edgs \\subseteq \\verts \\times \\verts$: a set of directed edges, $|\\edgs| = m$.\n224|\\end{itemize}\n225|Each We allow multiple edges between the same pair of vertices\n227|but not self-loops ($u \\neq v$).\n228|\\end{definition}\n229|\n230|\\subsection{Discrete Differential Forms: Vertex Functions and Edge Assignments}\n231|\n232|The discrete exterior calculus on graphs relates scalar functions ($0$-forms)\n233|to edge assignments ($1$-forms).\n234|\n235|\\begin{definition}[Vertex Space and Edge Space]\n236|Define the following real vector spaces:\n237|\\begin{itemize}\n238|    \\item {\\bf $0$-form space} $\\Omega^0(\\grph)$: all vertex functions $f: \\verts \\to \\R$,\n239|        dimension $n$. Denoted as vector $\\mathbf{f} = (f_1, \\dots, f_n)^T \\in \\R^n$.\n240|    \\item {\\bf $1$-form space} $\\Omega^1(\\grph)$: all edge assignments $\\alpha: \\edgs \\to \\R$,\n241|        dimension $m$. Denoted as vector $\\boldsymbol{\\alpha} = (\\alpha_1, \\dots, \\alpha_m)^T \\in \\R^m$.\n242|\\end{itemize}\n243|For $d$-dimensional vector-valued forms, we take the direct sum of $d$ copies:\n244|$\\Omega^k(\\grph; \\R^d) := \\bigoplus_{a=1}^d \\Omega^k(\\grph)$.\n245|\\end{definition}\n246|\n247|\\subsection{Discrete Exterior Derivatives}\n248|\n249|\\begin{definition}[Incidence Matrix and $d_0$]\n250|The {\\bf incidence matrix} $B \\in \\R^{m \\times n}$ of $\\grph$ is defined by:\n251|for each edge $e = (u \\to v)$ and each vertex $w$,\n252|\\[\n253|    B_{e,w} = \n254|    \\begin{cases}\n255|        +1, & \\text{if } w = v \\text{ (target)}, \\\\\n256|        -1, & \\text{if } w = u \\text{ (source)}, \\\\\n257|        0,  & \\text{otherwise}.\n258|    \\end{cases}\n259|\\]\n260|\n261|The {\\bf discrete exterior derivative} $d_0: \\Omega^0(\\grph) \\to \\Omega^1(\\grph)$\n262|is defined as $d_0 := B$, i.e., for $f \\in \\Omega^0(\\grph)$ and edge $e = (u \\to v)$:\n263|\\[\n264|    (d_0 f)_e = f_v - f_u.\n265|\\]\n266|In words, $d_0 f$ computes the difference of $f$ along each edge.\n267|\\end{definition}\n268|\n269|\\begin{definition}[Discrete Exterior Derivative $d_1$ on Loops]\n270|Let $\\mathcal{L}$ be a set of {\\bf elementary loops} of $\\grph$\n271|(e.g., the cycle basis determined by the complement edges of a spanning tree).\n272|Let $C \\in \\R^{|\\mathcal{L}| \\times m}$ be the {\\bf cycle matrix},\n273|where for each loop $\\gamma \\in \\mathcal{L}$ and each edge $e \\in \\edgs$:\n274|\\[\n275|    C_{\\gamma,e} = \n276|    \\begin{cases}\n277|        +1, & \\text{if edge } e \\text{ follows the positive direction of loop } \\gamma, \\\\\n278|        -1, & \\text{if edge } e \\text{ follows the negative direction of loop } \\gamma, \\\\\n279|        0,  & \\text{otherwise}.\n280|    \\end{cases}\n281|\\]\n282|\n283|The {\\bf discrete exterior derivative} $d_1: \\Omega^1(\\grph) \\to \\Omega^2(\\grph)$\n284|is defined as $d_1 := C$, where $\\Omega^2(\\grph) \\cong \\R^{|\\mathcal{L}|}$ is the loop space.\n285|For $\\alpha \\in \\Omega^1(\\grph)$ and loop $\\gamma = (e_1, e_2, \\dots, e_k)$:\n286|\\[\n287|    (d_1 \\alpha)_\\gamma = \\sum_{e \\in \\gamma} \\sigma_e \\, \\alpha_e,\n288|\\]\n289|where $\\sigma_e \\in \\{+1, -1\\}$ is the sign of edge $e$ relative to the loop orientation.\n290|\\end{definition}\n291|\n292|\\begin{theorem}[Fundamental Identity: $d_1 \\circ d_0 = 0$]\n293|\\label{thm:d1d0}\n294|The incidence and cycle matrices satisfy:\n295|\\[\n296|    d_1 \\circ d_0 = C B = 0.\n297|\\]\n298|Equivalently, $\\im(d_0) \\subseteq \\ker(d_1)$.\n299|This is the discrete analog of $\\dif^2 = 0$ in the continuous setting.\n300|\n301|{\\bf Proof.} For any $f \\in \\Omega^0(\\grph)$ and any loop\n302|$\\gamma = (v_0 \\to v_1 \\to \\cdots \\to v_k = v_0)$:\n303|\\[\n304|    (d_1 d_0 f)_\\gamma = \\sum_{i=0}^{k-1} (f_{v_{i+1}} - f_{v_i}) = f_{v_k} - f_{v_0} genuine Hodge theory, we need inner products and adjoint operators.\n312|\n313|\\begin{definition}[Inner Products and Adjoint Operators]\n314|Define the standard inner product on $\\Omega^0(\\grph)$ (with vertex weights\n315|$\\mathbf{w}_0 \\in \\R^n_{>0}$):\n316|\\[\n317|    \\langle f, g \\rangle_0 := \\sum_{v \\in \\verts} w_{0,v} \\, f_v \\, g_v.\n318|\\]\n319|On $\\Omega^1(\\grph)$ (with edge weights $\\mathbf{w}_1 \\in \\R^m_{>0}$):\n320|\\[\n321|    \\langle \\alpha, \\beta \\rangle_1 := \\sum_{e \\in \\edgs} w_{1,e} \\, \\alpha_e \\, \\beta_e.\n322|\\]\n323|\n324|The {\\bf formal \\langle f, \\dzeroT \\alpha \\rangle_0,\n328|    \\quad \\forall f \\in \\Omega^0(\\grph), \\; \\alpha \\in \\Omega^1(\\grph).\n329|\\]\n330|\n331|Explicitly, $\\dzeroT = W_0^{-1} B^T W_1$, \\Lzero := \\dzeroT d_0 = W_0^{-1} B^T W_1 B.\n345|\\]\n346|In the unweighted case, $\\Lzero = B^T B$ is the classical combinatorial Laplacian:\n347|\\[\n348|    (\\Lzero)_{uv} = \n349|    \\begin{cases}\n350|        \\deg(v), & u = v, \\\\\n351|        -1,      & (u, v) \\text{ or } (v, u) \\in \\edgs, \\\\\n352|        0,       & \\text{otherwise}.\n353|    \\end{cases}\n354|\\]\n355|\n356|The {\\bf graph $1$-Laplacian} (full Hodge Laplacian) is:\n357|\\[\n358|    \\Lone := d_0 \\dzeroT + \\doneT d_1 = B W_0^{-1} B^T W_1 + C^T W_2 C W_1^{-1},\n359|\\]\n360|where $W_2$ is an (optional) weight matrix on the loop space $\\Omega^2(\\grph)$.\n361|In the unweighted case this reduces to $\\Lone = B B^T + C^T C$.\n362|\\end{definition}\n363|\n364|\\begin{theorem}[Discrete Hodge Decomposition]\n365|\\label{thm:hodge_decomp}\n366|For any connected graph $\\grph$, $\\Omega^1(\\grph)$ admits the orthogonal decomposition:\n367|\\[\n368|    \\Omega^1(\\grph) = \\im(d_0) \\oplus \\ker(\\Lone) \\oplus \\im(\\doneT),\n369|\\]\n370|where:\n371|\\begin{itemize}\n372|    \\item $\\im(d_0)$: {\\bf exact \\Omega^1(\\grph)$ has a unique decomposition:\n378|\\[\n379|    \\alpha = d_0 f + h + \\doneT \\beta,\n380|\\]\n381|where $f \\in \\Omega^0(\\grph)$, $h \\in \\ker(\\Lone)$, $\\beta \\in
