# arXiv Template and SCX Compatibility Notes

This directory is for arXiv submission preparation for `papers/scx_climate/main.tex`.

I checked the official arXiv submit/help pages on 2026-07-01:

- https://arxiv.org/help/submit
- https://info.arxiv.org/help/submit_tex.html
- https://info.arxiv.org/help/faq/texlive.html
- https://info.arxiv.org/help/faq/multilang.html

No downloadable official LaTeX template archive was exposed from the submit page during this check. As a fallback, `template.tex` is a minimal `article.cls`-based starting point that matches the requested target: plain `article`, standard math packages, theorem environments, and no journal/referee formatting.

The compatibility report is in `ISSUES.md`. No files under `papers/` were modified.
