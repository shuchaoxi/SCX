28|### 1. Theorem 1：F1收敛界
29|
30|#### 漏洞1.1 [Codex: 致命] Hoeffding独立性假设不足以支撑
31|
32|**Codex 声称**: M_eff < M 因为专家相关性，且退化公式未被纳入定理陈述。
33|
34|**Meta 判断**: **部分正确，严重度被夸大**
35|
36|- 专家相关性确实存在，不应被忽视 ✓
37|- 但 Hoeffding 界在弱相关下可以通过 M_eff = M/(1+(M−1)ρ̄) 修正，这在文献中是标准处理（参见 Liang & Zeger 1986 GEE）
38|- Codex 说"公式未经证明"——这是对的，但 ρ̄ 的一致估计可以通过 bootstrap 或 jackknife 获得，属于统计标准做法
39|- **不致命的原因**: 即使 M_eff < M，指数收敛性 e^{-2M_eff·Δ²} 仍保持——收敛速度变慢但不是零。定理的结构不变，只是常数需要调整
40|- **建议**: 在定理陈述中加入 M_eff 注释，提供 ρ̄ 的估计方法引用。降级为**重大**
41|
42|#### 漏洞1.2 [Codex: 重大] 分母放缩过度
43|
44|**Meta 判断**: **正确** ✓
45|
46|- 在 M 较小时确实过于保守。但这属于"界不紧"而非"界错误"
47|- 保守下界仍然是下界——它不会导致假阳性（声称成立实际不成立）
48|- **建议**: 添加注释说明小 M 时的保守性。可接受
49|
50|#### 漏洞1.3 [Codex: 重大] Chernoff收紧KL方向
51|
52|**Meta 判断**: **错误**
53|
54|- Codex 的批评模糊不清。"两个KL的方向定义一致但被积变量不同"——这恰恰是正确的：KL(p∥q) 和 KL(p∥q') 方向一致（都是 p 在左），被积变量自然不同因为是不同的分布对
55|- 这就像说"两个积分的积分变量不同所以不对"——积分变量当然不同。Codex 混淆了形式不一致和实质不一致
56|- **建议**: 驳回。无需修改

60|### 2. Theorem 2：弱特征失效下界
61|
62|#### 漏洞2.1 [Codex: 致命] TV界从联合到条件的传递不成立
63|
64|**Codex 声称**: TV(P(·|Z=1), P̃(·|Z=1)) 不能被 TV(P,P̃)/η 所界
65|
66|**Meta 判断**: **错误——Codex 的反例不成立**
67|
68|详细分析：
69|```
70||P(A|Z=1) − P̃(A|Z=1)| = |P(A∩{Z=1})/P(Z=1) − P̃(A∩{Z=1})/P̃(Z=1)|
71|```
72|
73|Codex 的反例：设 P 和 P̃ 仅在 Z=0 上差异极大，在 Z=1 上完全相同。
74|- 此时 P(Z=1) = P̃(Z=1) = η（因为 P 和 P̃ 在 {Z=1} 上一致）
75|- |P(A∩{Z=1}) − P̃(A∩{Z=1})| = 0（在 Z=1 上完全一致）
76|- TV(P,P̃) 可以很大（在 Z=0 上差异大）
77|- 但我们有 |P(A|Z=1) − P̃(A|Z=1)| = 0/η = 0 ≤ TV(P,P̃)/η ✓
78|
79|Codex 说"除以η的放缩在此无效"——但不等式 0 ≤ 大数/η 恒成立。这个反例恰恰**证明了**上界有效。
80|
81|**真正的潜在问题**（Codex 没提到的）：如果 P(Z=1) ≠ P̃(Z=1)，则需要不同的处理。但在此定理上下文中，数据分布 P 和退化的 P̃ 在 Z 的边际上一致（因为 Z 是观测指标），这个假设是合理的。
82|
83|**建议**: Codex 这个批评是**错的**。驳回。"致命"标记完全不成立。

164|#### 漏洞5.2 [Codex: 重大] 参数位移消失的证明自相矛盾
165|
166|**Meta 判断**: **Codex 混淆了条件——此批评可能错误**
167|
168|- Codex 声称 ∑α_s < ∞ 与 Robbins-Monro ∑α_t = ∞ 矛盾
169|- 但 Robbins-Monro 条件是：α_t > 0, ∑α_t = ∞, ∑α_t² < ∞
170|- 取 α_t = 1/t：∑α_t = ∞ ✓, ∑α_t² = π²/6 < ∞ ✓
171|- 如果论文写的是 ∑α_s < ∞（绝对可和），那确实错误
172|- 但标准表述是：**尾部和** ∑_{s=t}^∞ α_s → 0 当 t → ∞——这要求 α_t → 0，不要求 ∑α_t < ∞
173|- 如果 α_t = 1/t，∑_{s=t}^∞ 1/s 发散（调和级数尾部发散），所以这个论证确实有问题！
174|- **实际情况**: 需要回到原论文看具体是怎么写的。如果确实声称了 ∑α_s < ∞，Codex 是对的
175|- **建议**: 标记为**待核实**。需要检查 spring_config/main.tex 的实际推导
