\subsection{问题}\label{ux95eeux9898}

给定 M 个专家 \{f\_1, \ldots, f\_M\}，输入空间 X，标签空间 Y。

\textbf{传统做法}：给每个专家一个全局准确率 R\_m = E{[}ℓ(f\_m(x),
y){]}。

\textbf{问题}：异质输入空间中，专家的准确度在不同区域差异很大。全局排序无法反映这一点。

\subsection{核心洞察}\label{ux6838ux5fc3ux6d1eux5bdf}

\begin{verbatim}
R_m 是不够的。正确的对象是 R_m(s)。
\end{verbatim}

其中 s: X → S 是状态映射，将输入分配到潜在状态。

\subsection{算法: State-Certified Active
Learning}\label{ux7b97ux6cd5-state-certified-active-learning}

\begin{verbatim}
输入: L (已标注), U (未标注), E (专家), φ (表示映射)

1. 在 φ(X) 中构造状态 S
2. 估计 C̄(s), ρ(s), D(s)
3. 估计 R̂_m(s) 对所有专家 m 和状态 s
4. 分类每个状态: valuable / redundant / noisy / expert-dependent
5. 对每个状态选择动作:
   a*(s) = arg max_a U(a, s)
   其中 a ∈ {acquire, relabel, downweight, discard, route}
6. 执行动作，更新模型和状态图
7. 重复
\end{verbatim}

\begin{quote}
步骤 2 中，D(s) 的计算现已不需要真实标签（使用 Theorem 1 的 C̄(s) 替代
r̄(s)）。C̄(s) 直接来自多专家一致性，使算法可在完全无监督环境下运行。
\end{quote}
