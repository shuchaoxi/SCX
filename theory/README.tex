\section{SCX Mathematical Framework Overview}\label{scx-mathematical-framework-overview}

\begin{quote}
State-Conditioned Expertise Theory --- From the reliability problem of multi-expert systems to the definition of data value
\end{quote}

\textbf{Last updated}: 2026-06-27

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{Problem}\label{problem}

Given M experts \{f\_1, \ldots, f\_M\}, input space X, label space Y.

\textbf{Traditional approach}: assign each expert a global accuracy R\_m = E{[}ℓ(f\_m(x), y){]}.

\textbf{Problem}: In heterogeneous input spaces, expert accuracy varies greatly across different regions. Global rankings fail to reflect this.

\subsection{Core Insight}\label{core-insight}

\begin{verbatim}
R_m is insufficient. The correct object is R_m(s).
\end{verbatim}

where s: X → S is the state mapping, assigning inputs to latent states.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{Three Core Definitions}\label{three-core-definitions}

\subsubsection{1. State-Conditioned Expert Risk}\label{state-conditioned-expert-risk}

\begin{verbatim}
R_m(s) = E_{x~P(·|s)} [ ℓ(f_m(x), f*(x)) ]
\end{verbatim}

\subsubsection{2. SCX Reliability}\label{scx-reliability}

\begin{verbatim}
SCX_m(s) = P( ℓ(f_m(x), y) < τ | x ∈ s )
\end{verbatim}

\subsubsection{3. State Data Value (Deprecated)}\label{state-data-value-deprecated}

\begin{quote}
\textbf{2026-06-27}: V(s) = r̄(s) · ρ(s) · L(s) · {[}1 - D(s){]} · max\_m SCX\_m(s) has been deprecated. Its functionality has been replaced by more precise component decomposition: redundancy score D(s) handles compression decisions, consensus score C̄(s) handles noise detection, and state-conditioned risk R̂\_m(s) handles expert routing.
\end{quote}

\subsection{Algorithm: State-Certified Active Learning}\label{algorithm-state-certified-active-learning}

\begin{verbatim}
Input: L (labeled), U (unlabeled), E (experts), φ (representation mapping)

1. Construct states S in φ(X)
2. Estimate C̄(s), ρ(s), D(s)
3. Estimate R̂_m(s) for all experts m and states s
4. Classify each state: valuable / redundant / noisy / expert-dependent
5. Select action for each state:
   a*(s) = arg max_a U(a, s)
   where a ∈ {acquire, relabel, downweight, discard, route}
6. Execute actions, update model and state graph
7. Repeat
\end{verbatim}

\begin{quote}
In step 2, computing D(s) no longer requires true labels (using Theorem 1's C̄(s) in place of r̄(s)). C̄(s) comes directly from multi-expert consensus, allowing the algorithm to operate in a fully unsupervised setting.
\end{quote}
