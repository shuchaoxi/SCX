## 1. Theorem 1（独醒不太可能定理）：F1收敛界

**文件**: `S1_thm1_noise_detection.tex`  
**核心声称**: F1 ≥ 1 − (1/η)∑ρ_s exp(−2MΔ_s^2)

### 漏洞1.1 [致命] Hoeffding独立性假设(A2')不足以支撑推论

**问题**: 作者声称"由(A1)不相交训练集可得(A2)条件独立"。该推理链为：

> A1: D_m ⟂ D_m' → f_m ⟂ f_m' → e_m ⟂ e_m' | x

该推导存在跳跃。即使训练集D_m相互独立，f_m是D_m的**确定性函数**，f_m(x)和f_m'(x)在给定x条件下的联合分布可能相关——因为不同的训练算法可能在相似的数据分布上学习到相关的误差模式。独立训练数据保证的是训练过程独立，而非模型输出条件独立。

**具体反例**：设两个专家在同一数据分布的不同子集上训练。即使子集不相交，若两者都使用相同的模型架构和学习算法，其错误模式会因为归纳偏置的相似性而正相关。Hoeffding不等式要求的独立性与实际存在的相关性之间的矛盾会导致：

- Hoeffding界exp(−2MΔ²)在实际中过于乐观
- 有效专家数M_eff < M

作者虽然提到了相关性退化M_eff=M/(1+(M−1)ρ̄)，但：
1. 该公式未经证明（仅为启发式声称）
2. ρ̄的估计方法未给出
3. 更严重的是——该退化公式**未被纳入定理陈述**。定理声明的F1下界仍使用M而非M_eff

**修复建议**: 必须在定理陈述中将M替换为M_eff或M/(1+(M−1)ρ̄)，并提供ρ̄的一致估计方法。或采用更弱的集中不等式（如Bernstein不等式处理弱相关情况）。

### 漏洞1.2 [重大] F1下界推导中分母的放缩过度

**位置**: 第396-398行

```
F1 ≥ 1 − (ηδ₁ + (1−η)δ₂)/(η(2−δ₁) + (1−η)δ₂)
   ≥ 1 − (ηδ₁ + (1−η)δ₂)/η    (因为分母≥η)
```

该放缩在M较小时极为保守。当δ₁, δ₂不小时（如M=10, Δ=0.2时δ₁≈0.45），分母远大于η，用η替代导致下界过于宽松。在实际参数范围内，该"下界"实际上远低于真实的F1。

### 漏洞1.3 [重大] Chernoff收紧中的KL方向不一致

**位置**: 第488-496行

Chernoff收紧声称：
```
P(C ≤ θ | noise, x, c) ≤ exp(−M·KL(θ ∥ 1−C_bal·μ_s/(K−1)))
```

此处KL方向正确（θ < p时KL(θ∥p)），但在F1下界公式(507-508)中，两个指数项的KL方向应当对称——然而在干净样本侧使用KL(θ∥μ_s)，在噪声样本侧使用KL(θ∥p₁)，这两个KL的方向定义一致但被积变量不同。读者容易混淆。

### 漏洞1.4 [轻微] 符号混乱

- Δ_s在Lemma 1和Theorem 1中有不同的定义（有无C_bal因子）
- 方程(343)中"ηδ₁ + (1−η)δ₂"的η与噪声率η符号重叠，建议分别标记

### 漏洞1.5 [风格] 数值示例误导

第448行显示F1 > 0.976，使用了全篇最优参数（η=0.1, μ_s=0.2, M=20, K=10, C_bal=1）。在实际中，μ_s≈0.2要求专家在CIFAR-10级别任务上仅犯20%错误——这是强假设。作者随后承认μ_s≈0.45时F1≥0.18，而0.18的"保证"在实际中毫无意义。

## 2. Theorem 2（强人所难定理）：弱特征失效下界

**文件**: `S2_thm2_weak_features.tex`  
**核心声称**: F1(h_SCX) ≤ F1_base + C_F·√(δ/2)

### 漏洞2.1 [致命] TV界从联合分布到条件分布的传递不成立

**位置**: Step 5, 第378-389行

```
|P(A|Z=1) − P̃(A|Z=1)| ≤ TV(P, P̃)/η
```

该推导为：
```
|P(A∩{Z=1}) − P̃(A∩{Z=1})|/η ≤ TV(P, P̃)/η
```

这里使用了**全变差在事件上的上界**，即|P(E) − P̃(E)| ≤ TV(P, P̃)。这是正确的。但问题在于：P(·|Z=1)和P̃(·|Z=1)是两个条件分布，它们之间的TV不能简单除以η来上界。

**正确的TV传递**应为：TV(P(·|Z=1), P̃(·|Z=1))没有被|P(E)−P̃(E)|/η所界。考虑反例：设P和P̃仅在Z=0的样本上差异极大，在Z=1的样本上完全相等。此时TV(P,P̃)可能很大，但TV(P(·|Z=1), P̃(·|Z=1))=0。除以η的放缩在此无效。

该错误直接导致AUC和PRAUC界的推导失去基础。

**修复建议**: 必须使用条件TV的直接界，或额外假设噪声指标Z与φ,S独立（当前框架未提供此假设）。

### 漏洞2.2 [重大] F1的Lipschitz常数推导不严谨

**位置**: Step 5, 第418-436行

声称"F1是(TP,FP,FN)的Lipschitz函数，C_F ≤ 2"。但F1 = 2TP/(2TP+FP+FN)在TP+FP+FN可能为零时无定义。即使在技术工作范围内（精度召回率≥0.1），作者仅通过偏导数界的粗糙估计得到C_F ≤ 2，未给出完整的Lipschitz常数证明。

具体地：
```
|∂F1/∂TP| ≤ 2/min(TP,1)
```
此处min(TP,1)恒为TP（因为TP≤1），即|∂F1/∂TP| ≤ 2/TP。在TP≈0.1时该界为20，远非2。

作者需要更为谨慎地处理分母上的奇异性。

### 漏洞2.3 [重大] Pinsker不等式的使用

Pinsker: TV(P,P̃) ≤ √(KL(P∥P̃)/2)。但KL(P∥P̃) ≠ I(φ(X);S)。实际上：

KL(P(φ,S) ∥ P(φ)P(S)) = I(φ(X);S)

然而，Pinsker不等式要求KL(P∥P̃)**或**KL(P̃∥P)。此处KL(P∥P̃)=I(φ;S)=δ成立，这是正确的。但作者在第328行将P(φ,S)与P(X,Y,{f_m})的联合分布混为一谈。KL在联合分布全体上的分解需要明确——如果处理的是(X,Y,{f_m})的完整联合分布，TV的传递需要更细致的论证。

### 漏洞2.4 [轻微] 符号不一致

第172-174行定义的ε_φ在第59-62行定义，但Corollary 4（第564-574行）使用了ε_φ ≈ 0.5的讨论与定理陈述中δ的nats单位不一致。

## 3. Theorem 3（老实人定理）：噪声-困难不可区分性

**文件**: `S3_thm3_unidentifiability.tex` + `taxonomic_nn/main.tex`

### 漏洞3.1 [重大] "当且仅当"声称过强

**位置**: 第30-37行

Theorem 3声称两个世界产生全同观测分布**当且仅当**η_err = η_amb。但定理实际证明的是：

- **如果**η_err = η_amb = η，则观测等价（充分性）
- **并未证明**仅当（必要性）

必要性需要证明：如果观测等价，则η_err = η_amb。该方向未被证明。考虑η_err ≠ η_amb但通过更复杂的构造（如调整专家模型参数）可能仍使观测等价。该"定理"的"当且仅当"部分仅对构造的特定世界成立——这不是一个关于所有可能世界对的定理。

**修复**: 将"当且仅当"改为"当"。

### 漏洞3.2 [重大] Fano证明中的I(W;D_n)=0仅对构造成立

**位置**: Lemma "观测数据无信息性"（taxonomic_nn/main.tex 第218-227行）

作者声称I(W;D_n)=0，因为P_noise(D_n)=P_hard(D_n)对所有D_n成立。这仅在**作者的构造世界**中成立——该构造已被精心设计使得该性质成立。但在一般设定下（任意M≥1个专家，任意状态空间），互信息可能不为零。

Fano证明声称"揭示了不可区分性的本质是观测数据与真实世界的互信息为零"——但该本质是作者通过构造强加的，而非推导出来的。

### 漏洞3.3 [重大] 逐样本Fano界缺失因子

**位置**: Lemma "逐样本Fano下界"（taxonomic_nn/main.tex 第230-254行）

```
P(z ≠ z*) ≥ 1 − I(z*;D_n)/(|A_amb|log2) − 1/|A_amb|
```

此处使用了Fano不等式的一个变体，但标准的Fano界是：
```
P_e ≥ (H(W) − I(W;D_n) − log2)/log|W|
```

在二元情况下有|W|=2。但逐样本界中使用了2^|A_amb|作为可能模式数——这对应于块误差。从块误差到逐样本期望误差的转换：

```
E[(1/k)∑₁ᵏ 1{z_i≠z*_i}] ≥ (1/2)P(z≠z*)
```

该不等式要求某种对称性（所有坐标在错误发生时等可能出错），但该对称性未被验证。实际上，在World A和World B的构造中，某些样本（状态s_2中的样本）完全可区分，误差可能集中在特定坐标上。

### 漏洞3.4 [轻微] 双重证明的一致性

构造性证明给出精确界ρ·min(η_err,η_amb)/2，Fano证明给出渐近界ρη/2。一致性仅在η_err=η_amb时成立，但作者并未明确讨论这一限制。

## 4. Theorem 4（极定理）：精确常数极小极大最优

**文件**: `S4_thm4_exact_constant_minimax.tex`

### 漏洞4.1 [重大] Bahadur-Rao条件未验证

Bahadur-Rao定理用于格点分布（lattice distribution）时需要处理格点修正因子。对于Bernoulli分布（自然格点间距为1/M），准确的Bahadur-Rao展开为：

```
P(S_M ≥ k) ~ exp(−M·KL(θ∥p)) / (√(2πMθ(1−θ)) · (1−e^{−λ*}))
```

其中(1−e^{−λ*})是格点修正因子。作者声称"格点修正被吸收到O(1/M)项中"（第192行），但这一处理只有在λ*很小时才有效。在定理的实际参数范围内（p₀和p₁差距大时λ*可不小），忽略格点修正在理论上是严重的。

### 漏洞4.2 [重大] O(1/M)余项的一致控制

第279行声称余项O(e^{−2M min(κ₀,κ₁)}/M)"渐近可忽略"。但该余项依赖于阈值θ的选择，在自适应阈值θ†偏移θ*O(1/M)的情况下，泰勒展开的余项可能累积。特别是：

- KL展开：KL(θ*+δ∥p₀) = κ + λ₀*δ + δ²/(2θ*(1−θ*)) + O(δ³)
- 当δ=O(1/M)，O(δ³)=O(1/M³)，乘以M后为O(1/M²)，看起来安全
- 但O(δ³)项的系数依赖于p₀,p₁，当p₀→0或p₁→1时可能爆炸

### 漏洞4.3 [重大] 自适应阈值FPR/FNR展开的因子失误

**位置**: 第424-433行

FPR和FNR的渐近展开中，θ†代替了θ*：

```
FPR_M(θ†) ∼ e^{−Mκ}/(λ₀*√(2πMθ*(1−θ*)))·((1−η)/η)^{−λ₀*/D*}
```

但Bahadur-Rao公式中的分母使用θ(1−θ)，在θ†处应使用θ†(1−θ†)而非θ*(1−θ*)。差异为O(1/M)，不影响极限但影响有限样本界。作者在陈述中对此进行了隐式替换。

### 漏洞4.4 [重大] Neyman-Pearson归约中的均匀o(1)声称

**位置**: Lemma E Part 1, 第543行

"the o(1) term is uniform over decision rules"（o(1)项对决策规则族一致）

这是一个极强的声称。Lemma B给出的F1展开余项依赖于TPR和FPR的具体值，而不同的检测器产生不同的TPR/FPR对。一致性的验证需要额外的论证（如泛函delta方法），但作者未提供。

### 漏洞4.5 [轻微] C_min定义的不自洽

第493-496行：
```
C_min/η = (1/2)((1−η)/η)^s · (1/λ₀*+1/|λ₁*|)/√(θ*(1−θ*))
```

与第75-79行的定义含有因式η/2，导致C_min中包含η因子。但定理声称C_min是"常量"——它依赖于η，不是绝对的常数。这主要是命名问题。

## 5. Spring SE-1/SE-2（固定点收敛+完备界）

**文件**: `spring_config/main.tex` + `06_fixed_point_convergence.tex`

### 漏洞5.1 [致命] Lyapunov函数Ψ的正确性依赖于未证明的C10/C11

Lyapunov下降声称（Lemma SE-1.1）：

```
E[Ψ(S_{t+1},θ_{t+1})|F_t] ≤ Ψ(S_t,θ_t) − η_t
```

其中Ψ定义在参考集M₀上。但实际算法在M_t（非M₀）上采样。重要性权重w=P₀/P_S_t被用于修正分布偏移——但这假设了P_S_t(x)>0对所有支撑集中的x成立，且||w||_∞有界（C11）。

问题在于：如果M₀是训练前采集的参考集，那么P_S_t随着S_t的演化可能与P₀（即原始数据分布）越来越不同。这意味着重要性权重w(x)=P₀(x)/P_S_t(x)可能无界，使得随机梯度方差发散。作者承认这需要ε_explore>0（C9），但从C9到有界w的推理假设了P₀的所有区域都被均匀覆盖——这在参考集是有限集时未必成立。

### 漏洞5.2 [重大] 参数位移消失的证明自相矛盾

**位置**: Lemma SE-1.3, spring_config/main.tex 第448-464行

```
||θ_{t'} − θ_t|| ≤ G·∑_{s=t}^{∞} α_s → 0
```

作者声称因为∑α_s < ∞所以尾部求和趋于零——但这与Robbins-Monro条件∑α_t = ∞（C4）直接矛盾。∑α_t = ∞意味着α_t不绝对可和，尾部求和发散。

作者可能意指"由Lyapunov下降确保的参数净移动有限"，但这正是待证明的——不能用作证明的手段。该循环推理破坏了整个Cauchy序列论证。

### 漏洞5.3 [重大] 条件C1-C11的分类：必要vs装饰性

作者将条件分为"核心条件"(C1-C7)和"充分性额外条件"(C8-C11)。然而：

- **C1**（有限覆盖维数）：通过ε-net构造，声称"对任何固定ε>0，可区分配置数有限"。但有限覆盖维数d_φ < ∞ ≠ 配置空间有限。C1真值为d_φ < ∞永远成立（任何实际嵌入空间都有有限维度），因此C1实质上是自动满足的——问题在于N_ε = O((L_S/ε)^{d_φ})，当ε→0时N_ε→∞。该假设置换并没有真正解决问题。

- **C3**（Lipschitz守门人）：从未验证SCXUpdate函数是否确实Lipschitz连续。共识评分C(x)使用指示函数，在阈值τ处不连续。Lemma"Lipschitz-几乎处处"试图通过测度零论证绕过此问题，但这只处理了C的连续性——SCXUpdate的内部构造未被给出。

- **C10**（参考集重放）：标记为"Lyapunov下降所需"。但该条件意味着系统需要一个在演化开始前已确定的、固定不变的、代表原始分布的参考集。在Spring的实际运行中，该参考集从何而来？如果来自训练数据，则数据已被过滤（Selection bias cycle）。

### 漏洞5.4 [轻微] 诚实标注的自我矛盾

第398行："引理SE-1.1的完整严格证明需要C10和C11。没有这些条件时，Lyapunov下降被定理~12.2证明为形式上不可能。"

但定理12.2证明的是什么？假定无参考集重放即可构造反例——这不等价于证明Lyapunov下降在所有无重放情况下都不可能。这是一个存在性论证与全面否定性论证的混淆。

## 6. Situs Lipschitz（位置编码Lipschitz常数）

**文件**: `situs_theory/main.tex` + `ppe_rigorous_derivation.tex`

### 漏洞6.1 [重大] d_min推导中的Nyquist条件错误

**位置**: situs_theory/main.tex, 推论1.2.1

声称最大波长λ_max ≈ 4dξ，且需满足λ_max ≥ 2L（Nyquist条件），故d_min ≈ L/(2ξ)。

问题一：Nyquist-Shannon采样定理要求采样频率至少为信号最高频率的两倍。在位置编码语境中，"信号"是区间[0,L]上的位置。λ_max对应**最低频率**（最长波长），捕获的是**最长距离相关性**而非最短。覆盖整个区间需要的是λ_max ≥ L（一个完整周期覆盖整个序列），而非2L。2L的要求混淆了"采样间距"和"波长"。

问题二：从λ_max ≈ 4dξ和λ_max ≥ L得到d_min ≈ L/(4ξ)而非L/(2ξ)。作者可能使用了λ_max ≥ 2L后得到L/(2ξ)，但如前所述，≥2L的前提本身是错误的。

### 漏洞6.2 [重大] 符号不一致：Lipschitz常数的两个版本

**situss_theory/main.tex**（定理1.2.3）给出：
L_scalar = (2√2·π/√d)·√(∑_j 1/λ_j²)

**ppe_rigorous_derivation.tex**（定理1.2.3）给出相同的表达式。

但在ppe_rigorous_derivation.tex的第349行声明的Lipschitz常数**未明确归一化因子√(2/d)**是否包含在PE定义中。两个文件间存在定义不统一的可能。审慎的读者无法确定situss版本与ppe版本使用的编码是否相同。

### 漏洞6.3 [重大] 旋转编码Lipschitz常数的高估确认

作者在第494-496行确认了先前版本将L_rot高估为2√(α²+β²+γ²)而非实际的max(α,β,γ)，高出约3.46倍。该修正说明此前的CC审计是正确的——但作者未说明这3.46倍的误差对下游定理（如Theorem 1的δ_s修正）的定量影响。

若δ_s^{PE}与Lipschitz常数线性相关，先前版本将所有结论夸大了约3.5倍。

### 漏洞6.4 [轻微] 旋转等变性缺失

第103行承认3D旋转编码缺少旋转等变性（编码依赖于绝对坐标方向α,β,γ的选择）。这是物理上严重的限制——对于旋转对称的物理系统（绝大多数分子和晶体），编码的输出应仅依赖于原子间的相对位置，而非它们在实验室坐标系中的绝对朝向。

## 7. Yajie NPE（非扩散博弈纳什均衡）

**文件**: `yajie_protocol/main.tex` §3

### 漏洞7.1 [重大] 非对称均衡不可能性的证明有误

**位置**: Theorem "非扩散博弈的纳什均衡完整刻画" 之(iv)，第263-281行

作者声称在非对称纯策略均衡中，所有辖区必须选择相同策略。证明如下：

对于选择A的辖区i和选择D的辖区j，由均衡条件：
```
u_i(A,s*_{-i}) − u_i(D,s*_{-i}) = Δ(|E|) + κ ≥ 0
u_j(D,s*_{-j}) − u_j(A,s*_{-j}) = −Δ(|E|) − κ ≥ 0
```
因此Δ(|E|) = −κ，是零测集。

**问题**：该推导假设了i的单边偏离和j的单边偏离时，其他辖区的策略不变。但在非对称组合中，i的偏离只影响i自身；而u_i(A,s*_{-i})的表达式依赖于**s*_{-i）中其他辖区的具体分布**。作者在第271-273行给出的差分公式过于简化。

具体地，设n_D(s*_{-i}) = n_D*（除i外有n_D*个开发者）。则：
```
u_i(A,s*_{-i}) − u_i(D,s*_{-i}) = Δ(|E|) − κ·n_D* − λ − (−κ·(n_D*+1) − λ)
                               = Δ(|E|) + κ
```

这个Δ(|E|)+κ与n_D*无关？仔细验证：-κ·n_D* + κ·(n_D*+1) = κ。确实消去了。这一步的代数在形式上是正确的。

但对j的差分：
```
u_j(D,s*_{-j}) − u_j(A,s*_{-j}) = −Δ(|E|) − κ
```
该结果同样与n_D*无关——这是正确的。因此两组不等式的联合条件确实是Δ(|E|)=−κ。

然而，此处有一个更微妙的问题：**作者未考虑混合均衡中不同辖区选择不同纯策略的非对称组合**。在纯策略空间中可能无非对称均衡，但混合策略允许不同辖区以不同的概率混合。该可能性被定理文字所排除，但未被证明排除。

### 漏洞7.2 [重大] 对称混合策略均衡的求解有误

**位置**: 第296-304行

在混合策略均衡中，作者推导了期望支付的无差异条件。仔细验证其代数：

选择A的期望支付：
```
E[u_i(A,s_{-i})] = V[θ(|E|)] − c^adopt − κ·E[n_D] − λ·P[n_D > 0]
```

E[n_D] = (n−1)(1−p)，P[n_D > 0] = 1−p^{n−1}。正确。

选择D的期望支付：
```
E[u_i(D,s_{-i})] = V[θ(0)] − c^develop − κ·(E[n_D]+1) − λ
```

此处λ乘以1因为一旦有开发者碎片化即发生？但仅当n_D(s_{-i})≥1时才有开发者——如果i自己是唯一的开发者且n_D(s_{-i})=0，碎片化仍然发生因为i选择了D。所以D→λ的项是正确的。

但无差异条件消去后得到的方程（第298行）：
```
V[θ(|E|)] − c^adopt − λ[1−p^{n−1}] = V[θ(0)] − c^develop − κ − λ
```

核对：从A的支付减去D的支付，κ·(n−1)(1−p)项消去（出现在两边）。但D的支付中κ项为κ·(E[n_D]+1) = κ·((n−1)(1−p)+1)，A的支付中κ项为κ·(n−1)(1−p)。相减得−κ。但方程右边是V[θ(0)]−c^develop−κ−λ，左边是V[θ(|E|)]−c^adopt−λ+λ·p^{n−1}。

所以λ·p^{n−1}项在哪里？A的支付中λ·P[n_D>0] = λ·(1−p^{n−1})，而D的支付中−λ（常数）。所以移到等式：
```
V[θ(|E|)] − c^adopt − λ(1−p^{n−1}) + λ = V[θ(0)] − c^develop − κ
```
即：
```
V[θ(|E|)] − c^adopt + λ·p^{n−1} = V[θ(0)] − c^develop − κ
```

这给出p^{n−1} = (V[θ(0)]−c^develop−κ−V[θ(|E|)]+c^adopt)/λ

在该参数区域内该值必在[0,λ]范围内——由区域定义保证——因而p*存在且唯一。推导是成立的。

不过，|S|=2^n对于n=190（全球国家数）是天文数字，有限博弈的纳什存在性证明虽然形式上正确，但均衡的精炼和选择问题在此规模下极为尖锐。

### 漏洞7.3 [轻微] 对称性假设遮盖非对称均衡

**位置**: 第189行Δ(|E|)定义

Δ(|E|)被定义为与i无关，但这是通过假设所有辖区有相同的c_adopt和相同的V(·)函数实现的。在实际中，不同辖区的采纳成本和审计精度价值可能有显著差异。作者在第8节讨论了放松对称性的影响，但这意味着定理7.1在非对称情况下不成立——该关键限定应在定理陈述中明确。

## 8. AAE不动点（采纳-审计均衡）

**文件**: `yajie_protocol/main.tex` §9（第1646-1760行）

### 漏洞8.1 [重大] 上半连续性验证不完备

**位置**: 第1708-1719行

证明Φ的上半连续性依赖于"控制收敛定理"和"边界G-测度为零"。具体问题：

1. **G绝对连续假设未声明**: 边界{(p,B): B = −p·V(μ)·L(p,B)}的G-测度为零要求G关于Lebesgue测度绝对连续，或至少该边界是G-零测集。但在假设E1-E5中未包含G绝对连续的要求。

2. **逐点收敛到积分收敛**: 控制收敛定理要求被积函数被一个可积函数所控制。指示函数被1所控制，这满足条件。但在μ_n→μ时，V(μ_n)连续（E3保证），逐点收敛确实成立。

3. **关键的隐藏问题**: 证明声称φ的图闭等价于上半连续性。在紧致值域上这是正确的（Closed Graph Theorem for correspondences, Aliprantis & Border Thm 17.11）。但要求值域是紧致的——[0,1]是紧致的，满足条件。

然而，有一个更根本的问题：φ(μ)被定义为：
```
φ(μ) = {μ̃: μ̃ = ∫_Θ 1[B ≥ −p·V(μ)·L(p,B)] dG(p,B)}
```

对于连续的G，该积分是μ的确定性函数。此时φ(μ)是单点集{Φ(μ)}，其中Φ(μ)即该积分的值。此时上半连续性退化为连续性。但对于非连续的G（例如有原子的分布），φ(μ)可能是多值的——当B正好等于−p·V(μ)·L时有原子位于边界上。此时φ(μ)不是凸的。

### 漏洞8.2 [重大] 凸值性条件在G有原子时不成立

第1700行声称φ(μ)是凸集。当G无原子时，积分为单点，退化为凸。但当G有原子时，即使只有一个原子，φ(μ)在μ使该原子恰位于边界时不是凸集（是两点集{μ̃₁, μ̃₂}）。Kakutani定理要求对**所有**μ值（非仅几乎处处）值集是凸的。

### 漏洞8.3 [重大] 审计概率V(μ)的定义有问题

第1670行定义：
```
V(μ) = 1 − (1−q(μ))^μ
```

此定义有两个问题：
1. μ是采纳率∈[0,1]，(1−q)^μ不是自然定义的（实指数的幂）。虽然数学上可定义，但其博弈论含义模糊。
2. 更严重的是，该公式隐含假设每个采纳者独立地以概率q(μ)审计i——但在连续统中，"独立"的审计者总数是μ（连续统测度），而非可数个独立试验。连续统上的独立乘积测度需要更精细的处理。

### 漏洞8.4 [轻微] 多重均衡的稳定性声称

第1728行声称"内部不动点是不稳定的（鞍点），除非Φ'(μ*) < 1"——但Φ是对应(correspondence)而非函数，其"导数"无定义。作者显然在将Φ视为单值函数的情形下讨论（连续G无原子时），但未明确声明该前提。

## 附录：跨定理的系统性问题

### A.1 假设体系的不一致性

SCX框架声称A1-A6是统一假设体系，但各定理实际使用了不同的假设子集：
- Theorem 1: A1-A6
- Theorem 2: 不使用A1-A6（使用信息论假设）
- Theorem 3: "不依赖任何假设"（实际使用了构造性假设）
- Theorem 4: A1-A6 + Bahadur-Rao条件
- Spring: C1-C11（完全不同体系）
- Situs: 物理假设 + 修改的A1-A6
- Yajie: 博弈论假设 A1-A7/E1-E5

这种"根据需求切换假设"的做法违背了公理化的初衷。

### A.2 核心概念的符号滑动

- "状态"(state)在Theorem 1中指s∈S（数据划分），在Spring中指S_t（评分函数），在Yajie中指S_1...S_5（锁入阶段）
- Δ_s在不同定理中有三种不同定义
- θ在Theorem 1为阈值，在Theorem 4为Chernoff点，在Spring中为网络参数

### A.3 证明的完整性问题

在声称的105个定理中，经审阅的10个核心证明链中：
- 2个包含致命错误（Theorem 1独立性假设、Theorem 2 TV传递）

- 仅有1个（Yajie NPE的均衡存在性）在有限博弈范围内形式化正确
