\subsubsection{1.2 Formal Statement}\label{ux5f62ux5f0fux5316ux9648ux8ff0}

\textbf{Theorem 3 (Noise-Difficulty Unidentifiability).} 对任意
\(K \geq 2\) 类分类Problem，任意 \(M \geq 1\) 个专家，任意has 限State Space
\(\mathcal{S}\)，存在两个不同的Data Generation过程 \(\mathcal{P}_1\) 和
\(\mathcal{P}_2\) 使得：

\textbf{(i)} 在 \(\mathcal{P}_1\) 下，观测Label \(y\)
的某些错误由\textbf{Label噪声}（label noise)导致：即 \(y \neq y^*\)，且
\(y^*\) 是真实None误的。

\textbf{(ii)} 在 \(\mathcal{P}_2\) 下，所has 观测Label \(y\) 等于真实Label
\(y^*\)，但某些样本具has \textbf{内在Difficulty度}（intrinsic
hardness)：专家系统性地犯错并非因is Label错误，而是因is 样本本身的歧义性。

\textbf{(iii)} 边际-Conditions分解恒等：两个过程产生完全相同的观测联合分布：

\[\mathcal{P}_1\bigl(x, y, \{f_m(x)\}_{m=1}^M\bigr) = \mathcal{P}_2\bigl(x, y, \{f_m(x)\}_{m=1}^M\bigr), \quad \forall (x, y, \{f_m\}) \in \mathcal{X} \times \mathcal{Y} \times \mathcal{Y}^M\]

\textbf{(iv)} 因此，任意算法
\(\mathcal{A}: (\mathcal{X} \times \mathcal{Y} \times \mathcal{Y}^M)^n \to \{0,1\}^n\)（将
\(n\)
个观测映射到''噪声/非噪声''Assessment)None法同时正确识别两个世界 in the 噪声：在to少一个世界 in the 期望错误率to少is 
\(\eta\rho/2\)（其中 \(\eta\) is 噪声率，\(\rho\)
is 歧义Status的Proportion)。特别地，当 \(\eta > 0\) 且 \(\rho > 0\)
时该下界严格is 正，Proof两个世界None法被完美区分。

\begin{quote}
\textbf{Interpretation}: Theorem 3
表明，在不附加额外Assumptions的Conditions下，``这个样本的Label是错的''和''这个样本太难了所以专家都做错''是观测上等价的Proposition。SCX
框架必须引入Assumptions (A1)-(A6) 来打破这种等价性。
\end{quote}

\subsection{2 Proof (Proof)}\label{ux8bc1ux660e-proof}

\subsubsection{2.1 Construction Framework}\label{ux6784ux9020ux6846ux67b6}

采用\textbf{构造性反例法}（constructive counterexample)。构造两个世界
\(\mathcal{P}_{\text{noise}}\)（噪声世界)和
\(\mathcal{P}_{\text{hard}}\)（Difficulty世界)，使得它们产生相同的观测分布但has 根本不同的因果解释。

设二分类Problem \(K = 2\)，Label Space \(\mathcal{Y} = \{0, 1\}\)。设
\(M \geq 1\) 个专家。Definition两个互斥的Status \(s_1, s_2 \in \mathcal{S}\)。

\subsubsection{2.2 世界 A：噪声驱动
(Noise-Driven)}\label{ux4e16ux754c-aux566aux58f0ux9a71ux52a8-noise-driven}

在 \(\mathcal{P}_{\text{noise}}\) 下：

\textbf{Status \(s_1\)}（``清洁易懂''Status)： -
样本占比：\(\mathbb{P}(X \in s_1) = \rho\)，其中 \(\rho \in (0, 1)\) -
真实Label恒定：对所has  \(x \in s_1\)，\(y^* = 0\) - 噪声机制：Label以概率
\(\eta \in (0, 1/2)\) 被翻转
\[\mathbb{P}(y \neq y^* \mid x \in s_1) = \eta\] -
专家准确率：对清洁样本，所has 专家独立地以 \(1 - \varepsilon_1\)
概率正确预测 \(y^*\)
\[\mathbb{P}(f_m(x) = y^* \mid x \in s_1, \text{clean}) = 1 - \varepsilon_1, \quad \varepsilon_1 \in (0, 1/2)\]

\textbf{Status \(s_2\)}（``Difficulty''Status)： -
样本占比：\(\mathbb{P}(X \in s_2) = 1 - \rho\) - 真实Label恒定：对所has 
\(x \in s_2\)，\(y^* = 0\) -
None噪声：\(\mathbb{P}(y \neq y^* \mid x \in s_2) = 0\) -
专家准确率较低（该Status对专家来说普遍Difficulty)：
\[\mathbb{P}(f_m(x) = y^* \mid x \in s_2) = 1 - \varepsilon_2, \quad \varepsilon_1 < \varepsilon_2 \leq 1/2\]

\subsubsection{2.3 世界 B：难度驱动
(Difficulty-Driven)}\label{ux4e16ux754c-bux96beux5ea6ux9a71ux52a8-difficulty-driven}

在 \(\mathcal{P}_{\text{hard}}\) 下：

\textbf{Status \(s_1\)}（``含难例''Status)： -
样本占比：\(\mathbb{P}(X \in s_1) = \rho\)（与世界 A 相同) -
\textbf{所has Label均is 真实Label}（None噪声)：\(y = y^*\) 对所has 样本 -
但真实Label本身具has 歧义性：
\[\mathbb{P}(y^* = 0 \mid x \in s_1) = 1 - \eta, \quad \mathbb{P}(y^* = 1 \mid x \in s_1) = \eta\]
其中 \(\eta\) 与世界 A 的噪声率相同 -
专家Conditions准确率：给定真实Label，专家的预测linesis 如下： - 当 \(y^* = 0\)
时：\(\mathbb{P}(f_m(x) = 0 \mid x \in s_1, y^* = 0) = 1 - \varepsilon_1\)
- 当 \(y^* = 1\)
时：\(\mathbb{P}(f_m(x) = 0 \mid x \in s_1, y^* = 1) = 1 - \varepsilon_1\)

即专家总是倾向于预测Category \(0\)（偏向性)，与真实LabelNone关。因此： - 当
\(y^* = 0\) 时，专家准确率is  \(1 - \varepsilon_1\)（表现良好) - 当
\(y^* = 1\) 时，专家准确率is  \(\varepsilon_1\)（系统性地犯错)

\textbf{Status \(s_2\)}（``Difficulty''Status)： -
样本占比：\(\mathbb{P}(X \in s_2) = 1 - \rho\)（与世界 A 相同) -
所has Labelis 真实Label：\(y = y^*\)，且
\(\mathbb{P}(y^* = 0 \mid x \in s_2) = 1\) -
专家准确率：\(\mathbb{P}(f_m(x) = y^* \mid x \in s_2) = 1 - \varepsilon_2\)（与世界
A 相同)

\subsubsection{2.4
World Equivalence Verification}\label{ux4e16ux754cux7b49ux4ef7ux6027ux9a8cux8bc1}

现验证两个世界的观测联合分布恒等。观测datais 
\((x, y, f_1(x), \dots, f_M(x))\)，其联合分布可分解is ：

\[\mathcal{P}(x, y, f_1, \dots, f_M) = \mathcal{P}(x) \cdot \mathcal{P}(y \mid x) \cdot \prod_{m=1}^M \mathcal{P}(f_m \mid x)\]

其中第二步使用了 Theorem 1 的Assumptions (A1)-(A2)（专家在给定 \(x\)
下Conditions独立)------但Note意此处我们\textbf{不要求}该Assumptions普遍成立，只是在构造中使其成立以产生最简洁的反例。

\textbf{Step 1: 边际分布 \(\mathcal{P}(x)\)}

两世界中
\(\mathcal{P}(X \in s_1) = \rho\)，\(\mathcal{P}(X \in s_2) = 1 - \rho\)。恒等。

\textbf{Step 2: ConditionsLabel分布 \(\mathcal{P}(y \mid x)\)}

在 \(\mathcal{P}_{\text{noise}}\) 中（世界 A，Status \(s_1\))：
\[\begin{aligned}
\mathcal{P}_{\text{noise}}(y = 0 \mid s_1) &= \mathbb{P}(\text{clean}) \cdot \mathbb{P}(y = y^* \mid \text{clean}) + \mathbb{P}(\text{noise}) \cdot \mathbb{P}(y = y^* \mid \text{noise}) \\
&= (1 - \eta) \cdot 1 + \eta \cdot 0 = 1 - \eta \\
\mathcal{P}_{\text{noise}}(y = 1 \mid s_1) &= (1 - \eta) \cdot 0 + \eta \cdot 1 = \eta
\end{aligned}\]

在 \(\mathcal{P}_{\text{hard}}\) 中（世界 B，Status \(s_1\))：
\[\begin{aligned}
\mathcal{P}_{\text{hard}}(y = 0 \mid s_1) &= \mathbb{P}(y^* = 0 \mid s_1) = 1 - \eta \\
\mathcal{P}_{\text{hard}}(y = 1 \mid s_1) &= \mathbb{P}(y^* = 1 \mid s_1) = \eta
\end{aligned}\]

对Status \(s_2\)，两世界均None噪声/歧义：
\[\mathcal{P}(y = 0 \mid s_2) = 1, \quad \mathcal{P}(y = 1 \mid s_2) = 0\]

因此ConditionsLabel分布在两世界中恒等。\(\checkmark\)

\textbf{Step 3: Conditions专家分布 \(\mathcal{P}(f_m \mid x)\)}

在 \(\mathcal{P}_{\text{noise}}\) 中（世界 A，Status \(s_1\))： -
专家预测分布由真实Label \(y^*\) 训练/习得决定，不-dependence观测Label \(y\)：
\[\mathcal{P}_{\text{noise}}(f_m = 0 \mid s_1) = \mathbb{P}(f_m = y^* \mid s_1) = 1 - \varepsilon_1\]
\[\mathcal{P}_{\text{noise}}(f_m = 1 \mid s_1) = \varepsilon_1\]

在 \(\mathcal{P}_{\text{hard}}\) 中（世界 B，Status \(s_1\))：
\[\begin{aligned}
\mathcal{P}_{\text{hard}}(f_m = 0 \mid s_1) &= \mathbb{P}(y^* = 0 \mid s_1) \cdot \mathbb{P}(f_m = 0 \mid s_1, y^* = 0) \\
&\quad + \mathbb{P}(y^* = 1 \mid s_1) \cdot \mathbb{P}(f_m = 0 \mid s_1, y^* = 1) \\
&= (1 - \eta) \cdot (1 - \varepsilon_1) + \eta \cdot (1 - \varepsilon_1) \\
&= 1 - \varepsilon_1
\end{aligned}\]

\[\begin{aligned}
\mathcal{P}_{\text{hard}}(f_m = 1 \mid s_1) &= (1 - \eta) \cdot \varepsilon_1 + \eta \cdot \varepsilon_1 = \varepsilon_1
\end{aligned}\]

对Status \(s_2\)
两世界相同：\(\mathcal{P}(f_m = 0 \mid s_2) = 1 - \varepsilon_2\)，\(\mathcal{P}(f_m = 1 \mid s_2) = \varepsilon_2\)。

因此Conditions专家分布在两世界中恒等。\(\checkmark\)

\textbf{Step 4: Conditions独立性验证}

在两世界中，我们构造的专家在给定 \(x\)（或等价地，给定
\(s(x)\))下Conditions独立。这是因is ： - 在世界 A
中，专家的训练data独立（由Assumptions A1 直接给出) - 在世界 B
中，专家的预测误差在给定Status和真实Label下独立

因此 \(\prod_{m=1}^M \mathcal{P}(f_m \mid x)\)
的分解成立，且各因子在两世界间匹配。\(\checkmark\)

\textbf{Step 5: 联合分布恒等}

综合 Step 1-4 得：

\[\mathcal{P}_{\text{noise}}\bigl(x, y, \{f_m\}\bigr) = \mathcal{P}_{\text{hard}}\bigl(x, y, \{f_m\}\bigr), \quad \forall (x, y, \{f_m\})\]

两世界的观测联合分布完全一致。\(\square\) (QED第 (iii) 部分)

\subsubsection{2.5
Algorithm Unidentifiability}\label{ux7b97ux6cd5ux4e0dux53efux533aux5206ux6027}

由第 (iii) 部分，两世界产生观测上不可区分的联合分布。设 \(\mathcal{A}\)
is 任意算法，使用 \(n\) 个观测样本
\(D_n = \{(x_i, y_i, \{f_m(x_i)\})\}_{i=1}^n\) 输出每个样本的噪声Assessment
\(z_i \in \{0, 1\}\)。

Definition两种AssessmentMeaning： - 在 \(\mathcal{P}_{\text{noise}}\) 下：\(z_i = 1\)
表示''该样本Label被翻转（噪声)'' - 在 \(\mathcal{P}_{\text{hard}}\)
下：\(z_i = 1\) 表示''该样本Label正确但专家系统犯错（Difficulty)''

由于 \(\mathcal{A}\) 仅-dependence \(D_n\)，而 \(D_n\)
在两个世界 in the 分布完全相同：

\[\mathcal{L}_{\mathcal{P}_{\text{noise}}}(D_n) = \mathcal{L}_{\mathcal{P}_{\text{hard}}}(D_n)\]

其中 \(\mathcal{L}\) 表示分布的 law。因此 \(\mathcal{A}\)
的输出分布在两世界中相同：

\[\mathcal{L}_{\mathcal{P}_{\text{noise}}}(\mathcal{A}(D_n)) = \mathcal{L}_{\mathcal{P}_{\text{hard}}}(\mathcal{A}(D_n))\]

但在两世界中，\textbf{正确的''噪声''集不同}： -
\(\mathcal{P}_{\text{noise}}\) 中真正噪声样本集
\(\mathcal{Z}_{\text{noise}}^* = \{i: y_i \neq y_i^*\} = \{i: x_i \in s_1, y_i = 1\}\)
- \(\mathcal{P}_{\text{hard}}\) 中真正噪声样本集
\(\mathcal{Z}_{\text{hard}}^* = \varnothing\)（None噪声)

由于 \(\mathcal{A}\) None法区分两世界，其输出Marked集 \(\hat{\mathcal{Z}}\)
必须同时适用于两世界。令 \(a\) is 算法在歧义子集
\(\{i: x_i \in s_1, y_i = 1\}\)
上的期望Marked率（该值在两世界中相同，因is 观测分布相同)。在世界 A
中，这些样本均is 噪声（正确Markedis  \(1\))，错误率is  \((1-a)\)；在世界 B
中，这些样本均非噪声（正确Markedis  \(0\))，错误率is  \(a\)。因此：

\[\max\bigl(\text{Error}_{\mathcal{P}_{\text{noise}}}(\mathcal{A}), \text{Error}_{\mathcal{P}_{\text{hard}}}(\mathcal{A})\bigr) \geq \frac{\text{Error}_{\text{noise}} + \text{Error}_{\text{hard}}}{2} \geq \frac{\eta\rho}{2}\]

其中 \(\eta\rho\) 是歧义子集在全部样本 in the 占比。该下界在 \(a = 1/2\)
时达到（算法在歧义样本上等同于随机猜测)。对于典型噪声率 \(\eta = 0.1\)
和 \(s_1\) 占比 \(\rho = 0.5\)，下界is 
\(0.025\)------虽小但严格is 正，proves两世界的不完全可区分性。定性的不可识别性Conclusion（None法完美区分噪声与Difficulty)仍然成立。\(\square\)
(QED第 (iv) 部分)

\subsection{3 Corollary
(Corollaries)：哪些Assumptions能打破不可识别性？}\label{ux63a8ux8bba-corollariesux54eaux4e9bux5047ux8bbeux80fdux6253ux7834ux4e0dux53efux8bc6ux522bux6027}

Theorem 3
表明，不加额外Assumptions时噪声与Difficulty是不可区分的。以下Corollary列出\textbf{Sufficient Condition}------每个Conditions单独出现即可打破不可识别性。

\subsubsection{3.1 Corollary 1：锚点Label (Anchor
Points)}\label{ux63a8ux8bba-1ux951aux70b9ux6807ux7b7e-anchor-points}

\textbf{Corollary 1 (Anchor Labels Break Unidentifiability).}
If 对某个已知子集
\(\mathcal{X}_{\text{anchor}} \subset \mathcal{X}\)，研究者能够获取\textbf{真实Label}
\(y^*\)（如人工复核)，则噪声与Difficulty在 \(\mathcal{X}_{\text{anchor}}\)
上可区分。进一步地，If  \(\mathcal{X}_{\text{anchor}}\)
覆盖了所has Status，则全局可区分。

\textbf{Proof Idea}：在锚点 \(x \in \mathcal{X}_{\text{anchor}}\)
上，比较观测Label \(y\) 与真实Label \(y^*\)： - If 
\(y \neq y^*\)，则该样本必然含噪声 - If  \(y = y^*\) 但专家
\(f_m(x) \neq y\)，则该样本必然是''Difficulty''的

将锚点上的专家错误率外推到非锚点样本，可打破全局不可识别性。\(\square\)

\textbf{与 SCX 的联系}：SCX 的''冷启动协议''（cold start
protocol)-dependence于少量人工审查样本作is 锚点来初始化Status级噪声率估计。

\subsubsection{3.2 Corollary 2：已知专家训练域 (Known Training Domains ---
A1)}\label{ux63a8ux8bba-2ux5df2ux77e5ux4e13ux5bb6ux8badux7ec3ux57df-known-training-domains-a1}

\textbf{Corollary 2 (Known Disjoint Training Data Break
Unidentifiability).} If  \(M\) 个专家在 \(M\)
个\textbf{不相交}的data子集上独立训练（Assumptions
A1)，且研究者知道这些训练集的构成，则噪声与Difficulty可区分。

\textbf{Proof Idea}：不相交训练集意味着专家的错误模式是\textbf{independent }（给定
\(x\))。在噪声样本上，所has 专家的错误率同时上升（因is 所has 专家的预测都针对真实
\(y^*\)，而观测Label
\(y \neq y^*\))。在Difficulty样本上，专家的错误可能相互独立（但每个专家的错误率本身更高)。

关键的数量Difference：令
\(\bar{e}(x) = \frac{1}{M}\sum_m \mathbf{1}\{f_m(x) \neq y\}\)
is 平均专家错误率。在噪声样本上的Conditions分布is ：

\subsubsection{5.2

Label噪声文献 in the 经典结果是\textbf{噪声转移矩阵不可识别性}（Menon et al.,

\((\tilde{Y}, X)\) 的Conditions下，\(T\) 和真实Conditions分布
\(\mathbb{P}(Y^* \mid X)\) 的联合是不可识别的------除非加上Sufficient Condition（如
\(T\) 的对角占优性、存在锚点、或存在多个噪声率已知的标Note者)。

Theorem 3
是此结果向\textbf{多专家场景}的推广。在多专家场景中，我们不仅观测到

\(\{f_m(X)\}\)，但不可识别性仍然成立。关键洞察是：专家本身的错误模式可能来自
\(Y^*\) 的Difficulty区域，也可能来自 \(Y^*\)

\subsubsection{5.4
因果推断的视角}\label{ux56e0ux679cux63a8ux65adux7684ux89c6ux89d2}

From 因果推断的角度，Theorem 3
可被理解is 一种\textbf{因果结构不可识别性}（causal structure
unidentifiability; Pearl, 2009)：

考虑两个因果图：

\textbf{图 A（噪声因果图)}：

\begin{verbatim}
S → Y* → Y ← Noise
↓
X → {f_m}
\end{verbatim}

其中箭头 \(Y^* \to Y\) 表示真实Label决定观测Label，但 \(Y\)
也可被外部噪声直接影响。

\textbf{图 B（Difficulty因果图)}：

\begin{verbatim}
S → Y* (= Y)
↓
X → {f_m}
\end{verbatim}

其中DifficultyStatus \(S\) 同时影响真实Label分布和专家准确率。

两个因果图生成相同的观测分布
\((X, Y, \{f_m\})\)，但蕴含不同的因果解释。仅凭观测dataNone法区分这两个图------这是典型的\textbf{马尔可夫等价类}（Markov
equivalence class)Problem。

\subsubsection{5.5

\(\eta\))

\subsection{附录 A：K 类推广}\label{ux9644ux5f55-ak-ux7c7bux63a8ux5e7f}

将Theorem 3 推广到 \(K > 2\)。采用与二分类不同的构造策略------在世界 B
中，专家被构造is \textbf{完全随机}（不relies on real Label)，这使得联合分布在任意

\textbf{世界 A（噪声)}：对所has 
\(x \in s_1\)，\(y^* = 0\)。噪声机制：以概率 \(\eta\) 将Label均匀翻转到
\(\mathcal{Y} \setminus \{0\}\)：
\[\mathbb{P}(y = c \mid \text{noise}, x \in s_1) = \frac{1}{K-1}, \quad c \neq 0\]

\[\mathbb{P}(f_m = c \mid s_1) = \frac{\varepsilon_1}{K-1}, \quad c \neq 0\]

\[\mathcal{P}_{\text{noise}}(y = 0 \mid s_1) = 1 - \eta, \quad \mathcal{P}_{\text{noise}}(y = c \mid s_1) = \frac{\eta}{K-1}, \; c \neq 0\]
\[\mathcal{P}_{\text{noise}}(f_m = 0 \mid s_1) = 1 - \varepsilon_1, \quad \mathcal{P}_{\text{noise}}(f_m = c \mid s_1) = \frac{\varepsilon_1}{K-1}, \; c \neq 0\]

且由于 (A4) 噪声与专家独立，\(y\) 与 \(f_m\) 在给定 \(s_1\) 下独立。

\textbf{世界 B（Difficulty)}：对
\(x \in s_1\)，所has Label均is 真实Label（None噪声)：
\[\mathbb{P}(y^* = 0 \mid s_1) = 1 - \eta, \quad \mathbb{P}(y^* = c \mid s_1) = \frac{\eta}{K-1}, \; c \neq 0\]
