\section{Definition 1: State-Conditioned Expert
Risk}\label{definition-1-state-conditioned-expert-risk}

\begin{quote}
SCX 框架的核心数学对象定义文档。定义状态、状态条件专家风险、SCX
可靠性、数据价值等基本概念。
\end{quote}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{1. 状态空间 (State
Space)}\label{ux72b6ux6001ux7a7aux95f4-state-space}

\subsubsection{定义 1.1：状态
(State)}\label{ux5b9aux4e49-1.1ux72b6ux6001-state}

状态 \(s\) 是输入空间 \(\mathcal{X}\) 的一个可测子集。状态划分
\(\Pi = \{s_1, s_2, \dots, s_K\}\) 满足：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \(\bigcup_{k=1}^K s_k = \mathcal{X}\)
\item
  \(s_k \cap s_{k'} = \emptyset\) 当 \(k \neq k'\)
\item
  每个 \(s_k\) 是 \(\mathcal{X}\) 的 Borel \(\sigma\)-代数中的可测集
\end{enumerate}

状态划分的数学本质是找到一个映射
\(\phi: \mathcal{X} \to \mathcal{S}\)，使得：

\[P(Y|X) \approx P(Y|\phi(X))\]

即 \(\phi(X)\) 是 \(Y\) 关于 \(X\) 的\textbf{近似充分统计量}。

\subsubsection{定义 1.2：软状态赋值 (Soft State
Assignment)}\label{ux5b9aux4e49-1.2ux8f6fux72b6ux6001ux8d4bux503c-soft-state-assignment}

软状态赋值函数 \(\gamma: \mathcal{X} \times \mathcal{S} \to [0,1]\)
满足：

\[\sum_{s \in \mathcal{S}} \gamma_s(x) = 1, \quad \forall x \in \mathcal{X}\]

常见实现方式：

\begin{itemize}
\tightlist
\item
  \textbf{GMM
  后验概率}：\(\gamma_s(x) = \frac{\pi_s \mathcal{N}(\phi(x) \mid \mu_s, \Sigma_s)}{\sum_{t \in \mathcal{S}} \pi_t \mathcal{N}(\phi(x) \mid \mu_t, \Sigma_t)}\)
\item
  \textbf{Kernel
  加权}：\(\gamma_s(x) = \frac{\kappa(\phi(x), \mu_s)}{\sum_{t \in \mathcal{S}} \kappa(\phi(x), \mu_t)}\)
\item
  \textbf{KNN 概率}：\(\gamma_s(x) = \frac{|\text{KNN}(x) \cap s|}{k}\)
\end{itemize}

\subsection{2. 条件风险 (Conditional
Risk)}\label{ux6761ux4ef6ux98ceux9669-conditional-risk}

\subsubsection{定义 2.1：点态风险 (Pointwise
Risk)}\label{ux5b9aux4e49-2.1ux70b9ux6001ux98ceux9669-pointwise-risk}

专家 \(f_m\) 在输入 \(x\) 处的点态风险：

\[R_m(x) = \mathbb{E}[\ell(f_m(x), f^*(x)) \mid X = x]\]

其中 \(\ell: \mathcal{Y} \times \mathcal{Y} \to \mathbb{R}_{\ge 0}\)
为损失函数，\(f^*: \mathcal{X} \to \mathcal{Y}\) 为真实标注函数。

\subsubsection{定义 2.2：状态条件专家风险 (State-Conditioned Expert
Risk)}\label{ux5b9aux4e49-2.2ux72b6ux6001ux6761ux4ef6ux4e13ux5bb6ux98ceux9669-state-conditioned-expert-risk}

\[R_m(s) = \mathbb{E}_{x \sim P(\cdot|s)}[\ell(f_m(x), f^*(x))]\]

这是 Kolmogorov 意义下的条件期望。由 Radon-Nikodym 定理保证存在性：

\[R_m(s) = \frac{1}{P(X \in s)} \int_{X \in s} \ell(f_m(x), f^*(x)) \, dP_X(x)\]

当 \(P(X \in s) = 0\) 时，\(R_m(s)\) 可定义为
\(\lim_{\epsilon \to 0^+} R_m(s_\epsilon)\)，其中 \(s_\epsilon\) 是
\(s\) 的 \(\epsilon\)-邻域。

\subsubsection{定义 2.3：条件贝叶斯风险 (Conditional Bayes
Risk)}\label{ux5b9aux4e49-2.3ux6761ux4ef6ux8d1dux53f6ux65afux98ceux9669-conditional-bayes-risk}

\[R^*(s) = \inf_{f \text{ 可测}} \mathbb{E}[\ell(f(X), Y) \mid X \in s]\]

专家风险 \(R_m(s)\) 衡量专家 \(f_m\) 相对于\textbf{不可知最优} \(f^*\)
的表现，而非相对于 \(R^*(s)\)。

\subsubsection{偏差-方差分解}\label{ux504fux5dee-ux65b9ux5deeux5206ux89e3}

对于平方损失 \(\ell(\hat{y}, y) = (\hat{y} - y)^2\)，令回归函数
\(r(x) = \mathbb{E}[Y \mid X = x]\)：

\[R_m(s) = \underbrace{\mathbb{E}[(f_m(X) - r(X))^2 \mid X \in s]}_{\text{近似误差}} + \underbrace{\mathbb{E}[(r(X) - f^*(X))^2 \mid X \in s]}_{\text{目标噪声}}\]

在 \(f^* = r\) 的特殊情况下得到\textbf{条件偏差-方差分解}：

\[R_m(s) = \mathbb{E}[(f_m(X) - r(X))^2 \mid X \in s] + \mathbb{E}[\text{Var}(Y \mid X) \mid X \in s]\]

\subsection{3. SCX 可靠性 (SCX
Reliability)}\label{scx-ux53efux9760ux6027-scx-reliability}

\subsubsection{定义 3.1：SCX
可靠性}\label{ux5b9aux4e49-3.1scx-ux53efux9760ux6027}

\[SCX_m(s) = P(\ell(f_m(x), f^*(x)) < \tau \mid x \in s)\]

其中 \(\tau > 0\) 为可靠性阈值。经验估计为：

\[\widehat{SCX}_m(s) = \frac{|\{x_i \in s : \ell(f_m(x_i), y_i) < \tau\}|}{|s|}\]

\subsubsection{与 IRT 的关系}\label{ux4e0e-irt-ux7684ux5173ux7cfb}

SCX 的 \(SCX_m(s)\) 可以映射到 Rasch 模型：

\[\log\left(\frac{SCX_m(s)}{1 - SCX_m(s)}\right) = \theta_s - b_m\]

其中 \(\theta_s\) 是状态 \(s\) 的''可处理性''参数，\(b_m\) 是专家 \(m\)
的能力阈值。

\subsubsection{与 Shapley
值的关系}\label{ux4e0e-shapley-ux503cux7684ux5173ux7cfb}

SCX 的 \(V(s)\) 是一个\textbf{代理 Shapley 值}，它将数据点级别的 Shapley
值计算分解为： - 先聚合成状态 \(s\)（复杂度 \(O(2^{|S|)}\)，其中
\(|S| \ll N\)） - 再使用乘法分解（五个可解释因子的乘积）

\subsection{5. 噪声与可学习性 (Noise \&
Learnability)}\label{ux566aux58f0ux4e0eux53efux5b66ux4e60ux6027-noise-learnability}

\subsubsection{定义
5.1：噪声分数}\label{ux5b9aux4e49-5.1ux566aux58f0ux5206ux6570}

\[\text{NoiseScore}(x_i) = r_i \cdot \frac{1}{\rho(s_i) + \varepsilon} \cdot [1 - C(s_i)]\]

其中 \(C(s)\) 为状态 \(s\) 的内部一致性：

\[C(s) = 1 - \frac{1}{|s|} \sum_{x_i \in s} \frac{|\ell(f_m(x_i), y_i) - \bar{\ell}(s)|}{\max \ell - \min \ell}\]

\subsubsection{定义
5.2：可学习性}\label{ux5b9aux4e49-5.2ux53efux5b66ux4e60ux6027}

\[L(s) = \inf_{f \in \mathcal{F}} \left[ \frac{R(f|s) - R^*(s)}{R^*(s)} \right]^{-1}\]

其中
\(R(f|s) = \mathbb{E}[\ell(f(X), Y) | X \in s]\)，\(R^*(s) = \inf_f R(f|s)\)
是贝叶斯风险。\(L(s) \to 0\) 表示状态在现有函数类 \(\mathcal{F}\)
下不可学习。

\subsubsection{定义
5.3：冗余度}\label{ux5b9aux4e49-5.3ux5197ux4f59ux5ea6}

状态冗余度估计：

\[D(s) = \rho(s) \cdot \bigl(1 - \bar{r}(s)\bigr) \cdot \text{Sim}(s) \cdot \bigl(1 - \text{Boundary}(s)\bigr)\]

其中 \(\text{Sim}(s)\) 为状态内部相似度，\(\text{Boundary}(s)\)
为边界样本比例。

\subsection{6. 专家路由 (Expert
Routing)}\label{ux4e13ux5bb6ux8defux7531-expert-routing}

\subsubsection{定义
6.1：状态条件专家权重}\label{ux5b9aux4e49-6.1ux72b6ux6001ux6761ux4ef6ux4e13ux5bb6ux6743ux91cd}

\[w_m(x) \propto \exp\left(-\alpha \cdot \sum_{s \in \mathcal{S}} \gamma_s(x) \cdot \hat{R}_m(s)\right)\]

归一化保证 \(\sum_{m=1}^M w_m(x) = 1\)，\(\alpha > 0\) 为逆温度参数。

\subsubsection{定义
6.2：硬路由}\label{ux5b9aux4e49-6.2ux786cux8defux7531}

\[m^*(x) = \arg\min_{m \in [M]} \sum_{s \in \mathcal{S}} \gamma_s(x) \cdot R_m(s)\]

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{7. 收敛性质}\label{ux6536ux655bux6027ux8d28}

\subsubsection{定理
7.1：经验风险估计的集中性}\label{ux5b9aux7406-7.1ux7ecfux9a8cux98ceux9669ux4f30ux8ba1ux7684ux96c6ux4e2dux6027}

设 \(\ell \in [0, L_{\max}]\)，状态 \(s\) 有 \(n_s\) 个样本。对任意
\(\delta > 0\)，以概率至少 \(1 - \delta\)：

\[|\hat{R}_m(s) - R_m(s)| \leq L_{\max} \sqrt{\frac{\log(2/\delta)}{2n_s}}\]

\subsubsection{定理
7.2：最小最大下界}\label{ux5b9aux7406-7.2ux6700ux5c0fux6700ux5927ux4e0bux754c}

存在分布使得任意估计器 \(\hat{R}_m(s)\) 满足：

\[\mathbb{E}[|\hat{R}_m(s) - R_m(s)|] \geq \frac{\sigma(s)}{2\sqrt{n_s}}\]

其中 \(\sigma^2(s) = \text{Var}(\ell(f_m(X), f^*(X)) \mid X \in s)\)。
