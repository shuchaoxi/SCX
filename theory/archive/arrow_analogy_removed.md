# [归档] Proposition 1: Arrow 不可能定理的类比

> **移除日期**: 2026-06-27
>
> **移除原因**: 本节将 SCX 的全局专家排序不足问题与 Arrow 不可能定理进行类比。虽然该类比在直觉层面有启发性，但存在以下问题：
>
> 1. **条件不匹配**: Arrow 定理要求普遍性（全域条件）、帕累托最优、非独裁性和无关选项独立性四个条件才能推出不可能性。SCX 的 Proposition 1 仅要求在存在状态级排序交叉时无全局最优专家——不要求 Arrow 的四个条件同时满足。
> 2. **误导性**: 类比暗示 SCX 的不可能性与 Arrow 定理同层次（即无论状态划分如何精细，都不可能存在全局排序），但实际上 SCX 的"不可能性"是条件性的——当状态划分足够细且无交叉时，全局排序是可能的。
> 3. **理论价值有限**: 该类比未对 SCX 框架的分析或推导做出实质贡献，属于启发式修辞而非数学论证。

## 4 Connection to Arrow's Impossibility Theorem

全局专家排序问题与社会选择理论存在深刻的类比关系。

| Arrow 定理 | SCX |
|-----------|-----|
| 投票者 $i$ | 状态 $s$ |
| 候选方案 $a$ | 专家 $f_m$ |
| 社会偏好排序 | 全局风险排序 $R_m$ |
| 帕累托最优 | 状态级排序一致性 |
| 无关选项独立性 | 引入新专家的独立性 |

Arrow 不可能定理指出：不存在将个体偏好映射为社会偏好的函数同时满足 Pareto 最优、非独裁性和无关选项独立性。

SCX 的状态级排序对应于"个体偏好"（每个状态 $s$ 有自己的专家排序），全局排序对应于"社会偏好"。Proposition 1 表明：**全局排序不能一致地反映状态级排序**，正如 Arrow 定理中社会偏好不能同时满足所有理性条件。

但与 Arrow 定理不同的是，SCX 中的"不可能性"不要求普遍性（全域条件），仅指出存在反例。
