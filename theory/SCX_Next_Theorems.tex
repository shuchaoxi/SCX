\subsection{一、因果发现定理：标签噪声
vs.~因果混淆的可分离性}\label{ux4e00ux56e0ux679cux53d1ux73b0ux5b9aux7406ux6807ux7b7eux566aux58f0-vs.-ux56e0ux679cux6df7ux6dc6ux7684ux53efux5206ux79bbux6027}

\subsubsection{（Causal Discovery Theorem ---
CD-Theorem）}\label{causal-discovery-theorem-cd-theorem}

\subsubsection{语境}\label{ux8bedux5883}

Theorem
3（不确定性原理）断言：在SCX审计框架内，\textbf{噪声与固有困难不可区分}。形式化地说，给定Cercis
Score S = Q +
ηN，审计者无法仅从S的观测值分解出质量分量Q和噪声分量N------存在一个不可约的信息壁垒。

但Theorem
3留了一个关键缺口未关闭：\textbf{噪声（标签噪声、测量噪声）与未观测混杂因子（unobserved
confounders）是否也不可区分？}
这是一个层次更深的问题。标签噪声是对监督信号的随机扰动；因果混淆是因未观测共同原因导致的虚假关联。两者在观测分布上可能表现出完全相同的统计特征，但它们的干预含义截然不同------前者可通过数据清洗缓解，后者必须通过因果识别（do-calculus、工具变量等）解决。

\subsubsection{定理陈述（草案）}\label{ux5b9aux7406ux9648ux8ff0ux8349ux6848}

\begin{quote}
\textbf{CD-Theorem（因果可分离定理）}： 令 D = \{(xᵢ, yᵢ)\}
为审计数据集，yᵢ = f*(xᵢ) + εᵢ + β·zᵢ，其中 εᵢ 为独立标签噪声，zᵢ
为未观测混杂因子，β 为混杂强度向量。定义审计者可获取的信息集为 I\_A =
\{S(x), ∇S(x), H\_S(x)\}（Cercis Score及其一阶、二阶梯度）。

\textbf{(i) 弱可分离性（开放问题）}：仅使用 I\_A，存在一个检验统计量
τ(I\_A) 使得当 β ≠ 0 时，τ
的势函数（power）严格大于仅含标签噪声情形下的显著性水平，即： P(τ
\textgreater{} c\_α \textbar{} β ≠ 0) \textgreater{} α 对于某些 β ≠ 0 且
ε 有界。 该结论是否为真，取决于 η（Sprint噪声参数）是否与 z
具有非零互信息 I(η; z) \textgreater{} 0。

\textbf{(ii) 强可分离性（严格可推，从Theorem 3直接导出）}：若 η 与 z
独立（即I(η; z) =

ε-主导样本与 z-主导样本。

\textbf{(iii) 中继定理（严格可推，从T7导出）}：若存在一个源域 D\_s

\subsection{二、联邦审计定理：零知识多方审计完备性}\label{ux4e8cux8054ux90a6ux5ba1ux8ba1ux5b9aux7406ux96f6ux77e5ux8bc6ux591aux65b9ux5ba1ux8ba1ux5b8cux5907ux6027}

\subsubsection{（Federated Audit Theorem ---
FA-Theorem）}\label{federated-audit-theorem-fa-theorem}

\subsubsection{语境}\label{ux8bedux5883-1}

``审计之剑''（Audit
Sword）确立了SCX的核心权力：\textbf{任何SCX使用行为可被独立审计}。但这把剑目前假设审计者拥有完整数据访问权。在联邦场景下------多方持有私密数据、仅通过梯度或摘要统计交互------审计之剑遇到了关节锁：你如何审计一个你无法看见的数据集？

联邦学习已有安全聚合（Secure
Aggregation）、差分隐私（DP）、多方计算（MPC）等工具。但它们解决的是''防止泄露''，而非''使可审计''。SCX的目标是逆向的：我们要在\textbf{不暴露原始数据的前提下，使审计结论具有与全数据审计相同的统计效力}。

\subsubsection{定理陈述（草案）}\label{ux5b9aux7406ux9648ux8ff0ux8349ux6848-1}

\begin{quote}
\textbf{FA-Theorem（联邦审计定理）}： 设有 K

\textbf{(i) 联邦审计完备性（开放问题，部分可推）}：
定义完全信息审计统计量 T\_full = T(D\_1,\ldots,D\_K)，联邦审计统计量
T\_fed = T\_F(I\_Fed)。若协议 Π 满足： (a) 各方提交的 S\_k
经零知识证明验证其诚实性（即 ∃ π\_ZK : Verify(π\_ZK, S\_k, Commit(D\_k))
= 1）； (b) Situs编码在各方间保持同态：Situs(∪D\_k) ≅ ⊕\_k
Situs(D\_k)（⊕为某种可交换的组合算子）， 则存在一个 ε-审计等价性：
\textbar Power(T\_fed) - Power(T\_full)\textbar{} ≤ ε(K, δ) 其中 δ

\textbf{(ii) 信息下界（严格可推，从Theorem 3导出）}： 若 ∃ k 使得 D\_k
的标签噪声参数 η\_k
对审计者完全未知（即无π\_ZK验证），则对任何联邦审计统计量

\textbar D\_k\textbar/\textbar D\_total\textbar{}

\textbf{(iii) 电报公共品扩展（开放问题）}：
若''电报公共品''（数据存储是基础设施）延伸至联邦层，即存在一个去中心化存储层承诺（commitment）了所有
D\_k 的 Situs 哈希，则FA-Theorem (i) 中的 π\_ZK
要求可被替换为存储证明（Proof of Storage），ε 上界进一步收紧。

\subsection{三、对抗鲁棒性定理：对抗扰动是''困难''的特例还是第三类？}\label{ux4e09ux5bf9ux6297ux9c81ux68d2ux6027ux5b9aux7406ux5bf9ux6297ux6270ux52a8ux662fux56f0ux96beux7684ux7279ux4f8bux8fd8ux662fux7b2cux4e09ux7c7b}

\subsubsection{（Adversarial Robustness Theorem ---
AR-Theorem）}\label{adversarial-robustness-theorem-ar-theorem}

\subsubsection{语境}\label{ux8bedux5883-2}

Theorem 3建立了噪声与困难的二元不可区分框架。但对抗样本（adversarial
examples）的发现给这个二元框架制造了麻烦：对抗扰动不是随机噪声------它是针对模型梯度的定向最优化结果；它也不是''自然困难''------人类毫不费力就能正确分类对抗样本。这引出一个根本问题：

\textbf{SCX的Cercis分类学中，是否存在''对抗性''作为第三基本类别，与Q（质量）、N（噪声）并列，即
S = Q + ηN + γA？}

\subsubsection{定理陈述（草案）}\label{ux5b9aux7406ux9648ux8ff0ux8349ux6848-2}

\begin{quote}
\textbf{AR-Theorem（对抗鲁棒性定理）}： 设对抗扰动 δ\_adv =
argmax\_\{‖δ‖≤ε\} L(f(x+δ), y)，且 ‖δ\_adv‖
在SCX审计框架内受Situs距离度量。定义对抗暴露度（Adversarial
Exposure）A(x) = ‖∇\_x S(x)‖ · ε / S(x)。

\textbf{(i) 还原定理（开放问题，部分可推）}： 若模型 f 的Cercis梯度场 ∇S
在x的Situs邻域内满足Lipschitz条件且Lipschitz常数 L \textless{}
S(x)/ε，则对抗扰动可被Theorem 3的吸收到''噪声''分量中： A(x) ≤
η\_eff(x)，其中 η\_eff = η + L·ε/S(x)（有效噪声率）。 反之，若 L ≥
S(x)/ε，则对抗扰动构成一个不可被Theorem 3吸收的\textbf{第三类奇点}。

\textbf{(ii) 审计可检测性（严格可推，从T5主动学习定理导出）}：
定义审计者的对抗检测统计量： D\_adv(x) = ‖∇S(x)‖ /
Var\_\{x'∈N(x)\}{[}S(x'){]} 其中 N(x) 为 Situs 局部邻域。若 γ
\textgreater{}
0（即存在第三类对抗分量），则当审计者采用T5主动学习策略以最大化I(S;
D\_adv)时，检测的样本复杂度为： n\_detect ≥ Ω(1/γ² · log(1/δ))
且该下界对最优主动学习策略是紧的。

\textbf{(iii) 相变定理（开放问题）}： 存在一个临界Cercis阈值
S\_crit，使得： - 当 S(x) \textgreater{}
S\_crit：对抗扰动为''困难''的特例（还原至Theorem 3框架）， - 当 S(x) ≤

的解析形式及其与模型架构的关系是开放的。

\subsection{四、时序SCX定理：非平稳分布下的审计漂移}\label{ux56dbux65f6ux5e8fscxux5b9aux7406ux975eux5e73ux7a33ux5206ux5e03ux4e0bux7684ux5ba1ux8ba1ux6f02ux79fb}

\subsubsection{（Temporal SCX Theorem ---
TS-Theorem）}\label{temporal-scx-theorem-ts-theorem}

\subsubsection{语境}\label{ux8bedux5883-3}

静态SCX框架假设数据分布P(X,Y)固定。Spring SE-1引入了动态演化（模型更新 →
数据分布漂移 →
SCX参数η漂移），但SE-1目前只处理了''模型更新作为外生冲击''的场景。现实中的分布漂移更复杂：
- \textbf{概念漂移}（concept drift）：P(Y\textbar X)变化； -
\textbf{协变量漂移}（covariate shift）：P(X)变化； -
\textbf{先验漂移}（label shift）：P(Y)变化。

更关键的是，\textbf{Sprint的核心参数η（噪声-困难耦合系数）自身随时间漂移}------这就是''审计的审计''问题：谁在审计Sprint本身？

\subsubsection{定理陈述（草案）}\label{ux5b9aux7406ux9648ux8ff0ux8349ux6848-3}

\begin{quote}
\textbf{TS-Theorem（时序SCX定理）}： 令时间索引 t =
0,1,\ldots,T。在每个时刻 t，定义数据分布 P\_t(X,Y)，对应的Cercis Score
S\_t，Sprint参数 η\_t，以及Situs编码 U\_t = Situs(P\_t)。假设 η\_t
的演化遵循： η\_\{t+1\} = η\_t + Δη(P\_t, M\_t) + ξ\_t 其中 Δη
为确定性漂移（由模型 M\_t 和数据分布 P\_t 共同决定），ξ\_t
为随机冲击（均值0，方差 σ²\_ξ）。

\textbf{(i) 漂移可检测性（严格可推，从T7跨域迁移导出）}：
若两时刻的Situs距离 d\_Situs(U\_t, U\_\{t+1\}) ≥ δ\_min，则在显著性水平
α 下，审计者能以概率 ≥ 1-β 检测到 η 的漂移，所需样本量满足： n\_t +
n\_\{t+1\} ≥ 2(σ²\_ξ + κ·δ²\_min)·(z\_\{α/2\} + z\_β)² / δ²\_min 其中 κ
为Situs距离与η漂移的Lipschitz常数（从T7继承）。

\textbf{(ii) 漂移归因（开放问题）}： 当检测到 η 漂移后，能否将其分解为：
Δη = Δη\_concept + Δη\_covariate + Δη\_label + Δη\_model
即归因至概念漂移、协变量漂移、先验漂移和模型更新的各自贡献？若无额外假设（如部分可观测的验证集），该分解在SCX框架内是\textbf{不可识别的}。然而，若存在一组锚点样本
A ⊂ X 满足 P\_t(Y\textbar X∈A) ≈
P\_\{t+1\}(Y\textbar X∈A)（概念稳定锚点），则 Δη\_concept 可被独立估计。

\textbf{(iii) Sprint自审计极限定理（开放问题，部分可推）}： Sprint
SE-1的自我修正机制（η → η'
通过模型重训练）是否总能将η拉回均衡？定义Lyapunov函数 V(η) = (η -
η\emph{)\^{}2，其中 η} 为最优噪声参数。若存在一个紧集 Ω ⊂ {[}0,1{]}
使得对 η\_t ∈ Ω，有： E{[}V(η\_\{t+1\}) \textbar{} η\_t{]} ≤
ρ·V(η\_t)，ρ \textless{} 1 则Sprint过程是几何遍历的（geometrically
ergodic），η\_t 收敛至 η*。但若分布漂移速率 v\_drift 满足 v\_drift
\textgreater{}
(1-ρ)/Δt，则Sprint发散------自审计能力有\textbf{速率上限}。
\end{quote}

\subsection{五、人机协同审计定理：Theorem
3的认知边界}\label{ux4e94ux4ebaux673aux534fux540cux5ba1ux8ba1ux5b9aux7406theorem-3ux7684ux8ba4ux77e5ux8fb9ux754c}

\subsubsection{（Human-AI Collaborative Audit Theorem ---
HC-Theorem）}\label{human-ai-collaborative-audit-theorem-hc-theorem}

\subsubsection{语境}\label{ux8bedux5883-4}

Theorem
3的核心主张是：噪声与困难在SCX的自动审计框架内不可区分。但这句话里最关键的两个字是：\textbf{``框架内''}。审计框架的操作边界取决于我们给审计者配备了什么样的信息处理器。如果审计者只是一个基于Cercis
Score的自动化系统，那么Theorem
3是铁律。但如果审计者可以是\textbf{人类专家 + AI系统的混合体}呢？

人类拥有些什么AI审计者没有的？ -
\textbf{先验因果知识}（关于世界如何运作的非统计知识）； -
\textbf{反事实推理能力}（``如果标注者更仔细，这个样本的标签会变吗？''）；
- \textbf{元认知判断}（``这个样本我都不确定，所以它可能真的很难''）。

Theorem 3的边界，就是人类认知可以介入并打破噪声-困难对称性的那条线。

\subsubsection{定理陈述（草案）}\label{ux5b9aux7406ux9648ux8ff0ux8349ux6848-4}

\begin{quote}
\textbf{HC-Theorem（人机协同审计定理）}： 定义审计系统为 A = (AI, H,
Π)，其中 AI 为基于SCX的自动审计模块，H 为人类审计者，Π
为人机交互协议。定义人类审计预算
B\_H（人类可审查的最大样本数），以及人类判断函数 J\_H: X → \{noise,
hard, uncertain\}。

\textbf{(i) 协同增益定理（开放问题，方向性结论）}： 若满足以下两个条件：
(a) \textbf{信息不对称条件}：存在一个样本子集 X\_H ⊂ X，使得人类的判断
J\_H(x) 与Cercis Score S(x) 在条件分布上有： P(J\_H=noise \textbar{}
S=s) ≠ P(J\_H=hard \textbar{} S=s) 对某些 s 成立
（即人类判断携带了S之外的额外信息）； (b)
\textbf{校准条件}：人类对''不确定''的判断本身是校准的，即
E{[}J\_H=uncertain \textbar{} S=s{]} 随 s → 0 单调递增，
则人机协同审计的噪声-困难分离效能 Ω\_collab 严格优于纯AI审计： Ω\_collab
\textgreater{} Ω\_AI = 0（在Theorem 3下）。

\textbf{(ii) 预算衰减律（严格可推，从Theorem 3导出）}： 当人类审计预算
B\_H → 0 时，协同审计退化为纯AI审计，且收敛速率为： Ω\_collab(B\_H) =
Ω\_AI + c·√(B\_H/n)·Δ\_I(S; J\_H) + o(√(B\_H/n)) 其中 Δ\_I
为人类判断与Cercis Score之间的信息增量（以互信息度量）。

\textbf{(iii) 失败定理------Theorem 3的绝对边界（严格可推）}：
即使引入任意人类审计能力，以下情形下噪声-困难仍\textbf{绝对不可区分}：
当且仅当 ∀x, I(J\_H(x); ε(x) \textbar{} S(x)) = 0
即人类的判断与标签噪声在给定S的条件下完全不携带互信息。

等价地，人类打破Theorem
3的充分必要条件是：存在至少一个样本，其标签噪声的统计特征在人类的认知表征中留下了S之外的痕迹。如果噪声是''完美噪声''（与任何可观测特征无关，包括人类的先验知识），则Theorem
3的屏障对人类同样成立。
\end{quote}
