\paragraph{F1: Reversed Entropy Inequality in Proposition 3 (§2.2, Step
3)}\label{f1-reversed-entropy-inequality-in-proposition-3-2.2-step-3}

\begin{itemize}
\tightlist
\item
  \textbf{Location}:
  \texttt{propositions/03\_state\_conditioned\_weighting.md}, Section
  2.2, Step 3
\item
  \textbf{Claim}: \texttt{H(w\_global)\ ≤\ E\_s{[}H(w(s)){]}}
\item
  \textbf{Correct}: By concavity of Shannon entropy,
  \texttt{H(E{[}w(s){]})\ ≥\ E{[}H(w(s)){]}}, so
  \texttt{H(w\_global)\ ≥\ E\_s{[}H(w(s)){]}}. The inequality is
  \textbf{reversed}.
\item
  \textbf{Impact on main result}: LOW. Theorem 3.1 uses a different,
  correct derivation via the Gibbs identity. The entropy step is only in
  the heuristic sketch.
\item
  \textbf{Action}: Fix inequality direction or remove Step 3 (Theorem
  3.1 already provides the complete proof).
\end{itemize}

\paragraph{F2: SE-2 Assumes a.s. Strict Descent but Only In-Expectation
Proven}\label{f2-se-2-assumes-a.s.-strict-descent-but-only-in-expectation-proven}

\begin{itemize}
\tightlist
\item
  \textbf{Location}: \texttt{self\_evolution/07\_completeness.md},
  Theorem SE-2, Assumption 4
\item
  \textbf{Issue}: Theorem 12.5 proves descent \textbf{in expectation}.
  SE-2 requires deterministic (or probability-1) strict descent at each
  step. The gap between \texttt{E{[}ΔΦ\_t{]}\ ≤\ -η\_t} and
  \texttt{Φ\ decreases\ with\ probability\ 1} is not bridged.
\item
  \textbf{Severity}: MODERATE. Martingale convergence theorems can
  bridge this gap with additional conditions.
\item
  \textbf{Action}: Modify SE-2's assumption to ``Φ is a supermartingale
  with E{[}Φ\_\{t+1\}\textbar F\_t{]} ≤ Φ\_t − γ\_t, γ\_t \textgreater{}
  0 unless at a fixed point.''
\end{itemize}

\item
  \textbf{Impact}: The cumulative distribution shift ΣTV(P\_\{t+1\},
  P\_t) \textless{} ∞ bound in Section 2 is unsupported.
\item
  \textbf{Action}: Either add the explicit condition Σβ\_t \textless{}
  ∞, or prove it follows from the specific canonical schedule β\_t =
  t\^{}\{-b\} with b \textgreater{} 1 (not just b \textgreater{} a).

\paragraph{C-MAJOR-3: Importance Weights Unbounded Under Aggressive
Filtering}\label{c-major-3-importance-weights-unbounded-under-aggressive-filtering}

\begin{itemize}
\tightlist
\item
  \textbf{Issue}: Theorem 12.3 requires bounded importance weights
  ‖w\_t‖\emph{∞ ≤ W \textless{} ∞. But when the gatekeeper excludes
  certain regions (S\_t(x,y) ≈ 0 where P\_0(x,y) \textgreater{} 0), w(x)

\item
  \textbf{Impact}: The importance-sampling fix only works when the
  gatekeeper is NOT doing aggressive filtering --- exactly the regime
  where Theorem 1's noise detection is most needed.
\item
  \textbf{Action}: Add explicit discussion of this trade-off: ``The
  importance weight bound W ≤ 1/ε implies that Lyapunov descent is
  provable only when the gatekeeper maintains coverage of all regions.
  Regions where S\_t(x,y) \textless{} ε·E{[}S\_t{]} may be effectively
  lost, creating a tension between aggressive filtering and convergence
  guarantees.''
\end{itemize}
